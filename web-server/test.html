9 测试 Servlets

 (require web-server/test) package: web-server-lib

Web Server 为编写 Web servlet 和调度器的测试提供了一个简单的工具。

核心功能允许向 servlet 发送一个请求并捕获响应:

value

tester/c : contract?

这个契约相当于
(->* ()
       ((or/c string? url? request? false/c)
        (listof binding?)
        #:raw? boolean?
        #:headers? boolean?)
       (or/c bytes?
             xexpr?
             (cons/c bytes?
                     (or/c bytes?
                           xexpr?))))

它代表一个接受请求并返回 servlet 对该请求的回答的函数。 这个交互函数有许多可能的调用模式:

  • 没有参数:对根 URL 路径的调用,没有绑定。

  • 至少一个参数:这可能是一个字符串、URL 或一个请求数据结构。

  • 两个参数:第一个参数必须是一个字符串或一个 URL,但第二个参数可以指定请求绑定。

  • 可选的 #:raw? 关键字控制是否作为结果返回一个 X 表达式或一个字节字符串。

  • 可选的 #:headers? 关键字控制是否将头信息作为字节字符串包含在返回值中。 当使用这个时,两个返回值会以弊端形式返回。

procedure

(make-servlet-tester servlet) → tester/c

  servlet : 
(-> request?
    can-be-response?)

这个函数接受一个 servlet 函数,并提供一个如上所述的测试器函数。 它等同于 (make-dispatcher-tester (dispatch/servlet servlet)) , 所以如果你需要 dispatch/servlet 的自定义参数,请使用 make-dispatcher-tester 。

procedure

(make-dispatcher-tester d) → tester/c

  d : dispatcher/c
这个函数接受一个 dispatcher 并提供一个如上所述的测试器函数。

这个设施被设计为与提取延续 URL 和相关值的技术协同使用; xml/path 是一种方法。 这里有一个扩展的例子,测试一个 Add-Two-Numbers.com:

(define (test-add-two-numbers -s>)
  (define x (random 500))
  (define xs (string->bytes/utf-8 (number->string x)))
  (define y (random 500))
  (define ys (string->bytes/utf-8 (number->string y)))
 
  (define r0 (-s>))
  (define k0 (se-path* '(form #:action) r0))
  (define i0 (se-path* '(form input #:name) r0))
  (define r1
    (-s> (format "~a?~a=~a" k0 i0 xs)
         (list (make-binding:form (string->bytes/utf-8 i0) xs))))
  (define k1 (se-path* '(form #:action) r1))
  (define i1 (se-path* '(form input #:name) r1))
  (define r2
    (-s> (format "~a?~a=~a" k1 i1 ys)
         (list (make-binding:form (string->bytes/utf-8 i1) ys))))
  (define n (se-path* '(p) r2))
  (check-equal? n
                (format "The answer is ~a" (+ x y))))
 
(require
 (prefix-in ex:add1: web-server/default-web-root/htdocs/servlets/examples/add)
 (prefix-in ex:add2: web-server/default-web-root/htdocs/servlets/examples/add-v2))
 
(test-add-two-numbers
 (make-servlet-tester ex:add1:start))
(test-add-two-numbers
 (make-servlet-tester ex:add2:start))

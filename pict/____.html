这些函数将适当的颜色应用于图片 p 。

Example:
> (red (disk 20))

Added in version 1.4 of package pict-lib.

procedure

(light color) → color/c

  color : color/c

procedure

(dark color) → color/c

  color : color/c
这些函数产生较浅或较深的颜色版本。

Example:
> (hc-append (colorize (disk 20) "red")
             (colorize (disk 20) (dark "red"))
             (colorize (disk 20) (light "red")))

Added in version 1.4 of package pict-lib.

value

color/c : flat-contract?

这个契约可以识别颜色字符串、 color% 实例和 RGB 颜色列表。

Added in version 1.4 of package pict-lib.

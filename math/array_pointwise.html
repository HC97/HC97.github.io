6.10 指向操作

本节中记录的大多数操作都是简单的宏,将 array-map 应用于一个函数和它们的数组参数。

procedure

(array-map f) → (Array R)

  f : (-> R)
(array-map f arr0) → (Array R)
  f : (A -> R)
  arr0 : (Array A)
(array-map f arr0 arr1 arrs ...) → (Array R)
  f : (A B Ts ... -> R)
  arr0 : (Array A)
  arr1 : (Array B)
  arrs : (Array Ts)
将 f 与给定的数组 procedures 进行组合。 当数组的形状不匹配时,它们首先被 broadcast 到相同的形状。 如果广播失败, array-map 会引发一个错误。

Examples:
> (array-map (λ: ([x : String]) (string-append x "!"))

布尔运算符被提升为对数组的操作。

当给定 nonstrict 数组时, array-and, array-or 和 array-if 的捷径行为可以使它们的元素不被引用(并因此被计算)。 然而,这些宏不能用来区分递归函数中的基数和归纳情况,因为数组参数被急切地评估。 例如,这个函数永远不会返回,即 array-strictness 是 #f:
(: array-factorial ((Array Integer) -> (Array Integer)))
(define (array-factorial arr)
  (array-if (array<= arr (array 0))
            (array 1)
            (array* arr (array-factorial (array- arr (array 1))))))

6.10.1 广播

parameter

(array-broadcasting) → (U Boolean 'permissive)

(array-broadcasting broadcasting) → void?
  broadcasting : (U Boolean 'permissive)
决定了在广播数组进行点操作时使用的规则。 参见 广播控制 。

procedure

(array-shape-broadcast dss [broadcasting]) → Indexes

  dss : (Listof Indexes)
  broadcasting : (U Boolean 'permissive) = (array-broadcasting)
如果一些具有 dss 形状的数组按照给定的广播规则被广播到一个点操作中,则确定结果数组的形状。 如果广播失败, array-shape-broadcast 会引发一个错误。

Examples:
> (array-shape-broadcast '())

- : Indexes

'#()

> (array-shape-broadcast (list (vector) ((inst vector Index) 10)))

- : Indexes

'#(10)

> (array-shape-broadcast (list ((inst vector Index) 2)
                               ((inst vector Index) 10)))

array-shape-broadcast: incompatible array shapes

(array-broadcasting #t): '#(2), '#(10)

> (array-shape-broadcast (list ((inst vector Index) 2)
                               ((inst vector Index) 10))
                         'permissive)

- : Indexes

'#(10)

procedure

(array-broadcast arr ds) → (Array A)

  arr : (Array A)
  ds : Indexes
返回一个通过插入新的轴和重复的行制作的具有 ds 形状的数组。 这用于 (array-broadcasting #t) 和 (array-broadcasting 'permissive) 。

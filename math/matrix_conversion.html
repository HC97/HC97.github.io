7.4 转换

procedure

(list->matrix m n xs) → (Matrix A)

  m : Integer
  n : Integer
  xs : (Listof A)

procedure

(matrix->list M) → (Listof A)

  M : (Matrix A)
将一个平面列表转换为一个 m×n 矩阵,然后再转换回来; m 和 n 都必须是正数,而且 (* m n) = (length xs) 。 xs 中的条目是以行为主的顺序。

Examples:
> (list->matrix 2 3 '(1 2 3 4 5 6))

- : (Array Positive-Byte)

(array #[#[1 2 3] #[4 5 6]])

> (matrix->list (matrix [[1 2] [3 4] [5 6]]))

- : (Listof Positive-Byte)

'(1 2 3 4 5 6)

procedure

(vector->matrix m n xs) → (Matrix A)

  n : Integer
  xs : (Vectorof A)

procedure

(matrix->vector M) → (Vectorof A)

  M : (Matrix A)
像 list->matrix 和 matrix->list, 但对向量而言。

Examples:

       Index

procedure

(->row-matrix xs) → (Matrix A)

  xs : (U (Listof A) (Vectorof A) (Array A))

procedure

(->col-matrix xs) → (Matrix A)

  xs : (U (Listof A) (Vectorof A) (Array A))
将一个列表、向量或数组转换成行或列矩阵。 如果 xs 是一个数组,它必须是非空的,并且 不 能有一个以上的长度大于 1 的轴。

Examples:
> (->row-matrix '(1 2 3))

- : (Array Positive-Byte)

(array #[#[1 2 3]])

> (->row-matrix #(1 2 3))

- : (Array Positive-Byte)

(array #[#[1 2 3]])

> (->row-matrix (col-matrix [1 2 3]))

- : (Array Positive-Byte)

(array #[#[1 2 3]])

> (->col-matrix (array #[#[#[1]] #[#[2]] #[#[3]]]))

- : (Array Positive-Byte)

(array #[#[1] #[2] #[3]])

> (->col-matrix (matrix [[1 0] [0 1]]))

array->col-matrix: contract violation

  expected: nonempty Array with exactly one axis of length

>= 1

  given: (array #[#[1 0] #[0 1]])

procedure

(list*->matrix xss) → (Matrix A)

  xss : (Listof (Listof A))

procedure

(matrix->list* M) → (Listof (Listof A))

  M : (Matrix A)
将一个列表中的条目转换成矩阵,然后再转换回来。

Examples:
> (list*->matrix '((1 2 3) (4 5 6)))

- : (Array Positive-Byte)

(array #[#[1 2 3] #[4 5 6]])

> (matrix->list* (matrix [[1 2 3] [4 5 6]]))

- : (Listof (Listof Positive-Byte))

'((1 2 3) (4 5 6))

这些函数就像 list*->array 和 array->list*, 但是使用固定深度(即非递归)的列表类型,并且不需要一个谓词来区分条目和行。

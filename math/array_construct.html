6.7 结构

syntax

(array #[#[...] ...] maybe-type-ann)

 
maybe-type-ann = 
  | : type
从表达式的嵌套行中创建一个 Array 。

矢量语法 #[...] 限定了行。 这些行可以嵌套到任何深度,并且必须有一个矩形的形状。 使用方括号不是必须的,但我们鼓励使用方括号,以帮助从视觉上区分数组内容和数组索引及其他向量。 (参见 indexes-array 的例子来说明)。

procedure

(build-array ds proc) → (Array A)

  ds : In-Indexes
  proc : (Indexes -> A)

procedure

(array->mutable-array arr) → (Mutable-Array A)

  arr : (Array A)
返回一个可变数组,其元素与 arr 相同。 其结果是 arr 的副本,即使 arr 是可变的。

procedure

(mutable-array-copy arr) → (Mutable-Array A)

  arr : (Mutable-Array A)
类似于 (array->mutable-array arr), 但是只限于可变数组。 它也更快。

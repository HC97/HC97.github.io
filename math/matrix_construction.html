procedure

(identity-matrix n [one zero]) → (Matrix A)

  n : Integer
  one : A = 1
  zero : A = 0
返回一个 n×n 身份矩阵,其对角线上的值为 one , 其他地方为 zero 。 高度/宽度 n 必须是正数。

Examples:
> (identity-matrix 3)

- : (Array (U One Zero))

(array #[#[1 0 0] #[0 1 0] #[0 0 1]])

> (identity-matrix 4 1.0+0.0i 0.0+0.0i)

procedure

(make-matrix m n x) → (Matrix A)

  m : Integer
  n : Integer
  x : A
返回一个充满 x 值的 m×n 矩阵; m 和 n 都必须是正数。 类似于 make-array (并以其为定义)。

procedure

(build-matrix m n proc) → (Matrix A)

  m : Integer
  n : Integer

返回一个 m×n 矩阵,其条目由 proc 返回; m 和 n 都必须是正数。 类似于 build-array (并以其定义)。

procedure

(vandermonde-matrix xs n) → (Matrix Number)

  xs : (Listof Number)
  n : Integer
Wikipedia: Vandermonde matrix 凡德蒙德矩阵返回一个 m×n 的凡德蒙德矩阵,其中 m = (length xs) 。

Examples:
> (vandermonde-matrix '(1 2 3 4) 5)

- : (Array Real)

(array #[#[1 1 1 1 1] #[1 2 4 8 16] #[1 3 9 27 81] #[1 4 16 64 256]])

> (vandermonde-matrix '(5.2 3.4 2.0) 3)

- : (Array Flonum)

(array

 #[#[1.0 5.2 27.040000000000003]

   #[1.0 3.4 11.559999999999999]

   #[1.0 2.0 4.0]])

使用范德蒙德矩阵来寻找拉格朗日多项式(通过给定的一组点的最小度数的多项式):
> (define (lagrange-polynomial xs ys)
    (array->list (matrix-solve (vandermonde-matrix xs (length xs))
                               (->col-matrix ys))))
> (define xs '(-3 0 3))
> (define ys '(13 3 6))
> (match-define (list c b a) (lagrange-polynomial xs ys))
> (plot (list (function (λ (x) (+ c (* b x) (* a x x))) -4 4)
              (points (map list xs ys))))

注意,上面的例子是在无类型的 Racket 中。

这个函数是用 array-axis-expand 来定义的。

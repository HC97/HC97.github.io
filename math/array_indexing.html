6.11 索引和切分

procedure

(array-ref arr js) → A

  arr : (Array A)
  js : In-Indexes
返回 arr 中位于 js 位置的元素。 如果 js 中的任何索引是负数或者不小于其相应的轴长, array-ref 将引发一个错误。

procedure

(array-set! arr js value) → Void

  arr : (Settable-Array A)
  js : In-Indexes
  value : A
将 arr 中位于 js 位置的元素设置为 value 。 如果 js 中的任何索引是负数或者不小于其相应的轴长, array-set! 会引发一个错误。

procedure

(array-indexes-ref arr idxs) → (Array A)

  arr : (Array A)
  idxs : (Array In-Indexes)
高级别的解释: 在一个新的数组中返回 arr 中的多个元素。

低级别的解释:返回一个与 idxs 形状相同的数组, 其元素是使用 idxs 中的索引从 arr 中 array-ref 。

Examples:
> (define arr (array #[#[1 2] #[10 20]]))
> (define idxs (array #['#(0 0) '#(1 1)]))
> (array-indexes-ref arr idxs)

procedure

(array-indexes-set! arr idxs vals) → Void

  arr : (Settable-Array A)
  idxs : (Array In-Indexes)
  vals : (Array A)
以与 array-indexes-ref 相同的方式对 arr 进行索引,但会改变元素。 如果 idxs 和 vals 没有相同的形状,它们会先被 broadcast 。

Examples:
> (define arr (mutable-array #[#[1 2] #[10 20]]))
> (define idxs (array #['#(0 0) '#(1 1)]))
> (array-indexes-set! arr idxs (array -1))
> arr

当 idxs 中的两个索引相同时,该索引上的元素会以某种未指定的顺序被变异一次以上。

procedure

(array-slice-ref arr specs) → (Array A)

  arr : (Array A)
  specs : (Listof Slice-Spec)
根据切片规范列表 specs 返回 arr 的转换。 讨论和例子见 切片 。

procedure

(array-slice-set! arr specs vals) → Void

  arr : (Settable-Array A)
  specs : (Listof Slice-Spec)
  vals : (Array A)
像 array-indexes-set!, 但是对于 slice 规范来说。相当于
(let ([idxs  (array-slice-ref (indexes-array (array-shape arr)) specs)])
  (array-indexes-set! arr idxs vals))

Examples:
> (define arr (array->mutable-array (axis-index-array #(5 5) 1)))
> (array-slice-set! arr (list (:: 1 #f 2) (::)) (array 1))
> arr

syntax

Slice-Spec

分片规范的类型。目前定义为

(U (Sequenceof Integer) Slice Slice-Dots Integer Slice-New-Axis)

一个 (Sequenceof Integer) 分片规范导致 array-slice-ref 从一个轴上挑选行。 一个 Integer 规范使 array-slice-ref 通过用一个轴的行来替换它来移除一个轴。

请看 切片 的扩展例子。

array-slice-ref 将 Slice 解释为类似于 in-range 的序列对象。 当 start 或 end 为 #f 时,它被解释为一个与轴长有关的端点。

procedure

(slice->range-values s dk) → (Values Fixnum Fixnum Fixnum)

  s : Slice
  dk : Index
给定一个 slice s 和一个轴长 dk, 返回 in-range 的参数, 这些参数将产生一个等价的 slice 规范。

这被 array-slice-ref 内部使用,以解释一个 Slice 对象为一连串的索引。

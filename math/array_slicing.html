6.4 切片

一个常见的数组转换是 slicing: 通过从每个轴上独立抽取行来提取子数组。

切片是通过将 array-slice-ref 或 array-slice-set! 应用于一个数组和一个与数组 axes 相对应的 slice specifications 列表来完成。 有五种类型的分片规范:

  • (Sequenceof Integer): 通过索引从一个轴上挑选行。

  • Slice: 从一个轴上挑选行,就像 in-range 序列一样。

  • Slice-Dots: 保留剩余的相邻轴

  • Integer: 通过用一个轴的行来替换它来移除一个轴。

  • Slice-New-Axis: 插入一个指定长度的轴。

使用 :: 创建 Slice 对象,使用 ::new 创建 Slice-New-Axis 对象。 只有一个 Slice-Dots 对象,即 ::... 。

当对一个有 n 个轴的数组进行切片时,除非切片规格的列表包含 ::..., 否则它必须正好包含 n 个切片规格。

6.4.2 Slice: 以长度感知的方式挑选行

作为一个分片规范, Slice 对象的行为类似于 in-range 返回的序列对象, 但是 start 或 end 可以是 #f 。

如果 start 是 #f, 它将被解释为 step 方向上的第一个有效轴索引。 如果 end 是 #f, 它将被解释为 step 方向上的最后一个有效轴的索引。

最常见的分片可能是 (::), 相当于 (:: #f #f 1) 。 在正 step = 1 的情况下, start 被解释为 0 ， end 被解释为轴的长度。 因此, (::) 从任何轴上挑选所有的行:

6.8 转换

syntax

(Listof* A)

相当于 (U A (Listof A) (Listof (Listof A)) ...) ,如果允许无限的联合的话。 这被用作 list*->array 的参数类型和 array->list* 的返回类型。

syntax

(Vectorof* A)

就像 (Listof* A), 但对向量而言。 参见 vector*->array 和 array->vector* 。

procedure

(list->array lst) → (Array A)

  lst : (Listof A)
(list->array ds lst) → (Array A)
  ds : In-Indexes
  lst : (Listof A)

procedure

(array->list arr) → (Listof A)

  arr : (Array A)
将列表转换为不可变的数组并返回。

list->array 的双参数变体假定 lst 中的元素是以行为主的顺序。

对于 array->list 来说,如果 arr 没有轴或者有多个轴, 在被转换为 list 之前,它(在概念上)会被压扁。

6.8.1 打印

parameter

(array-custom-printer) → 
(All (A) ((Array A)
          Symbol
          Output-Port
          (U Boolean 0 1) -> Any))
(array-custom-printer print-array) → void?
  print-array : 
(All (A) ((Array A)
          Symbol
          Output-Port
          (U Boolean 0 1) -> Any))
一个参数,其值用于打印 Array 的子类型。

procedure

(print-array arr name port mode) → Any

  arr : (Array A)
  name : Symbol
  port : Output-Port
  mode : (U Boolean 0 1)
使用 array 语法打印一个数组,使用 name 代替 'array 作为头部形式。 这个函数在第一次需要 math/array 时被设置为 array-custom-printer 的值。

行为良好的 Array 子类型不会直接调用这个函数来打印自己。 他们会调用当前的 array-custom-printer:
> ((array-custom-printer)
   (array #[0 1 2 3])
   'my-cool-array
   (current-output-port)
   #t)

(my-cool-array #[0 1 2 3])

关于 port 和 mode 参数的含义,请参见 prop:custom-write 。

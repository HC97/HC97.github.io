7.12 运算符规范和比较矩阵

内积空间运算 描述了在内积空间中把矩阵作为向量处理的函数。 本节描述了将矩阵作为 linear operators 处理的函数,或作为从列矩阵到列矩阵的函数。

Wikipedia: Induced norm 在这种情况下,规范是一个算子(即矩阵)对一个列矩阵所能产生的最大的相对幅度变化,其中 "幅度" 是由向量规范定义的。 (关于正式的定义,请见边上链接的维基百科文章)。 用向量规范定义的矩阵规范被称为 induced norms, 或 operator norms 。

procedure

(matrix-op-1norm M) → Nonnegative-Real

  M : (Matrix Number)
由向量规范 matrix-1norm 诱导的 operator norm 。

当 M 是一个列矩阵时, (matrix-op-1norm M) 等同于 (matrix-1norm M) 。

procedure

(matrix-op-2norm M) → Nonnegative-Real

  M : (Matrix Number)
由向量规范 matrix-2norm 引起的 operator norm 。

这个函数目前还没有定义, 因为所需的算法(奇异值分解或 eigendecomposition)还没有在 math/matrix 中实现。

当 M 是一个列矩阵时, (matrix-op-2norm M) 等同于 (matrix-2norm M) 。

procedure

(matrix-op-inf-norm M) → Nonnegative-Real

  M : (Matrix Number)
矢量规范 matrix-inf-norm 所引起的 operator norm 。

当 M 是一个列矩阵时, (matrix-op-inf-norm M) 等同于 (matrix-inf-norm M) 。

procedure

(matrix-basis-cos-angle M0 M1) → Number

  M0 : (Matrix Number)
  M1 : (Matrix Number)
返回 M0 和 M1 所跨越的两个子空间之间的角度的余弦。

这个函数目前还没有定义, 因为所需的算法(奇异值分解或 eigendecomposition)还没有在 math/matrix 中实现。

当 M0 和 M1 是列矩阵时, (matrix-basis-cos-angle M0 M1) 等同于 (matrix-cos-angle M0 M1) 。

procedure

(matrix-basis-angle M0 M1) → Number

  M0 : (Matrix Number)
  M1 : (Matrix Number)
相当于 (acos (matrix-basis-cos-angle M0 M1)) 。

该函数目前未定义,因为 matrix-basis-cos-angle 目前未定义。

parameter

(matrix-error-norm) → ((Matrix Number) -> Nonnegative-Real)

(matrix-error-norm norm) → void?
  norm : ((Matrix Number) -> Nonnegative-Real)
matrix-relative-error 和 matrix-absolute-error 使用的准则。 默认值是 matrix-op-inf-norm 。

除了是一个真正的法线, norm 还应该是 submultiplicative:

(norm (matrix* M0 M1)) <= (* (norm M0) (norm M1))

这个额外的类似三角形的不等式使得证明涉及矩阵乘法的公式的误差界限成为可能。

根据定义,所有的算子规范 (matrix-op-1norm, matrix-op-2norm, matrix-op-inf-norm) 都是 submultiplicative 的,Frobenius 规范 (matrix-2norm) 也是。

procedure

(matrix-absolute-error M R [norm]) → Nonnegative-Real

  M : (Matrix Number)
  R : (Matrix Number)
  norm : ((Matrix Number) -> Nonnegative-Real)
   = (matrix-error-norm)
基本上等同于 (norm (matrix- M R)), 但特别处理非有理数的楞次, 如 +inf.0 和 +nan.0 。

关于这个函数的标量版本,请参见 absolute-error 。

procedure

(matrix-relative-error M R [norm]) → Nonnegative-Real

  M : (Matrix Number)
  R : (Matrix Number)
  norm : ((Matrix Number) -> Nonnegative-Real)
   = (matrix-error-norm)
在规范 norm下,测量 M 相对于真实矩阵 R 的误差。 基本上等同于 (/ (norm (matrix- M R)) (norm R)), 但特别处理非有理数的 flonums, 如 +inf.0 和 +nan.0 ,以及 (norm R) = 0 的情况。

关于这个函数的标量版本,请参见 relative-error 。

procedure

(matrix-zero? M [eps]) → Boolean

  M : (Matrix Number)
  eps : Real = (* 10 epsilon.0)
当 M 非常接近于零矩阵时,返回 #t (默认情况下,在几个ε之内)。 相当于

(<= (matrix-absolute-error M (make-matrix m n 0)) eps)

其中 m n 是 M 的形状。

procedure

(matrix-identity? M [eps]) → Boolean

  M : (Matrix Number)
  eps : Real = (* 10 epsilon.0)
当 M 非常接近于同一矩阵时,返回 #t (默认情况下,在几个ε之内)。 相当于
(and (square-matrix? M)
     (<= (matrix-relative-error M (identity-matrix (square-matrix-size M)))
         eps))

procedure

(matrix-orthonormal? M [eps]) → Boolean

  M : (Matrix Number)
  eps : Real = (* 10 epsilon.0)
当 M 非常接近于正态时,返回 #t ; 也就是说,当 (matrix* M (matrix-hermitian M)) 非常接近于一个身份矩阵时。等同于

(matrix-identity? (matrix* M (matrix-hermitian M)) eps)

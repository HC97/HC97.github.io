9.5 Real Distribution Families

本节中记录的 distribution object 构造函数为最大可能的参数域返回唯一定义的分布。 这通常意味着它们返回的分布比它们的数学对应物所定义的域更大。

例如,那些有比例参数的分布,如 cauchy-dist,logistic-dist, exponential-dist 和 normal-dist, 通常在零比例下没有定义。 然而,在浮点数学中,使用 +inf.0 这样的特殊值在有限时间内模拟极限往往很有用。 因此,当一个标度参数化的族的构造函数接收到 0 时,会返回一个行为类似 Delta-Dist 的分布对象:
> (pdf (normal-dist 1 0) 1)

+inf.0

> (pdf (normal-dist 1 0) 1.0000001)

0.0

此外,负标度也是可以接受的,即使是 exponential-dist, 也会产生一个反映在零左右的正标度分布。

一些参数的边界值会产生非唯一的极限。 有时,这种模糊性可以用必要的属性来解决;例子见 Gamma-Dist 。 当不存在解决方法时,如 (beta-dist 0 0), 它把一个不确定的概率放在 0 值上, 其余的放在 1 上,构造函数返回一个 undefined distribution 。

一些分布对象构造函数在给定特殊值(如 +inf.0 )作为参数时试图返回合理的分布。 先不要指望这些。

许多分布系列,如 Gamma-Dist, 可以在规模或速率(这是规模的倒数)上进行参数化。 在所有这些情况下,由 math/distributions 提供的实现都是以比例为参数的。

9.5.1 贝塔分布

Wikipedia: Beta Distribution.

syntax

Beta-Dist

procedure

(beta-dist alpha beta) → Beta-Dist

  alpha : Real
  beta : Real

procedure

(beta-dist-alpha d) → Flonum

  d : Beta-Dist

procedure

(beta-dist-beta d) → Flonum

  d : Beta-Dist
代表贝塔分布系列,由两个形状参数或假数来设置参数,这两个参数必须是非负的。

Examples:
> (plot (for/list ([α  (in-list '(1 2 3 1/2))]
                   [β  (in-list '(1 3 1 1/2))]
                   [i  (in-naturals)])
          (function (distribution-pdf (beta-dist α β))
                    #:color i #:label (format "Beta(~a,~a)" α β)))
        #:x-min 0 #:x-max 1 #:y-max 4 #:y-label "density")

> (plot (for/list ([α  (in-list '(1 2 3 1/2))]
                   [β  (in-list '(1 3 1 1/2))]
                   [i  (in-naturals)])
          (function (ordered-dist-cdf (beta-dist α β))
                    #:color i #:label (format "Beta(~a,~a)" α β)))
        #:x-min 0 #:x-max 1 #:y-label "probability")

(beta-dist 0 0) 和 (beta-dist +inf.0 +inf.0) 是 undefined distributions 。

当 a = 0 或 b = +inf.0 时,返回的分布就像 (delta-dist 0) 。

当 a = +inf.0 或 b = 0 时,返回的分布就像 (delta-dist 1) 。

9.5.5 伽马分布

Wikipedia: Gamma Distribution.

syntax

Gamma-Dist

procedure

(gamma-dist [shape scale]) → Gamma-Dist

  shape : Real = 1
  scale : Real = 1

procedure

(gamma-dist-shape d) → Flonum

  d : Gamma-Dist

procedure

(gamma-dist-scale d) → Flonum

  d : Gamma-Dist
代表以形状和尺度为参数的伽马分布系列。 shape 参数必须为非负值。

警告: 伽马分布系列通常以形状和 rate 为参数,而比率是比例的倒数。使用以下方法从比率中构建伽马分布

(gamma-dist shape (/ 1.0 rate))

Examples:
> (plot (for/list ([k  (in-list '(1 2 3 9))]
                   [s  (in-list '(2 2 3 1/2))]
                   [i  (in-naturals)])
          (function (distribution-pdf (gamma-dist k s))
                    #:color i #:label (format "Gamma(~a,~a)" k s)))
        #:x-min 0 #:x-max 15 #:y-label "density"
        #:legend-anchor 'top-right)

> (plot (for/list ([k  (in-list '(1 2 3 9))]
                   [s  (in-list '(2 2 3 1/2))]
                   [i  (in-naturals)])
          (function (ordered-dist-cdf (gamma-dist k s))
                    #:color i #:label (format "Gamma(~a,~a)" k s)))
        #:x-min 0 #:x-max 15 #:y-label "probability"
        #:legend-anchor 'bottom-right)

shape = 0 的 gamma 分布的 cdf 在 x = 0 时可以返回 0.0 或 1.0 , 这取决于是先对 scale 进行双极限还是先对 x 进行双极限。 无论采取怎样的极限,对于 x > 0 的 cdf 必须返回 1.0 。 因为 cdf 是右连续的,唯一正确的选择是
> (cdf (gamma-dist 0 1) 0)

1.0

因此, shape = 0 的伽马分布的行为与 (delta-dist 0) 类似。

  min : Real = 0

                   [b  (in-list '(0 1 3))]
                   [m  (in-list '(-2 0 2))]
                   [i  (in-naturals)])
          (function (ordered-dist-cdf (triangle-dist a b m)) #:color i
                    #:label (format "Triangle(~a,~a,~a)" a b m)))
        #:x-min -3.5 #:x-max 3.5 #:y-label "probability")

(triangle-dist c c c) 对任何实数 c 的行为就像一个以 c 为中心的支持有限的 delta 分布。

9.5.9 截断分布

syntax

Truncated-Dist

procedure

(truncated-dist d) → Truncated-Dist

  d : Real-Dist
(truncated-dist d max) → Truncated-Dist
  d : Real-Dist
  max : Real
(truncated-dist d min max) → Truncated-Dist
  d : Real-Dist
  min : Real
  max : Real

procedure

(truncated-dist-original t) → Real-Dist

  t : Truncated-Dist

procedure

(truncated-dist-min t) → Flonum

  t : Truncated-Dist

procedure

(truncated-dist-max t) → Flonum

  t : Truncated-Dist
代表像 d 一样的分布,但在 x < min 和 x > max 时密度为零。 区间 [min, max] 的概率被重新规范化为 1。

(truncated-dist d) 等同于 (truncated-dist d -inf.0 +inf.0) 。 (truncated-dist d max) 等同于 (truncated-dist d -inf.0 max) 。 如果 min > max, 在构造分布对象之前,它们会被交换。

通过将截断分布的 inverse cdf 应用于均匀样本来获取样本。

Examples:
> (define d (normal-dist))
> (define t (truncated-dist d -2 1))
> t

(truncated-dist (normal-dist 0.0 1.0) -2.0 1.0)

> (plot (list (function (distribution-pdf d) #:label "N(0,1)" #:color 0)
              (function (distribution-pdf t) #:label "T(N(0,1),-2,1)"))
        #:x-min -3.5 #:x-max 3.5 #:y-label "density")

> (plot (list (function (ordered-dist-cdf d) #:label "N(0,1)" #:color 0)
              (function (ordered-dist-cdf t) #:label "T(N(0,1),-2,1)"))
        #:x-min -3.5 #:x-max 3.5 #:y-label "probability")

        #:x-min -3.5 #:x-max 3.5 #:y-label "density")

> (plot (for/list ([a  (in-list '(-3 -1 -2))]
                   [b  (in-list '(0 1 3))]
                   [i  (in-naturals)])
          (function (ordered-dist-cdf (uniform-dist a b)) #:color i
                    #:label (format "Uniform(~a,~a)" a b)))
        #:x-min -3.5 #:x-max 3.5 #:y-label "probability")

(uniform-dist x x) 对于任何实数 x , 其行为就像一个以 x 为中心的支持有限的 delta 分布。

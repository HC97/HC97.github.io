5.4 缓存表

 (require web-server/private/cache-table)
  package: web-server-lib
该模块提供了一组缓存哈希表函数。

procedure

(make-cache-table) → cache-table?

构建一个缓存表。

procedure

(cache-table-lookup! ct id mk) → any/c

  ct : cache-table?
  id : symbol?
  mk : (-> any/c)
在 ct 中查找 id 。 如果它不存在,则调用 mk 来构建值并将其添加到 ct 中。

procedure

(cache-table-clear! ct [entry-ids finalize]) → void?

  ct : cache-table?
  entry-ids : (or/c false/c (listof symbol?)) = #f
  finalize : (-> any/c void?) = void
如果 entry-ids 是 #f, 则清除 ct 中的所有条目。 否则,只清除键值为 entry-ids 的条目。 在每个条目被清除之前,会对其调用程序 finalize 。

Changed in version 1.3 of package web-server-lib: 增加了条目键列表的可选参数。
Changed in version 1.3: 增加了 finalizer 过程的可选参数。

procedure

(cache-table? v) → boolean?

  v : any/c
确定 v 是否是一个缓存表。

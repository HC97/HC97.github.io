5.7 URL Param

 (require web-server/private/url-param)
  package: web-server-lib
Web Server 需要对 URL 中的信息进行编码。 如果这些数据被存储在查询字符串中,那么它将被那些向这些 URL 发出 GET 请求的浏览器以更多的查询数据来覆盖。 因此,它必须在 URL 参数中进行编码。 这个模块提供了帮助这个过程的函数。

procedure

(insert-param u k v) → url?

  u : url?
  k : string?
  v : string?
将 k 与 u 的最终 URL 参数中的 v 联系起来,覆盖 k 的任何当前绑定。

procedure

(extract-param u k) → (or/c string? false/c)

  u : url?
  k : string?
如果有的话,提取 u 的最终 URL 参数中与 k 相关的字符串,否则返回 #f 。

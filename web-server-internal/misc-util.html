5.9 其他实用程序

 (require web-server/private/util) package: web-server-lib

procedure

(bytes-ci=? b1 b2) → boolean?

  b1 : bytes?
  b2 : bytes?
对两个字节进行不敏感的比较。

procedure

(url-replace-path proc u) → url?

  proc : ((listof path/param?) . -> . (listof path/param?))
  u : url?
将 u 的 URL 路径替换为原路径的 proc 。

procedure

(url-path->string url-path) → string?

  url-path : (listof path/param?)
将 url-path 格式化为一个以 "/" 为分隔符的字符串,没有参数。

procedure

(explode-path* p) → (listof path-piece?)

  p : path-string?
与 normalize-path 类似,但不解析符号链接。

procedure

(path-without-base base p) → (listof path-piece?)

  base : path-string?
  p : path-string?
假设 base 是 p 的前缀,则以列表形式返回 base 之后的部分 p 。

procedure

(directory-part p) → path?

  p : path-string?
返回 p 的目录部分,如果是相对的,返回 (current-directory) 。

procedure

(build-path-unless-absolute base p) → path?

  base : path-string?
  p : path-string?
除非 p 是绝对的,否则将 base 加到 p 上。

procedure

(network-error s fmt v ...) → void

  s : symbol?
  fmt : string?
  v : any/c
像 error 一样,但抛出一个 exn:fail:network 。

procedure

(exn->string exn) → string?

  exn : (or/c exn? any/c)
用 (error-display-handler) 将 exn 格式化为字符串。

procedure

(read/bytes bstr) → printable/c

  bstr : bytes?
使用 read 从 bstr 中提取一个值。

procedure

(write/bytes v) → bytes?

  v : printable/c
使用 write 将 v 打印成一个字节字符串。

value

path-piece? : contract?

相当于 (or/c path-string? (symbols 'up 'same)) 。

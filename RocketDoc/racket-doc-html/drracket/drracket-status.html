3.4 状态信息

DrRacket 窗口包含一些不同的状态信息,试图以相对不显眼的方式揭示 DrRacket 正在做什么。

在DrRacket 窗口的右下角,有许多信息。 从左到右:

  • DrRacket 当前使用的语言名称。 这也是一个弹出菜单,你可以用它从最近使用的语言列表中改变语言。

  • 再往右是一些数字,显示插入点的行和列或字符偏移量(当你下次输入时,字符将出现在哪里)。

  • 当前 DrRacket 内部使用的堆大小。 这个数字不经常更新;要想看到准确的数字,请点击它,这也会触发一次大的垃圾回收。

  • 一个小方块,除非 DrRacket 正在进行垃圾收集,否则它是空白的,当 DrRacket 正在进行垃圾收集时,它是一个回收图标。

  • 一个狭长的空白处或一对小括号。 如果 DrRacket 正在确定颜色和定义中匹配的小括号的位置,那将是一对小括号,如果不是,那将是空白。

  • 一个小人,表示 DrRacket 是否在交互窗口中运行程序。 (当 DrRacket 正在运行你的程序时,这个人就会运行)。

  • 一个表示 DrRacket 在线扩展状态的圆圈(仅在使用 "从源代码确定语言" 时可用)。 如果是红色,说明程序中存在错误,或者在线扩展已被禁用。 如果是蓝色,DrRacket 正在努力扩展你的程序。 如果是(深或浅)紫色,DrRacket 正在处理扩展的结果,计算箭头的位置,以及重命名任何一个变量对其他变量的影响。 如果是绿色,说明 DrRacket 的在线扩展已经完成。 如果是一个带刺的绿球(而不是一个圆形的绿球),那么 DrRacket 将使用在线扩展的结果来加快 Run 按钮的速度。 也就是说,当绿色尖球出现时,DrRacket 不会对你的程序进行扩展或编译,而只是直接运行它。

1.2 选择一种语言

DrRacket 支持 Racket 的多种方言,以及一些非 Racket 语言。你可以通过以下两种方式之一指定语言:

  • 选择 Use the language declared in the source 选项(通过 Language|Choose Language... 菜单项), 然后将特定的语言作为程序的一部分,通常在定义窗口内容中以 #lang 开头。

  • 选择 Language|Choose Language... 菜单项,并选择一种特定的语言。 更改语言后,点击 Run 以重置交互窗口中的语言。 DrRacket 主窗口的左下角也有一个快捷菜单项,用于选择以前选择的语言。

使用程序源码中声明的语言是推荐模式,在 在源代码中声明的语言 中会进一步说明。

Language|Choose Language... 对话框包含一个 Show Details 按钮,用于配置所选语言的某些细节。 每当选择的选项与默认的语言规范不一致时,对话框顶部的语言选择控件旁边就会出现一个 Custom 指示器。

有关 DrRacket 支持的语言的更多信息,请参见 语言 (本手册后面)。

1.4 搜索

DrRacket 的搜索和替换功能是交互式的,与 Safari、Firefox 和 Emacs 中的搜索和替换功能类似,但有一些不同。

要开始搜索,首先从 Edit 菜单中选择 Find 菜单项。 这将在 DrRacket 窗口的底部打开一个小编辑器。在这里开始输入,随着你的输入, 你要搜索的字符串的所有出现都会在编辑器窗口中被圈出来。 观察搜索窗口旁边的空间,看看你的文件中出现了多少个搜索字符串。 当你准备好了,你可以使用 Find Again 菜单项来跳到搜索字符串的第一次出现。 这将在其中一个圆圈中着色。 第二次使用 Find Again 来跳到下一个出现的地方。

如果你点击回到定义窗口, Find Again 菜单项,DrRacket 将把选择移动到搜索字符串的下一个出现处。

DrRacket 还支持一种模式,即在搜索编辑器中打字会直接进入搜索字符串的下一次出现, 而无需选择 Find Again 菜单项。 在偏好对话框中的 Editing 部分和 General Editing 部分有一个复选框, 标有 Search using anchors 。 选中后,DrRacket 会显示一个小红点和一条红线,表示 search anchor 的位置。 启用搜索锚后,在搜索窗口中输入的内容会跳到锚后搜索字符串的第一次出现。

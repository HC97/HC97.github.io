1.12 Minimal Library

 (require syntax/parse/pre) package: base

syntax/parse/pre 库对于访问大多数语法解析功能非常有用,同时最大限度地减少对库的依赖。 它提供了大部分的 syntax/parse, 但省略了这些绑定:

expr/c
pattern-expander?
prop:syntax-class
pattern-expander
prop:pattern-expander
syntax-local-syntax-parse-pattern-introduce

此外,所提供的 static 的变体是一个不同的绑定,缺乏明确的契约检查。

16 内部定义的上下文帮助器

 (require syntax/intdef) package: base

Added in version 6.3.0.4 of package base.

procedure

(internal-definition-context-track intdef-ctx    
  stx) → syntax?
  intdef-ctx : internal-definition-context?
  stx : syntax?
调整 stx 的 syntax properties , 以记录 stx 的部分内容是通过 intdef-ctx 展开的。

具体来说,由 (internal-definition-context-binding-identifiers intdef-ctx) 产生的标识符被添加到 stx 的 'disappeared-binding 属性中。

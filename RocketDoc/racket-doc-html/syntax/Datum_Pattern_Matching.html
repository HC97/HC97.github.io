3 Datum Pattern Matching

 (require syntax/datum) package: base
syntax/datum 库提供了实现 syntax-case 的模式和模板语言的形式,但用于匹配和构造基准值而不是语法。

对于大多数模式匹配的目的, racket/match 是比 syntax/datum 更好的选择。 syntax/datum 库的用处主要在于它的模板支持(即 datum) , 在较小的程度上,它与 syntax-case 模式直接对应。

syntax

(datum-case datum-expr (literal-id ...)
  clause ...)

syntax

(datum template)

与 syntax-case 和 syntax 一样, 但是 datum-case 中的 datum-expr 应该产生一个 datum (即普通的 S 表达式), 而不是在 clause 中匹配的 syntax object ， datum 也同样产生一个 datum。 在 datum-case (或 syntax-case,见下文) 的每个 clause 中绑定的模式变量可以通过 datum 而不是 syntax 访问。 当一个 literal-id 出现在 clause 的模式中时, 它与相应的符号相匹配(使用 eq?)。

使用 datum-case 和 datum 类似于使用 datum->syntax 将输入转换为 syntax-case ,然后用 syntax->datum 包装每次使用的 syntax , 但是 datum-case 和 datum 不会创建中间的 syntax 对象, 也不会破坏 datum-expr 的 S-expression 结构中现有的 syntax 对象。

Example:
> (datum-case '(1 "x" -> y) (->)
    [(a ... -> b) (datum (b (+ a) ...))])

'(y (+ 1) (+ "x"))

datum 形式还与 syntax pattern variables 合作, 比如那些由 syntax-case 绑定的变量和由 syntax-parse 绑定的 attributes (关于属性的更多信息,请参见 模式变量和属性 。 作为一个结果, datum 提供了一个方便的方法来获取与深度为 1 的语法模式变量绑定的语法对象列表。 例如,下面的表达式是等价的,只是 datum 版本避免了创建和消除一个多余的语法对象包装器:

> (with-syntax ([(x ...) #'(a b c)])
    (syntax->list #'(x ...)))

'(#<syntax:eval:3:0 a> #<syntax:eval:3:0 b> #<syntax:eval:3:0 c>)

> (with-syntax ([(x ...) #'(a b c)])
    (datum (x ...)))

'(#<syntax:eval:4:0 a> #<syntax:eval:4:0 b> #<syntax:eval:4:0 c>)

一个模板也可以使用多个语法或数据化模式变量和数据化常量,并且可以使用 ~@ 和 ~? 模板形式:

> (with-syntax ([(x ...) #'(a b c)])
    (with-datum ([(y ...) (list 1 2 3)])
      (datum ([x -> y] ...))))

'((#<syntax:eval:5:0 a> -> 1)

  (#<syntax:eval:5:0 b> -> 2)

  (#<syntax:eval:5:0 c> -> 3))

> (with-syntax ([(x ...) #'(a b c)])
    (with-datum ([(y ...) (list 1 2 3)])
      (datum ((~@ x y) ...))))

'(#<syntax:eval:6:0 a> 1 #<syntax:eval:6:0 b> 2 #<syntax:eval:6:0 c> 3)

关于 ~? 与 datum 的例子,见 属性和 datum 。

如果在语法模板中使用了 datum 变量,会产生一个编译时错误。

Changed in version 7.8.0.9 of package base: 改变了 datum , 以便与 syntax-case, syntax-parse 等合作。

syntax

(with-datum ([pattern datum-expr] ...)
  body ...+)
类似于 with-syntax, 但针对 datum-case 和 datum ， 而不是 syntax-case 和 syntax 。

Example:
> (with-datum ([(a ...) '(1 2 3)]
               [(b ...) '("x" "y" "z")])
    (datum ((a b) ...)))

'((1 "x") (2 "y") (3 "z"))

syntax

(define/with-datum pattern datum-expr)

with-datum 的定义形式。 类似于 define/with-syntax, 但针对 datum-case 和 datum 。

Examples:
> (define/with-datum ((x y) ...)
    '((a 1) (b 2) (c 3)))
> (datum ((x ...)
          (y ...)))

'((a b c) (1 2 3))

Added in version 8.2.0.8 of package base.

syntax

(quasidatum template)

syntax

(undatum expr)

syntax

(undatum-splicing expr)

类似于 quasisyntax, unsyntax 和 unsyntax-splicing 。

Example:
> (with-datum ([(a ...) '(1 2 3)])
    (quasidatum ((undatum (- 1 1)) a ... (undatum (+ 2 2)))))

'(0 1 2 3 4)

2.5 配置

parameter

(current-font-size) → exact-nonnegative-integer?

(current-font-size n) → void?
  n : exact-nonnegative-integer?
决定 t,para 等所使用的字体大小的参数。默认大小为 32 。

parameter

(current-main-font) → text-style/c

(current-main-font style) → void?
  style : text-style/c
决定 t,para 等使用的字体的参数。 默认值是特定平台的;可能的初始值包括 'swiss, "Verdana" 和 "Gill Sans" 。

parameter

(current-tt-font) → text-style/c

(current-tt-font style) → void?
  style : text-style/c
决定 tt 使用的字体的参数。 默认值为 '(bold . modern) 。

Added in version 1.9 of package slideshow-lib.

parameter

(current-tt-font-size) → (or/c #f exact-nonnegative-integer?)

(current-tt-font-size size) → void?
  size : (or/c #f exact-nonnegative-integer?)
决定 tt 所使用的字体大小的参数。 默认值是 #f, 它使 tt 使用 current-font-size 。

Added in version 1.9 of package slideshow-lib.

parameter

(current-line-sep) → exact-nonnegative-integer?

(current-line-sep n) → void?
  n : exact-nonnegative-integer?
控制 para, item 和 subitem 的行间空间大小的参数。

parameter

(current-para-width) → exact-nonnegative-integer?

(current-para-width n) → void?
  n : exact-nonnegative-integer?

procedure

(get-current-para-width [#:aspect aspect])

 → (parameter/c exact-nonnegative-integer?)
  aspect : aspect? = #f
控制 para, item 和 subitem 创建的图片的宽度的参数。 current-para-width 的值与 (get-current-para-width) 相同。

Changed in version 1.5 of package slideshow-lib: 增加了 get-current-para-width 。

parameter

(current-title-color) → (or/c string? (is-a?/c color%))

(current-title-color color) → void?
  color : (or/c string? (is-a?/c color%))
默认的 current-titlet 用来给标题着色的参数。 默认为 "black" 。

parameter

(current-slide-assembler) → 
((or/c string? #f)
 exact-nonnegative-integer?
 pict?
 . -> .
 pict?)
(current-slide-assembler proc) → void?
  proc : 
((or/c string? #f)
 exact-nonnegative-integer?
 pict?
 . -> .
 pict?)
参数,其值是一个用于将幻灯片内容组合成一个图片的函数。 组装函数需要:一个标题、一个垂直空间量和一个幻灯片内容的图片(不算标题)。

默认的装配函数使用 titlet 将标题字符串(如果有的话)变成一个图片, 并使用垂直空间将标题图片放在幻灯片内容的上方。 参见 current-titlet 和 set-title-h! 。

装配器的结果是与客户区 ct-superimpose ,但所产生的 pict 可能也会在客户区之外画出屏幕的边缘。

幻灯片装配程序 not 负责在幻灯片上添加页码;这项工作属于浏览者。 参见 current-page-number-font, current-page-number-color 和 set-page-numbers-visible! 。

parameter

(current-titlet) → (string? . -> . pict?)

(current-titlet proc) → void?
  proc : (string? . -> . pict?)
用于配置 titlet 的参数。默认的是

(lambda (s)
  (colorize (text s (current-main-font) 40)
            (current-title-color)))

如果这个参数被改变,导致结果是一个不同的高度, 那么应该调用 set-title-h! 来更新 title-h, titleless-page 等产生的值。

procedure

(set-margin! amt) → void?

  amt : real?
改变包围客户区域的边距。请参阅 margin 。

procedure

(set-title-h! amt) → void?

  amt : real?
改变标题的预期高度,调整 title-h, client-h, full-page 和 titleless-page 。

procedure

(make-slide-inset left-inset    
  top-inset    
  right-inset    
  bottom-inset) → slide-inset?
  left-inset : exact-nonnegative-integer?
  top-inset : exact-nonnegative-integer?
  right-inset : exact-nonnegative-integer?
  bottom-inset : exact-nonnegative-integer?
创建一个幻灯片嵌套,它描述了在每一侧的幻灯片上嵌套查看器的像素数。

procedure

(slide-inset? v) → boolean?

  v : any/c
如果 v 是由 make-slide-inset 创建的幻灯片嵌入,则返回 #t ,否则返回 #f 。

2.4 常量和布局变量

procedure

(aspect? v) → boolean?

  v : any/c
如果 v 是 'fullscreen,'widescreen 或 #f, 返回 #t ,否则返回 #f 。

符号化的 v 选择了一个特定的宽幅, 而 #f 作为一个宽幅则对应于用户通过 --widescreen 或 --fullscreen 标志选择的宽幅。

参见 全屏与宽屏的长宽比 。

Added in version 1.5 of package slideshow-lib.

value

gap-size : 24

一种常用于布局的宽度。

parameter

(current-gap-size) → real?

(current-gap-size sep-gap-size) → void?
  sep-gap-size : real?
一个参数,其值是用于 slide 项目之间的分隔的宽度, item 的子弹的大小和间距, 'tall 模式下幻灯片标题和内容之间的空间,等等。默认值是 gap-size 。

value

bullet : pict?

item 默认使用的填充子弹。

它要么是 (t "•"), 如果该字符在 t 使用的字体中可用, 要么它使用类似于 o-bullet 的实现,但不是空心的(使用 disk, 而不是 circle)。

value

o-bullet : pict?

subitem 默认使用的是空心子弹。

它的实现是:
(baseless
 (cc-superimpose
  (circle (/ gap-size 2))
  (blank 0 gap-size)))

syntax

client-w

procedure

(get-client-w [#:aspect aspect]) → exact-nonnegative-integer?

  aspect : aspect? = #f
产生显示区域的宽度,减去给定 aspect 的 margin , 其中 client-w 等同于 (get-client-w) 。 如果通过 set-margin! 调整边距,结果就会改变。

Changed in version 1.5 of package slideshow-lib: 增加了 get-client-w 。

syntax

client-h

procedure

(get-client-h [#:aspect aspect]) → exact-nonnegative-integer?

  aspect : aspect? = #f
产生显示区域的高度,减去给定 aspect 的 margin , 其中 client-h 等同于 (get-client-h) 。 如果通过 set-margin! 调整边距,结果会改变。

Changed in version 1.5 of package slideshow-lib: 增加了 get-client-h 。

syntax

full-page

procedure

(get-full-page [#:aspect aspect]) → pict?

  aspect : aspect? = #f
产生一个与客户区大小相同的空图,就像 (blank client-w client-h) 。 full-page 形式等同于 (get-full-page) 。

Changed in version 1.5 of package slideshow-lib: 增加了 get-full-page 。

syntax

titleless-page

procedure

(get-titleless-page [#:aspect aspect]) → pict?

  aspect : aspect? = #f
在 'top 布局模式下,产生一个与客户端区域减去标题区域大小相同的空图片, 这就像 (blank client-w (- client-h title-h (* 2 gap-size))) 。 titleless-page 形式等同于 (get-titleless-page) 。

Changed in version 1.5 of package slideshow-lib: 增加了 get-titleless-page 。

syntax

margin

产生一个与当前边距相对应的数字,它围绕着幻灯片的每一面。 幻灯片的客户端区域相当于显示区域(1024 x 768 或 1360 x 766)减去每边的这个边距。 默认的边距是 20 。

边距可以通过 set-margin! 来调整。

syntax

title-h

产生一个数字,对应于由 titlet 创建的标题的高度。

如果 titlet 通过 current-titlet 参数被改变, 标题的高度应该通过 set-title-h! 来更新。

value

printing? : boolean?

如果幻灯片是为打印输出而生成的,该值为 #t ;如果是正常的屏幕显示,则为 #f 。 打印模式通常是通过 --print 或 --ps 命令行标志触发的。

value

condense? : boolean?

如果幻灯片是以浓缩模式生成的,数值为 #t , #f 为正常模式。 浓缩模式通常是通过 --condense 命令行标志触发的。

2.8 幻灯片到图片

 (require slideshow/slides-to-picts) package: slideshow-lib

procedure

(get-slides-as-picts path    
  width    
  height    
  condense?    
  [stop-after    
  #:aspect aspect]) → (listof pict?)
  path : path-string?
  width : real?
  height : real?
  condense? : any/c
  stop-after : (or/c #f exact-nonnegative-integer?) = #f
  aspect : (or/c 'fullscreen 'widescreen) = inferred-aspect
在一个新的命名空间中执行由 path 指示的 Slideshow 程序,并返回一个幻灯片的图片列表。 每张图片都有给定的 width 和 height, condense? 决定 Slideshow 程序是否以 condense 模式执行。

aspect 参数表示哪种幻灯片应该占满 width 和 height 的区域, 而其他方面的幻灯片将被缩放以适应; aspect 的默认值是由 width 和 height 推断出来的。 aspect 参数还设置了加载 path 时的默认方面。

如果 stop-after 不是 #f, 那么在 stop-after 的幻灯片被转换为图片后,列表会被截断。

Changed in version 1.8 of package slideshow-lib: 增加了 #:aspect 。

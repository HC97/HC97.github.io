Slideshow: 图和演示工具

Matthew Flatt
and Robert Bruce Findler


吴华均 译

Slideshow 是一个用于创建演示幻灯片的库。 与 Powerpoint 不同,Slideshow 没有提供所见即所得的界面来构建幻灯片。 相反,像 Beamer 一样,演示文稿是由一个程序生成的。

要开始使用,请运行 slideshow 的可执行程序,并点击 Run Tutorial 的链接。

要了解更多关于 Slideshow 的信息,请参见 "Slideshow: 功能性演示文稿" [Findler06] 。

 #lang slideshow package: slideshow-lib
手册中定义的大部分绑定都是由 slideshow 语言提供的, 它也重新输出了所有的 racket , 除了 printable<%> (由于向后兼容的问题)和所有的 pict 。

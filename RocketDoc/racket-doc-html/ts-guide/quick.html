1 快速入门

给定一个用 racket 语言编写的模块,使用 Typed Racket 需要以下步骤:

  1. 将语言改为 typed/racket 。

  2. 将 (require mod) 的用途改为 (require typed/mod) 。

  3. 用类型来注释结构定义和顶层定义。

然后,当程序运行时,在执行之前会自动进行类型检查,任何类型错误都会被报告。 如果有任何类型错误,程序将不会运行。

下面是一个用 racket 语言编写的例子程序:

#lang racket
(struct pt (x y))
 
; distance : pt pt -> real
(define (distance p1 p2)
  (sqrt (+ (sqr (- (pt-x p2) (pt-x p1)))
           (sqr (- (pt-y p2) (pt-y p1))))))

下面是同样的程序,在 typed/racket 中:

#lang typed/racket
(struct pt ([x : Real] [y : Real]))
 
(: distance (-> pt pt Real))
(define (distance p1 p2)
  (sqrt (+ (sqr (- (pt-x p2) (pt-x p1)))
           (sqr (- (pt-y p2) (pt-y p1))))))

1.1 从 Racket REPL 使用 Racket 语言

我们可以在 Racket REPL 中使用 Typed Racket。 要做到这一点,请用以下命令行启动 Racket:
  racket -I typed/racket

4 转换 yacc 或 bison 语法

 (require parser-tools/yacc-to-scheme)
  package: parser-tools-lib

procedure

(trans file) → any/c

  file : path-string?
从 file 中读取 C 语言 yacc/bison 语法, 并产生一个代表 Racket 解析器的 s 表达式,供 parser 使用。

这个函数的目的是协助手动转换语法以用于 parser, 而不是作为一个完全自动的转换工具。它并不完全健壮。 例如,如果原始语法中的 C 动作有嵌套块,该工具将失败。

注释的例子在 "parser-tools" 集合的 "examples" 子目录中。

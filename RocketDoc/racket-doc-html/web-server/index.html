Racket 中的网络应用

Jay McCarthy


吴华均 译

本手册介绍了用于构建 Web 应用程序的 Racket 库。

运行 Web Servlets 描述了如何运行你编写的 Servlets。

有状态的小程序 和 无状态 Servlet 描述了两种编写 Web 应用程序的方法。 有状态的小程序 使用整个 Racket 语言,但它们的延续被存储在 Web 服务器的内存中。 无状态 Servlet 使用稍有限制的 Racket 语言,但它们的延续可以由 Web 客户端或 Web 服务器的磁盘上存储。 如果可以的话,你想使用无状态 Servlets,因为它的可扩展性得到了改善。

HTTP: 超文本传输协议 部分描述了用于操作 HTTP 请求和创建 HTTP 响应的常用库函数。 特别是,这一节涵盖了 cookies、认证和请求绑定。

最后五个部分 (基于 URL 的调度, Formlets: 功能性表单抽象, 模板: 视图的分离, 页: 常用模式的简称 和 测试 Servlets) 涵盖了便于创建典型 Web 应用程序的实用库。

本手册最后有一个常见问题部分: 故障排除和提示 。

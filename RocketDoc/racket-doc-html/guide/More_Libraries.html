22 更多的库

本指南只涵盖记录在 The Racket Reference 的 Racket 语言和库。Racket 分配包括许多额外的库。

22.1 图形和图形用户界面

Racket 为图形和图形用户界面（GUI）提供许多库：

  • racket/draw 提供了基本的绘图工具，包括绘制背景如位图（bitmap）和 PostScript 文件。 参见 Racket 绘图工具箱获取更多信息。

  • racket/gui库提供的 GUI 部件， 如窗口（window）、按钮（button）、复选框（checkboxe）和文本字段（text field）。 该库还包括一个复杂的、可扩展的文本编辑器。 参见 Racket 图形界面工具箱 获取更多信息。

  • pict 库一个在 racket/draw 之上的更多功能的抽象层。 这一层对用 Slideshow 创建幻灯片演示特别有用， 但它对于为 Scribble 文档或其他绘图任务创建图片也很有用。 用 pict 库创建的图片可以被渲染到任何绘图环境中。 参见 theSlideshow: Figure and Presentation Toolsdocumentation 获取更多信息。

  • 2htdp/image 库与 pict 相似。 它对教学使用来说更加精简,但对屏幕和位图绘图来说也略显特殊。

    参见 2htdp/image 以了解更多信息。

  • sgl 库为三维（3-D）图形提供 OpenGL。 渲染 OpenGL 的上下文可以是用 racket/gui创建的的一个窗口或位图。 参见 the SGL documentation 获取更多信息。

22.2 Web 服务器

theWeb Applications in Racketdocumentation 介绍了 Racket 的 Web 服务器，它支持 Racket 的 servlet 实现。

22.3 使用外部库

Racket 外国接口描述使用 Racket 访问 C 程序的工具。

22.4 其它更多的库

Racket Documentation 列出了其他库的文档,包括作为包安装的库。 运行 raco docs 可以找到安装在你的系统上的、特定于你的用户账户的库的文档。

在 https://pkgs.racket-lang.org 的 Racket Package Catalog 提供了更多由 Racketeers 贡献的可下载软件包。 online Racket documentation 包括该目录中软件包的文档,每天更新。关于软件包的更多信息, 请参见 Racket 中的包管理 。

PLaneT 提供的包是用旧的包系统开发的。 Racket 包应该使用较新的系统,而不是。

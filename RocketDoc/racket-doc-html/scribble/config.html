6.11 扩展和配置 Scribble 输出

有时,Scribble 的基元和内置样式不足以产生你需要的输出。 你需要扩展或配置 Scribble 的情况可以分为两类:

  • 你可能需要进入 CSS 或 Latex 的后端 "语言",以创建一个特定的输出效果。 对于这种扩展,你多半会给style附加一个 css-addition 或 tex-addition style property ,其中 addition 实现了样式名称。 这种扩展将在 实现样式 中描述。

  • 你可能需要制作一个页面布局与 Racket 文档风格不同的文档。 对于这种配置,你可以运行 scribble 命令行工具并提供像 --prefix 或 ++style 这样的标志, 或者你可以将 html-defaults 或 latex-defaults style property 关联到主文档的样式。 这种配置将在 配置输出 中描述。

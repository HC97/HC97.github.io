Scribble: Racket 文档工具

Matthew Flatt
and Eli Barzilay

Scribble 是一个工具集合,用于以 HTML 或 PDF (通过 Latex )形式创建散文文件–论文、书籍、库文档等。 更广泛地说,Scribble 可以帮助你编写富含文本内容的程序,无论这些内容是要排版的散文还是要以编程方式生成的任何其他形式的文本。

这份文件本身就是用 Scribble 写的。你可以在 https://github.com/racket/scribble/tree/master/scribble-doc/scribblings/scribble, 从 "scribble.scrbl" 文件开始看到它的源代码。

6.11.3 基本 CSS 风格类

下面是 "demo.scrbl" 的效果图, 展示了 scribble/base 表单和函数所使用的所有 CSS 样式类:

  • S1 All-Styles Document, Title in “H2” 显示了没有搜索框的单页渲染中的默认样式。

  • M1 All-Styles Document, Title in “H2” 在没有搜索框的多页面渲染中显示默认样式。

  • S2 All-Styles Document, Title in “H2” 显示了当前手动样式的调整, 在一个有搜索框的单页渲染中。

  • M2 All-Styles Document, Title in “H2” 显示了当前手动样式在多页渲染中的调整, 并带有搜索框。

风格类:

 

maincolumn

 

主栏中所有内容的外包装。

 

main

 

主栏中所有内容的内部包装,包括导航条。

 

 

 

 

 

refpara

 

右侧 margin-note 注释的外包装。

 

refparaleft

 

左侧 margin-note 注释的外包装。

 

refelem

 

右边 margin-note* 的外包装。

 

refelemleft

 

左侧 margin-note* 的外包装。

 

refcolumn

 

右侧 margin-note 和 margin-note* 的中间包络符。

 

refcolumnleft

 

左边的 margin-note 和 margin-note* 注释的中间包装。

 

refcontent

 

margin-note 和 margin-note* 注释的内部包装。

 

 

 

 

 

tocset

 

将内容表的面板分组:主面板和 “on this page.”

 

 

 

 

 

tocview

 

包裹主(多页模式)或唯一(单页模式)内容表面板。

 

tocviewlist

 

在主内容表面板中的一个分层的内容。

 

tocviewlisttopspace

 

与 tocviewlist 同为第一层。

 

tocviewtoggle

 

一个层的始终可见的名称。

 

tocviewtitle

 

与第一层的 tocviewtoggle 一起。

 

tocviewsublist

 

一个图层中的一个项目,前后有多个项目和更多的项目。

 

tocviewsublistonly

 

单一项目层中的一个项目。

 

tocviewsublisttop

 

多项目层中的第一个项目。

 

tocviewsublistbottom

 

多项目层中的最后一个项目。

 

tocviewlink

 

当链接到不同的页面时,层中的项目的内部包装。

 

tocviewselflink

 

当链接到同一页面时,图层中每个项目的内部包装器。

 

 

 

 

 

tocsub

 

包裹 "在此页面"(仅限多页面模式)的内容表面板。

 

tocsubtitle

 

包裹 "在此页面 "字样。

 

tocsublist

 

"此页 "面板的内部表格。

 

tocsublinknumber

 

在 "本页 "面板中的条目的编号。

 

tocsubseclink

 

在 "此页 "面板上的一个 section 条目的标题。

 

tocsubnonseclink

 

在 "此页 "面板中的一个 non-section 条目的标题,该条目有一些章节链接。

 

tocsublink

 

在 "本页 "面板中没有章节链接的 non-section 条目的标题。

 

toctoplink

 

内联(非面板)目录中的顶级条目。

 

toclink

 

内联(非面板)目录中的嵌套条目。

 

 

 

 

 

versionbox

 

版本的外部包装

 

version

 

在搜索框和/或导航的情况下,版本的内包装。

 

versionNoNav

 

在没有搜索框和导航的情况下,版本的内包装。

 

 

 

 

 

SAuthorListBox

 

作者列表的外包装。

 

SAuthorList

 

作者列表的内部包装器。

 

author

 

单个作者的包装器。

 

 

 

 

 

navsettop

 

包裹顶部的导航栏(在多页模式或有搜索栏时)。

 

navsetbottom

 

包裹底部导航栏(在多页模式下或当搜索栏存在时)。

 

navleft

 

包裹导航条中的左侧元素。

 

navright

 

将右侧元素包裹在导航栏中。

 

nonavigation

 

导航栏中禁用的链接。

 

searchform

 

顶部导航栏中搜索框的外部包装。

 

searchbox

 

顶部导航栏中搜索框的内部包装。

 

nosearchform

 

取代顶部导航栏中不存在的搜索框。

 

 

 

 

 

SSubSubSubSection

 

嵌套很深的小节(在 <h5> 下面)。

 

 

 

 

 

SIntrapara

 

与 <div> 一起使用, 而不是与 <p> 一起用于 compound-paragraph 中的段落。

 

 

 

 

 

SubFlow

 

对于没有样式名称的 nested-flow :没有插入。

 

SCodeFlow

 

对于具有 'code-inset 样式名称的 nested-flow : 适用于代码的插入。

 

SVInsetFlow

 

对于具有 'vertical-inset 样式名称的 nested-flow ： 在适合代码的前后添加空格。

 

SCentered

 

对于 centered 创建的 nested-flow : 水平居中。

 

SVerbatim

 

对于 verbatim 创建的 table :禁用换行符。

 

 

 

 

 

boxed

 

对于一个具有 'boxed 风格名称的 table :作为一个定义框。

 

 

 

 

 

compact

 

对于具有 'compact 样式名称的 itemlist 。

 

 

 

 

 

techoutside

 

技术术语参考的外包装。

 

techinside

 

技术术语参考的内部包装。

 

 

 

 

 

indexlink

 

对于索引中的条目。

 

 

 

 

 

stt

 

固定宽度文本。

 

sroman

 

Serif 文本。

 

ssanserif

 

Sans serif 文本。

 

slant

 

倾斜(与斜体相反)的文本。

 

Smaller

 

较小的文本(由 smaller 创建)。

 

Larger

 

较小的文本(由 larger 创建)。

 

hspace

 

用于 hspace 生成的空白。

 

nobreak

 

禁用断开链接。

 

badlink

 

断开的交叉引用。

 

plainlink

 

不带下划线的超链接。

此外, SIEHidden 样式类内置于所有 Scribble HTML 输出中,用于隐藏 Internet Explorer 6 上的元素。

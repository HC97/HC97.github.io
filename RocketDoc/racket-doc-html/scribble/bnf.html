4.6 BNF 语法

 (require scribble/bnf) package: scribble-lib
scribble/bnf 库提供了用于排版语法的实用程序。

例如,

  @(let ([open @litchar{(}]

         [close @litchar{)}])

     @BNF[(list @nonterm{expr}

                @nonterm{id}

                @BNF-seq[open @kleeneplus[@nonterm{expr}] close]

                @BNF-seq[open @litchar{lambda}

                          open @kleenestar[@nonterm{id}] close

                          @nonterm{expr} close]

                @nonterm{val})

          (list @nonterm{val}

                @BNF-alt[@nonterm{number} @nonterm{primop}])

          (list @nonterm{id}

                @elem{any name except for @litchar{lambda}})])

产生的输出是

 

‹expr›

 ::= 

‹id›

 

  |  

( ‹expr›+ )

 

  |  

( lambda ( ‹id›* ) ‹expr› )

 

  |  

‹val›

 

‹val›

 ::= 

‹number›  |  ‹primop›

 

‹id›

 ::= 

any name except for lambda

参见 racketgrammar 。

procedure

(BNF prod ...) → table?

  prod : 
(cons/c (or/c block? content?)
        (non-empty-listof (or/c block? content?)))
对语法表进行排版。每个生产都以被定义的非术语的元素(通常用 nonterm 构建)开始, 然后是一个可能性的列表(通常用 BNF-seq 构建,等等),在不同的行中显示。

procedure

(nonterm pre-content ...) → element?

  pre-content : pre-content?
对非术语进行排版:角括号内的斜体。

procedure

(BNF-seq elem ...) → (or/c element? "")

  elem : content?
排列一个序列。

procedure

(BNF-seq-lines elems ...) → block?

  elems : (listof content?)
排列一个分成多行的序列,其中每个 elems 是一行。

procedure

(BNF-group pre-content ...) → element?

  pre-content : pre-content?
排列一个由大括号包围的组(例如,这样整个组可以重复)。

procedure

(optional pre-content ...) → element?

  pre-content : pre-content?
输入一个可选的元素:在方括号中。

procedure

(kleenestar pre-content ...) → element?

  pre-content : pre-content?
输入一个0或更多的重复。

procedure

(kleeneplus pre-content ...) → element?

  pre-content : pre-content?
输入一个1或更多的重复。

procedure

(kleenerange n m pre-content ...) → element?

  n : any/c
  m : any/c
  pre-content : pre-content?
输入一个 n 到 m 的重复。 n 和 m 参数用 (format "~a" n) 和 (format "~a" m) 转换为字符串。

procedure

(BNF-alt elem ...) → element?

  elem : element?
对生产的右侧进行排版,使其出现在单行上。 其结果通常被用作 BNF 生产列表中的一个可能性。

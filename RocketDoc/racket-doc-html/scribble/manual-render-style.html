4.2.13 手动渲染风格

对文档的主要 part 使用 #lang scribble/manual , 可以在 doc 导出时关联 style properties ,以选择 Racket 手动渲染样式。

一个 html-defaults style property 被添加到 doc 中, 除非 doc 的样式已经有一个 html-defaults style property (例如,提供给 title)。 同样地,如果还没有 latex-defaults style property 的话,也会添加一个。 最后,一个 css-style-addition 属性总是被添加。

对于 HTML 的渲染:

  • 文档的 prefix file 被设置为 "scribble-prefix.html", 就像通常在 html-defaults 中一样。

  • 文档的 style file 被设置为 html-defaults 中 "scribble" 集合的 "manual-style.css" 。

  • 来自 "scribble" 集合的文件 "manual-fonts.css" 被指定为 html-defaults 中的一个额外的附带文件。

  • 来自 "scribble" 集合的文件 "manual-racket.css" 被作为 css-style-addition 加入。

要获得这种配置而不使用 #lang scribble/manual , 请使用 manual-doc-style 。

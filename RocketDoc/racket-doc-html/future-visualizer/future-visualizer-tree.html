2 Future 创建树

creation tree 显示了一个由程序创建的每个 future 都有一个节点的树。 这种显示对于以嵌套方式生成期货(期货中的期货)的程序特别有用。 对于任何给定的未来节点,该节点的子节点代表由该未来创建的未来(在其思维范围内)。 对于所有程序,树的根是一个特殊的节点,表示主计算线程(运行时线程),并表示为 RTT 。

procedure

(creation-tree-pict events    
  [#:x x    
  #:y y    
  #:width width    
  #:node-width node-width    
  #:height height    
  #:padding padding    
  #:zoom zoom]) → pict?
  events : (listof indexed-future-event?)
  x : (or #f exact-nonnegative-integer?) = #f
  y : (or #f exact-nonnegative-integer?) = #f
  width : (or #f exact-nonnegative-integer?) = #f
  node-width : (or #f exact-nonnegative-integer?) = #f
  height : (or #f exact-nonnegative-integer?) = #f
  padding : (or #f exact-nonnegative-integer?) = #f
  zoom : exact-nonnegative-integer? = 1
返回一个 pict ,显示 events 中跟踪的未来创建树。 可选参数 x, y, width 和 height 可用于获取创建树图像的特定区域(以像素为单位)。 node-width 参数指定每个节点的直径(以像素为单位)。 padding 参数指定每个深度之间垂直和兄弟之间水平的最小空间。 zoom 参数指定范围为 1-5 的树图像的缩放系数,其中 5 返回 500% 的缩放。

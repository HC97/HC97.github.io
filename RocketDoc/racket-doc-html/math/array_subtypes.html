6.15 子类型

6.15.1 Flonum 数组

syntax

FlArray

flonum arrays 的类型,是 (Settable-Array Flonum) 的一个子类型, 将其元素存储在 FlVector 中。 一个 flonum 数组总是 strict 。

syntax

(flarray #[#[...] ...])

像 array 一样,但是创建 flonum arrays 。 列出的元素必须是实数,而且可以是精确的。

Examples:
> (flarray 0.0)

- : FlArray

(flarray 0.0)

> (flarray #['x])

eval:312:0: Type Checker: type mismatch

  expected: Real

  given: 'x

  in: #((quote x))

> (flarray #[#[1 2] #[3 4]])

- : FlArray

(flarray #[#[1.0 2.0] #[3.0 4.0]])

procedure

(array->flarray arr) → FlArray

  arr : (Array Real)
返回一个 flonum 数组,其元素与 arr 大致相同。 精确的元素在转换过程中可能会丢失精度。

procedure

(flarray-data arr) → FlVector

  arr : FlArray
返回一个 flonum 向量中 arr 的元素,以行为主的顺序。

Example:
> (flarray-data (flarray #[#[1 2] #[3 4]]))

- : FlVector

(flvector 1.0 2.0 3.0 4.0)

procedure

(flarray-map f arrs ...) → FlArray

  f : (Flonum ... -> Flonum)
  arrs : FlArray
将函数 f 映射到数组 arrs 上。 如果数组不具有相同的形状,它们将被首先 broadcast 。 如果数组有相同的形状,这个操作就会相当快。

函数 f 是为了接受与它下面的 flonum 数组参数数量相同的参数。 然而,目前 Typed Racket 的一个限制要求 f 可以接受 任何 数量的参数。 要映射一个单极函数,如 fl+,目前,请使用 inline-flarray-map 或 array-map 。

syntax

(inline-flarray-map f arrs ...)

 
  f : (Flonum ... -> Flonum)
  arrs : FlArray
像 inline-array-map,但是对于 flonum 数组。

这一点目前在非类型化的 Racket 中是不可用的。

6.15.2 Float-Complex 数组

syntax

FCArray

float-complex arrays 的类型,是 (Settable-Array Float-Complex) 的子类型,将其元素存储在一对 FlVector 中。 一个浮点复数数组总是 strict 。

syntax

(fcarray #[#[...] ...])

像 array 一样,但是创建 float-complex arrays 。 列出的元素必须是数字,而且可以是精确的。

Examples:
> (fcarray 0.0)

- : FCArray

(fcarray 0.0+0.0i)

> (fcarray #['x])

eval:316:0: Type Checker: type mismatch

  expected: Number

  given: 'x

  in: #((quote x))

> (fcarray #[#[1 2+1i] #[3 4+3i]])

- : FCArray

(fcarray #[#[1.0+0.0i 2.0+1.0i] #[3.0+0.0i 4.0+3.0i]])

procedure

(array->fcarray arr) → FCArray

  arr : (Array Number)
返回一个浮点复数数组,其元素与 arr 大致相同。 确切的元素在转换过程中可能会失去精度。

procedure

(fcarray-real-data arr) → FlVector

  arr : FCArray

procedure

(fcarray-imag-data arr) → FlVector

  arr : FCArray
返回 arr 元素的实部和虚部,以 flonum 向量的形式,按行主次排列。

Examples:
> (define arr (fcarray #[#[1 2+1i] #[3 4+3i]]))
> (fcarray-real-data arr)

- : FlVector

(flvector 1.0 2.0 3.0 4.0)

> (fcarray-imag-data arr)

- : FlVector

(flvector 0.0 1.0 0.0 3.0)

procedure

6.14.1 快速傅里叶变换

Wikipedia: Discrete Fourier Transform

procedure

(array-axis-fft arr k) → (Array Float-Complex)

  arr : (Array Number)
  k : Integer
对 arr 的轴 k 进行离散傅里叶变换。 k 的长度必须是二的整数次方。(见 power-of-two?) 。 缩放惯例由参数 dft-convention 决定,默认为信号处理中使用的惯例。

变换是使用 max-math-threads 线程并行完成的。

procedure

(array-axis-inverse-fft arr k) → (Array Float-Complex)

  arr : (Array Number)
  k : Integer
array-axis-fft 的逆运算,通过参数化的正向变换 (dft-inverse-convention) 进行。

procedure

(array-fft arr) → FCArray

  arr : (Array Number)
使用 array-axis-fft 对 arr 的每个轴进行离散傅立叶变换。

procedure

(array-inverse-fft arr) → FCArray

  arr : (Array Number)
array-fft 的逆运算,通过对 (dft-inverse-convention) 的正向变换进行参数化。

Math Library

Neil Toronto <ntoronto@racket-lang.org>


Jens Axel Søgaard <jensaxel@soegaard.net>

 (require math) package: math-lib

math 库提供了对处理数字和数字集合有用的函数和数据结构。 这些功能包括

  • math/base: 常量和基本函数

  • math/flonum: Flonum 函数,包括高精确度支持

  • math/special-functions: 特殊(即非基本)函数

  • math/bigfloat: 任意精度的浮点函数

  • math/number-theory: 数论函数

  • math/array: 用于操作大型矩形数据集的功能数组

  • math/matrix: 阵列的线性代数函数

  • math/distributions: 概率分布

  • math/statistics: 统计函数

通过这个库,我们希望能在 Racket 中支持各种各样的应用数学,包括模拟、统计推断、信号处理和组合学。 如果你发现它对你的各种数学知识有所欠缺,请

  • 访问 Math Library Features wiki page， 看看有什么计划。

  • 联系我们或在 mailing lists 中的某一个上发表建议或提交补丁。

这是一个类型化的 Racket 库。 在类型化的 Racket 中使用它是最有效的,这样可以静态地检查契约。 然而,它的几乎所有内容都可以在非类型化的 Racket 中使用。异常和性能警告以 粗体字 显示。

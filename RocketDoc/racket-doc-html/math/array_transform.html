procedure

(array-transform arr ds proc) → (Array A)

  arr : (Array A)
  ds : In-Indexes
  proc : (Indexes -> In-Indexes)

procedure

(array-axis-permute arr perm) → (Array A)

  arr : (Array A)
  perm : (Listof Integer)
返回一个类似于 arr 的数组,但是其轴是按照 perm 进行排列的。

列表 perm 表示从源轴数到目标轴数的映射:源是列表的位置,目标是列表的元素。 例如, permutation '(0 1 2) 是三维数组的同一 permation, '(1 0) 交换轴 0 和 1, '(3 1 2 0) 交换轴 0 和 3 。

这个排列组合必须包含从 0 到 (- (array-dims arr) 1) 的每个整数一次。

Examples:
> (array-axis-swap (array #[#[0 1] #[2 3]]) 0 1)

7.5 入门操作和算术

procedure

(matrix+ M N ...) → (Matrix Number)

  M : (Matrix Number)
  N : (Matrix Number)

procedure

(matrix- M N ...) → (Matrix Number)

  M : (Matrix Number)
  N : (Matrix Number)

procedure

(matrix* M N ...) → (Matrix Number)

  M : (Matrix Number)
  N : (Matrix Number)
矩阵的加法、减法和乘积分别。

对于矩阵加法和减法,所有的矩阵必须具有相同的形状。

对于矩阵乘法,一个矩阵的列数必须等于下面矩阵的行数。

Examples:
> (define A (matrix ([1 2]
                     [3 4])))
> (define B (matrix ([5 6]
                     [7 8])))
> (define C (matrix ([ 9 10 11]
                     [12 13 14])))
> (matrix+ A B)

(array #[#[6 8] #[10 12]])

> (matrix- A B)

(array #[#[-4 -4] #[-4 -4]])

> (matrix* A C)

(array #[#[33 36 39] #[75 82 89]])

procedure

(matrix-expt M n) → (Matrix Number)

  M : (Matrix Number)
  n : Integer
用 n 个参数计算 (matrix* M ...) ,但效率更高。 M 必须是一个 square-matrix? ， n 必须是非负数。

Examples:
> (matrix* (matrix-expt (matrix [[1 1] [1 0]]) 100)
           (col-matrix [0 1]))

(array #[#[354224848179261915075] #[218922995834555169026]])

> (->col-matrix (list (fibonacci 100) (fibonacci 99)))

(array #[#[354224848179261915075] #[218922995834555169026]])

procedure

(matrix-scale M z) → (Matrix Number)

  M : (Matrix Number)
  z : Number
计算矩阵 zM, 一个与 M 相同形状的矩阵, 其中 M 的每个条目都与 z 相乘。

Example:
> (matrix-scale (matrix [[1 2] [3 4]]) 2)

(array #[#[2 4] #[6 8]])

procedure

(matrix-kronecker M N ...) → (Matrix Number)

  M : (Matrix Number)
  N : (Matrix Number)
Wikipedia: Kronecker product 计算Kronecker乘积。

Example:
> (matrix-kronecker (matrix [[1 2] [3 4] [5 6]])
                    (matrix [[7 8] [9 10]]))

(mutable-array

 #[#[7 8 14 16]

   #[9 10 18 20]

   #[21 24 28 32]

   #[27 30 36 40]

   #[35 40 42 48]

   #[45 50 54 60]])

Added in version 1.2 of package math-lib.

procedure

(matrix-map f M) → (Matrix R)

  f : (A -> R)
  M : (Matrix A)
(matrix-map f M0 M1 N ...) → (Matrix R)
  f : (A B Ts ... -> R)
  M0 : (Matrix A)
  M1 : (Matrix B)
  N : (Matrix Ts)
和 array-map 一样,但需要至少一个矩阵参数,而且从不 broadcasts 。

Examples:
> (matrix-map sqr (matrix [[1 2] [3 4]]))

(array #[#[1 4] #[9 16]])

> (matrix-map  +  (matrix [[1 2] [3 4]])
                  (matrix [[5 6] [7 8]]))

(array #[#[6 8] #[10 12]])

procedure

(matrix-sum Ms) → (Matrix Number)

  Ms : (Listof (Matrix Number))
与 (apply matrix+ Ms) 类似,但当 Ms 为空时,会引发运行时错误。

procedure

(matrix= M0 M1 N ...) → Boolean

  M0 : (Matrix Number)
  M1 : (Matrix Number)
  N : (Matrix Number)
当其参数大小相同且入口相等时,返回 #t 。

参见 matrix-relative-error 和 matrix-absolute-error ,以获得对浮点误差的容忍度。

7.10 基于行的算法

procedure

(matrix-gauss-elim M 
  [jordan? 
  unitize-pivot? 
  pivoting]) 
 → (Values (Matrix Number) (Listof Index))
  M : (Matrix Number)
  jordan? : Any = #f
  unitize-pivot? : Any = #f
  pivoting : (U 'first 'partial) = 'partial
Wikipedia: Gaussian elimination, Gauss-Jordan elimination 执行 Gaussian 消除法或 Gauss-Jordan 消除法。

如果 jordan? 为真,则在枢轴上方和下方都进行行操作。 如果 unitize-pivot? 为真,则枢轴的行被缩放,以便枢轴的值为 1 。 当两者都为真时,该算法被称为 Gauss-Jordan 消除法,结果矩阵为 缩小 行梯形形式。

如果 pivoting 是 'first, 那么当前列中的第一个非零条目被用作枢轴。 如果 pivoting 是 'partial, 则使用幅度最大的非零条目, 当 M 包含不精确的条目时,这将提高平均的数值稳定性。

第一个返回值是高斯消除法的结果。

第二个返回值是一个没有非零支点的列的索引列表。

例子见 matrix-row-echelon 。

procedure

(matrix-row-echelon M    
  [jordan?    
  unitize-pivot?    
  pivoting]) → (Matrix Number)
  M : (Matrix Number)
  jordan? : Any = #f
  unitize-pivot? : Any = #f
  pivoting : (U 'first 'partial) = 'partial
Wikipedia: Row echelon form 与 matrix-gauss-elim 类似,但只返回高斯消除的结果。

Examples:
> (define M (matrix [[2 1 -1] [-3 -1 2] [-2 1 2]]))
> (matrix-row-echelon M)

- : (Array Real)

(mutable-array #[#[-3 -1 2] #[0 5/3 2/3] #[0 0 1/5]])

> (matrix-row-echelon M #t)

- : (Array Real)

(mutable-array #[#[-3 0 0] #[0 5/3 0] #[0 0 1/5]])

> (matrix-identity? (matrix-row-echelon M #t #t))

- : Boolean

#t

最后一个例子表明, M 是可逆的。

使用 matrix-row-echelon 解决线性方程组(不检查失败):
> (define B (col-matrix [8 -11 -3]))
> (define MB (matrix-augment (list M B)))
> (matrix-col (matrix-row-echelon MB #t #t) 3)

- : (Array Real)

(array #[#[2] #[3] #[-1]])

> (matrix-solve M B)

- : (Array Real)

(array #[#[2] #[3] #[-1]])

使用 matrix-row-echelon 来反转一个矩阵(同样不需要检查是否失败):
> (define MI (matrix-augment (list M (identity-matrix 3))))
> (submatrix (matrix-row-echelon MI  #t #t) (::) (:: 3 #f))

- : #(struct:Array

      (Indexes Index (Boxof Boolean) (-> Void) (-> Indexes Real))

      #<syntax:.../array/typed-array-struct.rkt:56:13 prop:equal+hash>

      #<syntax:.../array/typed-array-struct.rkt:55:13 prop:custom-write>

      #<syntax:.../array/typed-array-struct.rkt:54:13 prop:custom-print-quotable>)

(array #[#[4 3 -1] #[-2 -2 1] #[5 4 -1]])

> (matrix-inverse M)

- : (Array Real)

(array #[#[4 3 -1] #[-2 -2 1] #[5 4 -1]])

procedure

(matrix-lu M [fail])

 → (Values (U F (Matrix Number)) (Matrix Number))
  M : (Matrix Number)
  fail : (-> F) = (λ () (error ...))
Wikipedia: LU decomposition 如果存在 M 的 LU 分解,则返回 M 的 LU 分解(必须是一个 square-matrix?) 。 如果 M 可以在不交换行的情况下以行-歇尔形式存在,则存在 LU 分解。

因为 matrix-lu 返回的是一个 unit 下三角矩阵(即对角线上只有 1 的下三角矩阵), 所以如果存在的话,这个分解是唯一的。

Examples:
> (define-values (L U)
    (matrix-lu (matrix [[4 3] [6 3]])))
> (values L U)

- : (values (Array Real) (Array Real))

(mutable-array #[#[1 0] #[3/2 1]])

(mutable-array #[#[4 3] #[0 -3/2]])

> (matrix* L U)

- : (Array Real)

(array #[#[4 3] #[6 3]])

如果 M 没有 LU 分解,则第一个结果是应用 failure thunk fail 的结果, 而第二个结果是原始参数 M:
> (matrix-lu (matrix [[0 1] [1 1]]))

matrix-lu: contract violation

  expected: LU-decomposable matrix

  given: (array #[#[0 1] #[1 1]])

> (matrix-lu (matrix [[0 1] [1 1]]) (λ () #f))

- : (values (U (Array Real) False) (Array Real))

#f

(array #[#[0 1] #[1 1]])

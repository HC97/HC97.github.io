6.11 集合名称的 API

 (require setup/collection-name) package: base

procedure

(collection-name? v) → boolean?

  v : any/c
如果 v 是一个语法上有效的字符串,即它是一个或多个 / 分离的字符串, 并且 collection-name-element? 返回true,则返回 #t 。

procedure

(collection-name-element? v) → boolean?

  v : any/c
如果 v 是一个在语法上有效的字符串,作为顶级集合名称或集合名称的一部分, 这意味着它是非空的,并且只包含 ASCII 字母、ASCII 数字 -, +, _ 和 %, 其中 % 只允许在两个小写十六进制数字之后,并且数字必须构成一个数字, 而不是字母、数字、 -, + 或 _ 的 ASCII 值，则返回 #t 。

6 raco setup: 安装管理

raco setup 命令为所有安装的集合构建字节码、文档、可执行文件和元数据索引。

由 raco setup 构建的集合可以是原始 Racket 发行版的一部分, 通过软件包管理器安装(见 Racket 中的包管理), 通过 PLaneT 安装(见 PLaneT: Automatic Package Distribution), 通过 raco link 链接,在 PLTCOLLECTS 环境变量中列出的目录中,或者放在默认集合目录中。

raco setup 工具本身并不直接支持集合的安装, 除了通过现在不鼓励使用的 -A 标志(见安装 安装 ".plt" 档案) 。 raco setup 命令被安装工具所使用,如软件包管理器或 PLaneT 。 修改已安装的图集的程序员可能会发现运行 raco setup 是非常有用的,它可以替代取消安装和重新安装一组图集的做法。

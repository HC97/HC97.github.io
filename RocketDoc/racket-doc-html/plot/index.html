Plot: 图表绘制

Neil Toronto <neil.toronto@gmail.com>

 (require plot) package: plot-gui-lib

Plot 库提供了一个灵活的接口来生成几乎所有类型的图。 它已经包括了许多常见的图,如散点图、直线图、等值线图、直方图、三维曲面和等值线图。 得益于 Racket 优秀的多后端绘图库,Plot 可以在 DrRacket 中以交互式剪辑的方式呈现绘图, 在幻灯片中以图片的方式呈现,以 PNG、PDF、PS 和 SVG 文件的方式呈现,或者在任何设备上呈现。

Plot 是一个类型化的 Racket 库, 但它也可以用于非类型化的 Racket 程序, 几乎没有性能损失 。 旧的类型化接口模块 plot/typed 仍可用于旧的类型化 Racket 程序。 新的 Typed Racket 程序应该使用 plot 。

对于没有 GUI 的绘图,参见 plot/no-gui 。 对于在 DrRacket 之外的类似 REPL 的环境中绘图,包括 Scribble 手册, 请参见 plot/pict 和 plot/bitmap 。

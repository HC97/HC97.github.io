28 签名和单元

racket/draw/draw-sig 和 racket/draw/draw-unit 库定义了 draw^ 签名和 draw@ 实现。

28.1 绘图单元

 (require racket/draw/draw-unit) package: draw-lib

value

draw@ : unit?

重新输出了 racket/draw 的所有输出。

28.2 绘图签名

 (require racket/draw/draw-sig) package: draw-lib

signature

draw^ : signature

包括所有由 racket/draw 导出的标识符。

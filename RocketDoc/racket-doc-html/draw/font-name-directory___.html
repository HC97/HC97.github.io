interface

font-name-directory<%> : interface?

有一个 font-name-directory<%> 对象: the-font-name-directory 。 它实现了从字体规格(面、族、样式和重量)到在特定设备上渲染文本的信息的映射。 程序员很少需要直接调用 the-font-name-directory 的方法。 它在绘制文本到 dc<%> 对象时被自动使用。 然而, the-font-name-directory 是可用的,这样程序员就可以手动查询或修改映射。 程序员可能还需要了解 face-and-family 映射是如何工作的。

要从 the-font-name-directory 中提取映射信息,首先要获得一个 font ID, 这是一个基于家族和可选脸部字符串的索引。 字体 ID 由 find-or-create-font-id 和 get-font-id 返回。 字体ID可以与重量和样式相结合,通过 via get-screen-name 或 get-post-script-name 获得一个特定的映射值。

对于一个没有脸部字符串的字体系列,相应的字体 ID 对每个平台和设备都有一个有用的内置映射。 对于有字面字符串的字体家族, the-font-name-directory 会解释该字符串(以特定于平台的方式), 以生成用于绘图的映射(到画布的 dc<%>, bitmap-dc% 或 printer-dc%)。

目前,在所有平台上,当一个面孔字符串包含一个逗号时,它被解释为一个 Pango 字体描述,否则它被视为一个家族名称。 因此,一个字体可以只是一个族名,如 "Helvetica", 一个族名后面跟着一个逗号和字体修饰词, 如 "Helvetica, Bold" ； 或者是一个由逗号分隔的族名后面跟着空格分隔的字体选项的序列,如 "Helvetica, Arial, bold italic" 。 字体描述中的任何尺寸都会被一个给定的 font% 的尺寸所覆盖。 字体描述中的任何(倾斜的)样式或重量选项分别被一个给定的 font% 的样式或重量的非 'normal 值所覆盖。

method

(send a-font-name-directory find-family-default-font-id family)

 → exact-integer?
  family : 
(or/c 'default 'decorative 'roman 'script
      'swiss 'modern 'symbol 'system)
获取代表一个家族的默认字体的字体 ID。关于字体家族的信息请参见 font% 。

method

(send a-font-name-directory find-or-create-font-id name 
  family) 
 → exact-integer?
  name : string?
  family : 
(or/c 'default 'decorative 'roman 'script
      'swiss 'modern 'symbol 'system)
获取一个字体 ID 的字体名称,如果有必要,可以初始化该字体名称的映射。

字体 ID 只作为 the-font-name-directory 的映射索引而有用。

method

(send a-font-name-directory get-face-name font-id)

 → (or/c string? #f)
  font-id : exact-integer?
获取一个字体 ID 的面名。 如果字体 ID 与某一字体家族的默认字体相对应,则返回 #f 。

method

(send a-font-name-directory get-family font-id)

 → 
(or/c 'default 'decorative 'roman 'script
      'swiss 'modern 'symbol 'system)
  font-id : exact-integer?
获取一个字体 ID 的字体家族。 有关字体家族的信息,请参见 font% 。

method

(send a-font-name-directory get-font-id name 
  family) 
 → exact-integer?
  name : string?
  family : 
(or/c 'default 'decorative 'roman 'script
      'swiss 'modern 'symbol 'system)
获取一个与默认字体家族配对的脸部名称的字体 ID。如果给定配对的映射尚未被初始化,则返回 0 。 另请参阅 find-or-create-font-id 。

字体 ID 仅作为 the-font-name-directory 的映射索引而有用。

method

(send a-font-name-directory get-post-script-name font-id 
  weight 
  style) 
 → (or/c string? #f)
  font-id : exact-integer?
  weight : (or/c 'normal 'bold 'light)
  style : (or/c 'normal 'italic 'slant)
获取一个字体 ID、重量和样式组合的 PostScript 字体描述。

有关 weight 和 style 的信息请参见 font% 。

method

(send a-font-name-directory get-screen-name font-id 
  weight 
  style) 
 → (or/c string? #f)
  font-id : exact-integer?
  weight : (or/c 'normal 'bold 'light)
  style : (or/c 'normal 'italic 'slant)
获取字体 ID、重量和样式组合的平台依赖型屏幕字体描述 (用于绘制到画布的 dc<%>, bitmap-dc% 或 printer-dc%) 。

有关 weight 和 style 的信息,请参见 font% 。

method

(send a-font-name-directory set-post-script-name font-id 
  weight 
  style 
  name) 
 → void?
  font-id : exact-integer?
  weight : (or/c 'normal 'bold 'light)
  style : (or/c 'normal 'italic 'slant)
  name : string?
为一个字体 ID、重量和样式组合设置一个 PostScript 字体描述。 参见 get-post-script-name 。

有关 weight 和 style 的信息请参见 font% 。

method

(send a-font-name-directory set-screen-name font-id    
  weight    
  style    
  name) → void?
  font-id : exact-integer?
  weight : (or/c 'normal 'bold 'light)
  style : (or/c 'normal 'italic 'slant)
  name : string?
为字体 ID、重量和样式组合设置与平台有关的屏幕字体描述 (用于绘制到画布的 dc<%>, bitmap-dc% 或 printer-dc%) 。

有关 weight 和 style 的信息,请参见 font% 。

3.9 C 联盟类型

procedure

(make-union-type type ...+) → ctype?

  type : ctype?
用于创建新的 C 联合类型的原始类型构造函数。 像 C 结构类型一样,联合类型是新的原始类型,没有相关的转换函数。 联合类型总是被视为具有 'atomic 分配模式的结构。

Example:
> (make-union-type (_list-struct _int _int)
                   (_list-struct _double _double))

#<compound-ctype>

procedure

(_union type ...+) → ctype?

  type : ctype?
创建一个联合类型,其 Racket 表示法是一个与 union-ref 和 union-set! 一起工作的联合。 联盟不被复制;Racket 表示法由底层的 C 表示法支持。

Example:
> (_union (_list-struct _int _int)
          (_list-struct _double _double))

#<compound-ctype>

procedure

(union? v) → boolean?

  v : any/c
如果 v 是通过 _union 的 C 值的 Racket 表示,则返回 #t ,否则返回 #f 。

Examples:
> (define a-union-type
    (_union (_list-struct _int _int)
            (_list-struct _double _double)))
> (define a-union-val
    (cast (list 3.14 2.71)
          (_list-struct _double _double)
          a-union-type))
> (union? a-union-val)

#t

> (union? 3)

#f

procedure

(union-ref u i) → any/c

  u : union?
  i : exact-nonnegative-integer?
从一个 union 中提取一个变体。 变体的索引从 0 开始。

Examples:
; see examples for union? for definitions
> (union-ref a-union-val 1)

'(3.14 2.71)

procedure

(union-set! u i v) → void?

  u : union?
  i : exact-nonnegative-integer?
  v : any/c
在一个联盟中设置一个变体。

Examples:
; see examples for union? for definitions
> (union-set! a-union-val 0 (list 4 5))
> a-union-val

#<union>

> (union-ref a-union-val 0)

'(4 5)

procedure

(union-ptr u) → cpointer?

  u : union?
提取一个 union 的存储指针。

Example:
> (union-ptr a-union-val)

#<cpointer+offset>

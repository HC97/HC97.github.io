3.3 其他原子类型

value

_stdbool : ctype?

_stdbool 类型代表　<stdbool.h>　中的 C99 bool 类型。 从 Racket 到 C, _stdbool 将　#f 转换为 0 bool , 将任何其他值转换为　1 bool　。 从 C 语言到 Racket, _stdbool 将　0 转换为 #f ,将其他任何值转换为 #t　。

Added in version 6.0.0.6 of package base.

value

_bool : ctype?

和 _stdbool　一样,但在 C 语言中使用 int 表示,反映了许多传统(即 C99 之前)的布尔编码。

value

_void : ctype?

表示 Racket 的 #<void> 返回值,它不能用来将值转化为 C 语言。 这种类型不能用于函数输入。

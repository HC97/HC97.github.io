7.5 从上下文中生成一个单元

syntax

(unit-from-context tagged-sig-spec)

创建一个使用包围环境中的绑定实现一个接口的单元。 生成的单元在本质上与

(unit
  (import)
  (export tagged-sig-spec)
  (define id expr) ...)

为每个必须被定义以满足出口的 id , 并且每个相应的 expr 在 unit-from-context 表达式的环境中产生 id 的值。 (然而,单元不能像上面那样写,因为单元内的每个 id 定义都会在 unit 形式之外的绑定上留下阴影)。

关于 tagged-sig-spec 的语法,参见 unit 。

syntax

(define-unit-from-context id tagged-sig-spec)

和 unit-from-context 一样,单元是由包围的环境构建的,和 define-unit 一样, id 被绑定到静态信息上,以便以后用于推理。

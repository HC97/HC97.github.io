7.8 Unit Utilities

procedure

(unit? v) → boolean?

  v : any/c
如果 v 是一个 unit,返回 #t ,否则返回 #f 。

syntax

(provide-signature-elements sig-spec ...)

扩展到一个由 sig-spec 暗示的所有标识符的 provide 。 关于 sig-spec 的语法,请参见 unit 。

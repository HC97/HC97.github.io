Racket 参考

Matthew Flatt
and PLT

本手册定义了 Racket 的核心语言,并介绍了其最主要的库。 配套手册 Racket 指南 提供了一个更友好的(尽管不那么精确和完整)的语言概述。

本手册的源代码可以在 GitHub 上找到。

 #lang racket/base package: base
 #lang racket
除非另有说明,本手册中定义的绑定是由 racket/base 和 racket 语言导出的。

racket/base 库比 racket 库小得多,通常加载速度更快。

racket 库结合了 racket/base, racket/bool, racket/bytes, racket/class, racket/cmdline, racket/contract, racket/dict, racket/file, racket/format, racket/function, racket/future, racket/include, racket/list, racket/local, racket/match, racket/math, racket/mutability, racket/path, racket/place, racket/port, racket/pretty, racket/promise, racket/sequence, racket/set, racket/shared, racket/stream, racket/string, racket/system, racket/tcp, racket/udp, racket/unit, and racket/vector 。

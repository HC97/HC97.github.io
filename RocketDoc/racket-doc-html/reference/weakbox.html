16.1 弱盒

weak box 类似于普通的盒(见 Boxes) , 但是当垃圾收集器(见 垃圾回收)可以证明一个弱盒的内容值只能通过弱引用到达时, 弱盒的内容将被替换成 #f 。 weak reference 是指通过弱盒的引用,通过弱散列表(见 Hash Tables)中的键的引用, 通过可以被 #f 替换的 ephemeron 中的值(见 外显子), 或者通过监护人(见 Custodians) 。

procedure

(make-weak-box v) → weak-box?

  v : any/c
返回一个新的弱箱,最初包含 v 。

procedure

(weak-box-value weak-box [gced-v]) → any/c

  weak-box : weak-box?
  gced-v : any/c = #f
返回 weak-box 中包含的值。 如果垃圾收集器已经证明 weak-box 的前一个内容值只能通过弱引用到达, 那么将返回 gced-v (默认为 #f) 。

procedure

(weak-box? v) → boolean?

  v : any/c
如果 v 是一个弱箱,返回 #t ,否则返回 #f 。

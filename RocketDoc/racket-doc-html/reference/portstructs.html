13.1.8 Structures as Ports

value

prop:input-port : struct-type-property?

value

prop:output-port : struct-type-property?

prop:input-port 和 prop:output-port 结构类型属性分别标识了其实例可以作为输入和输出端口的结构类型。

每个属性值都可以是以下两种情况之一:

  • 一个输入端口(对于 prop:input-port )或输出端口(对于 prop:output-port): 在这种情况下,使用该结构作为端口等同于使用给定的输入或输出端口。

  • 一个精确的非负整数,介于 0 (包括)和结构类型中的非自动字段数量之间(不包括,不计算超类型字段): 这个整数标识了结构中的一个字段,并且该字段必须被指定为不可变的。 如果该字段包含一个输入端口(对于 prop:input-port) 或输出端口(对于 prop:output-port), 则使用该端口。 否则, prop:input-port 将使用一个空字符串的输入端口, 而 prop:output-port 将使用一个丢弃所有数据的端口。

一些程序,比如 file-position, 同时作用于输入和输出端口。 当给定一个同时具有 prop:input-port 和 prop:output-port 属性的结构类型的实例时, 该实例被用作输入端口。

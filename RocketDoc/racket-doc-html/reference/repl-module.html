14.13 racket/repl 库

 (require racket/repl) package: base

racket/repl 提供了与 racket/base 相同的 read-eval-print-loop , 但其内部依赖性比 racket/base 更少。 它在某些情况下会在启动时被加载,如 初始化 所述。

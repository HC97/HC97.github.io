4.11 可变对和列表

mutable pair 就像用 cons 创建的对, 但它支持 set-mcar! 和 set-mcdr! 突变操作来改变可变对的部分 (像传统的 Lisp 和 Scheme 对)。

mutable list 类似于用对创建的列表,但它是用 mutable pairs 创建的。

一个 mutable pair 不是一个 pair; 它们是完全独立的数据类型。 同样地,一个 mutable list 也不是一个列表,除了空列表也是空的可变列表。 与其用易变对和易变列表编程,数据结构如对、列表和哈希表实际上都是更好的选择。

一个 mutable list 可以作为一个单值序列使用(参见 Sequences)。 mutable list 的元素作为序列的元素。也见 in-mlist 。

4.11.1 可变对的构造器和选择器

procedure

(mpair? v) → boolean?

  v : any/c
如果 v 是一个 mutable pair , 返回 #t ,否则返回 #f 。

procedure

(mcons a d) → mpair?

  a : any/c
  d : any/c
返回一个新分配的 mutable pair , 其第一个元素是 a ,第二个元素是 d 。

procedure

(mcar p) → any/c

  p : mpair?
返回 mutable pair p 的第一个元素。

procedure

(mcdr p) → any/c

  p : mpair?
返回 mutable pair p 的第二个元素。

procedure

(set-mcar! p v) → void?

  p : mpair?
  v : any/c
更改 mutable pair p ,使其第一个元素为 v 。

procedure

(set-mcdr! p v) → void?

  p : mpair?
  v : any/c
改变 mutable pair p ,使其第二个元素为 v 。

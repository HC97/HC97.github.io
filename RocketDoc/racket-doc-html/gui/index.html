Racket 图形界面工具箱

Matthew Flatt,
Robert Bruce Findler,
and John Clements

 (require racket/gui/base) package: gui-lib
racket/gui/base 库提供了本手册中定义的所有类、接口和过程的绑定, 此外还有 racket/draw 和 file/resource 。

 #lang racket/gui package: gui-lib
racket/gui 语言结合了 racket 语言和 racket/gui/base 以及 racket/draw 模块的所有绑定。

racket/gui 工具箱大致上分为两部分:

  • windowing toolbox, 用于实现窗口、按钮、菜单、文本字段和其他控件。

  • editor toolbox, 用于开发传统的文本编辑器,混合文本和图形的编辑器, 或自由形式的布局编辑器(如文字处理器,HTML 编辑器,或基于图标的文件浏览器)。

工具箱的这两部分都广泛地依赖于 racket/draw 绘图库。

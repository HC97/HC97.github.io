class

list-box% : class?

  superclass: object%

  extends: list-control<%>
列表框允许用户从一个滚动的列表中选择一个或多个字符串项。 列表框要么是一个单选控件(如果一个项目被选中,先前的选择就会被删除), 要么是一个多选控件(点击一个项目就会打开或关闭该项目,与其他选择无关)。

每当用户改变列表框中的选择时,列表框的回调过程就会被调用。 当每个列表框被创建时,一个回调过程被作为初始化参数提供。

一个列表框可以有多个列,并有可选的列标题。 列表中的一个项目对应于横跨所有列的一行。 当显示列头时,列的宽度可以由用户改变。 此外,列可以选择性地支持用户拖动以改变列的显示顺序,而逻辑顺序保持固定。

List box rows and columns are indexed from 0.

另见 choice% 。

如果 label 不是 #f, 它将被用作列表框的标签。 否则,列表框将不显示其标签。

如果 & 出现在 label 中,就会像 button% 一样被特别解析。

choices 列表指定了出现在列表框中的初始项目列表。 如果列表框有多列, choices 决定了第一列的内容,其他列被初始化为空字符串。

当用户通过选择、重新选择、取消选择或双击一个项目来改变列表框的选择时, callback 过程被调用。 当用户双击一个项目时,提供给回调程序的事件类型是 'list-box-dclick , 否则就是 'list-box 。

columns 列表决定了列表框中的列的数量。 columns 中的列标题只有在 style 包括 'column-headers 时才会显示。 如果 style 还包括 'clickable-headers, 那么点击标题就会触发对 callback 的调用, 其参数为 column-control-event% ,事件类型为 'list-box-column; 由于历史原因, 'clickable-headers 对 Windows 没有影响,标题的点击总是被报告。

style 说明必须准确地包括以下内容之一:

  • 'single — 创建一个单选列表。

  • 'multiple — 创建一个多选列表,单次点击会取消选择其他项目并选择一个新的项目。 当单选是常见的,但允许多选的时候,使用这种风格的列表。

  • 'extended — 创建一个多选列表,通过切换被点击的项目来扩展或收缩选择。 当多选是常规而非例外时,使用这种风格的列表。

'multiple 和 'extended 样式决定了对未经修改的鼠标点击的独立平台解释, 但拖动、移位点击、控制点击等有平台标准的解释。 无论特定平台的界面如何,用户总是可以选择不相干的项目集或取消选择项目(并且不留任何项目被选择)。 在一些平台上,用户可以在一个 'single 列表框中取消选择(唯一)的项目。

如果 style 包括 'vertical-label, 那么 list box 会在控件上方创建标签; 如果 style 不包括 'vertical-label (可选择包括 'horizontal-label), 那么标签会在 list box 左侧创建。 如果 style 包含 'deleted, 那么 list box 就被创建为隐藏的,它不会影响其父辈的几何形状; list box 可以通过调用 parent 的 add-child 方法,在以后变成活动的。

如果 style 包括 'variable-columns, 那么列表框中的列数可以通过 append-column 和 delete-column 改变。

如果 selection 是一个整数,它被传递给 set-selection 来设置初始选择。 该 selection 必须小于 choices 的长度。

参数 font 决定了控件内容的字体、而 label-font 则决定了控件标签的字体。 关于 enabled 参数的信息,请参见 window<%> 。 关于 horiz-margin 和 vert-margin 参数的信息,请参见 subarea<%> 。 关于 min-width, min-height, stretchable-width 和 stretchable-height 参数的信息,请参见 area<%> 。

如果 column-order 参数不是 #f, 它决定了最初显示逻辑列的顺序。 更多信息见 set-column-order 。 如果 style 包括 'column-headers 和 'reorderable-headers, 那么用户可以在显示时重新排序列(但显示顺序不会改变列的逻辑顺序)。

(send a-list-box select n [select?]) → void?

  n : exact-nonnegative-integer?
  select? : any/c = #t
选择或取消选择一个项目。对于单选列表框中的选择,如果当前选择了一个不同的选项,它将被自动取消选择。 对于在多选列表框中的选择,其他选择会被保留,这与 set-selection 不同。

如果 select? 是 #f, 以 n 为索引的项目被取消选择;否则它被选中。 List box rows are indexed from 0. 。 如果 n 等于或大于选择的数量, an exn:fail:contract exception is raised 。

A list box’s selection 可以被 the user clicking the control 改变,这种改变不经过这个方法;使用 control 回调程序(作为初始化参数提供) 来监控 selection 的变化。

控件的回调过程 not 被调用。

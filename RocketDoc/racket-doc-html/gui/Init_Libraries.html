13 Init Libraries

 (require racket/gui/init) package: gui-lib
racket/gui/init 库是 GRacket 的默认启动库。 它重新输出了 racket/init 和 racket/gui/base 库, 并将 current-load 设置为使用 text-editor-load-handler 。

 (require racket/gui/interactive) package: gui-lib
类似于 racket/interactive, 但用于 GRacket。 这个库可以通过修改 (find-config-dir) 中 "config.rktd" 文件中的 'gui-interactive-file 来改变。 此外,如果文件 "gui-interactive.rkt" 存在于 (find-system-path 'addon-dir) 中, 它将被运行,而不是安装范围内的图形交互模块。
这个库运行用户主目录中的 (find-graphical-system-path 'init-file) 文件, 如果它存在,而不是他们的 (find-system-path 'init-file) 。 与 racket/interactive 不同,这个库不启动 xrepl 。

Added in version 1.27 of package gui-lib.

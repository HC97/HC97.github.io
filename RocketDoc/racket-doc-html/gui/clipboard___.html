interface

clipboard<%> : interface?

一个单一的 clipboard<%> 对象, the-clipboard, 管理全系统的剪贴板内容,用于剪切和粘贴。

在 Unix 上,第二个 clipboard<%> 对象, the-x-selection-clipboard, 管理全系统的 X11 选择内容。 如果 'GRacket:selectionAsClipboard preference 偏好(见 首选项) 被设置为非零的真值, 那么 the-clipboard 总是与 the-x-selection-clipboard 相同,并且不使用全系统的 X11 剪贴板。

在Windows和Mac OS上, the-x-selection-clipboard 总是与 the-clipboard 相同。

数据可以通过以下两种方式之一输入剪贴板:通过设置当前的剪贴板字符串或字节字符串, 或者通过安装一个 clipboard-client% 对象。 当一个客户端被安装后,对剪贴板数据的请求被引导到客户端。

通用数据总是以字节串的形式从剪贴板中检索。 当检索剪贴板数据时,一个数据类型字符串指定了数据字符串的格式。 不同剪贴板格式的可用性是由当前剪贴板所有者决定的。

method

(send a-clipboard get-clipboard-bitmap time)

 → (or/c (is-a?/c bitmap%) #f)
  time : exact-integer?
以位图形式获取当前剪贴板内容(Windows、Mac OS),如果剪贴板不包含位图,则返回 #f 。

关于事件空间和当前剪贴板客户端的信息见 get-clipboard-data 。

关于 time 参数的讨论,见 剪切和粘贴的时间戳 。 如果 time 超出了平台特定的时间范围, an exn:fail:contract exception is raised 。

method

(send a-clipboard get-clipboard-data format 
  time) 
 → (or/c bytes? string? #f)
  format : string?
  time : exact-integer?
获取特定格式的当前剪贴板内容,如果剪贴板不包含请求格式的数据,返回 #f 。

如果剪贴板客户端与非当前的事件空间相关联,则通过客户端的事件空间的回调事件获取数据。 如果在一秒钟内没有结果,请求将被放弃,并返回 #f 。

关于 format 的信息,见 add-type in clipboard-client% 。

关于 time 参数的讨论,见 剪切和粘贴的时间戳 。 如果 time 超出了平台特定的时间范围, an exn:fail:contract exception is raised 。

method

(send a-clipboard get-clipboard-string time) → string?

  time : exact-integer?
获取当前剪贴板内容的简单文本,如果剪贴板不包含任何文本,返回 "" 。

关于事件空间和当前剪贴板客户端的信息,见 get-clipboard-data 。

关于 time 参数的讨论,见 剪切和粘贴的时间戳 。 如果 time 超出了平台特定的时间范围, an exn:fail:contract exception is raised 。

method

(send a-clipboard same-clipboard-client? owner) → boolean?

  owner : (is-a?/c clipboard-client%)
如果 owner 目前拥有剪贴板,返回 #t ,否则返回 #f 。

method

(send a-clipboard set-clipboard-bitmap new-bitmap    
  time) → void?
  new-bitmap : (is-a?/c bitmap%)
  time : exact-integer?
将当前剪贴板内容改为 new-bitmap (Windows,Mac OS),并释放当前剪贴板客户端(如果有)。

关于 time 参数的讨论,请参见 剪切和粘贴的时间戳 。 如果 time 超出了平台特定的时间范围, an exn:fail:contract exception is raised 。

method

(send a-clipboard set-clipboard-client new-owner    
  time) → void?
  new-owner : (is-a?/c clipboard-client%)
  time : exact-integer?
改变剪贴板拥有者的客户端:将客户端设置为 new-owner , 并将 new-owner 与当前的事件空间联系起来(由 current-eventspace 决定)。 当客户端不再是当前客户端时,事件空间关联将被移除。

关于 time 参数的讨论,见 剪切和粘贴的时间戳 。 如果 time 超出了平台特定的时间范围, an exn:fail:contract exception is raised 。

method

(send a-clipboard set-clipboard-string new-text    
  time) → void?
  new-text : string?
  time : exact-integer?
将当前剪贴板的内容改为 new-text,并释放当前剪贴板的客户端(如果有的话)。

关于 time 参数的讨论,见 剪切和粘贴的时间戳 。 如果 time 超出了平台特定的时间范围, an exn:fail:contract exception is raised 。

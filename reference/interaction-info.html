18.4 交互配置

 (require racket/interaction-info) package: base
The bindings documented in this section are provided by the racket/interaction-info library, not racket/base or racket.

racket/interaction-info 库提供了一种为 read-eval-print 循环和编辑器注册一个语言配置的方法。

parameter

(current-interaction-info)

 → (or/c #f (vector/c module-path? symbol? any/c))
(current-interaction-info info) → void?
  info : (or/c #f (vector/c module-path? symbol? any/c))
一个 parameter ,为一种语言提供配置,供交互式开发工具使用,如支持语法着色和缩进的命令行评估提示。 这个参数通常由 configure-runtime 模块设置;另见 语言运行时配置 。

current-interaction-info 参数不是直接提供配置信息,而是指定一个要加载的模块, 一个要调用的导出函数,以及要作为参数传递给导出函数的数据。 该函数的结果应该是另一个函数,它接受两个参数: 一个符号一个表示所要求的信息种类的符号(由外部工具定义),以及一个默认值,如果该符号不被识别,通常应该返回。

关于定义一个新的 #lang 语言的信息,见 syntax/module-reader 。

Added in version 8.3.0.2 of package base.

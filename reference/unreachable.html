10.8 不可及的表达式

procedure

(assert-unreachable) → none/c

通过抛出 exn:fail:contract 来报告断言失败,这是 unsafe-assert-unreachable 的安全对等体。

Added in version 8.0.0.11 of package base.

10.8.1 自定义不可达报表

 (require racket/unreachable) package: base
The bindings documented in this section are provided by the racket/unreachable library, not racket/base or racket.

Added in version 8.0.0.11 of package base.

syntax

(with-assert-unreachable
  body ...+)
类似于 (assert-unreachable), 断言不应该到达 body 形式。

除非表达式是包含 (#%declare #:unsafe) 的模块的一部分, 否则它等价于 (let-values () body ...+) 。 其目的是 body forms 将引发 exn:fail:contract 。

当 with-assert-unreachable 表达式是带有 (#%declare #:unsafe) 的模块的一部分时, 它等价于 (unsafe-assert-unreachable) 。

符号法

每个语法形式都由类似 BNF 的符号来描述,该符号描述了(语法包裹的)对、符号和其他数据(不是字符序列)的组合。 这些语法规范如下面的 something 形式的规范所示:

(something id thing-expr ...)
 
  thing-expr : number?

在这种规格内,

  • ... 表示前一个基准点的零次或多次重复; 更一般地说, N 个连续的 ...一行表示前 N 基准点的连续重复。

  • ...+ 表示前面数据的一次或多次重复。

  • 斜体元标识符起到了非终端的作用。 一些元标识符的名称意味着语法上的限制:

    • 一个以 id 结尾的元标识符代表一个标识符。

    • 以 keyword 结尾的元标识符代表一个关键词。

    • 以 expr 结尾的元标识符 (比如 thing-expr) 代表一个被扩展为表达式的子形式。

    • 以 body 结尾的元标识符代表在内部定义上下文中被展开的子形式 (参见 内部定义)。

  • 契约表示对子表达式结果的约束。例如, thing-expr : number? 表示表达式 thing-expr 必须产生一个数字。

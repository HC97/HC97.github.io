8.9 racket/contract/base

 (require racket/contract/base) package: base

racket/contract/base 模块提供了 racket/contract 模块出口的一个子集。特别是,它包含了所有在

  • Data-structure Contracts

  • Function Contracts

  • 将合约附加到值上 和

  • 契约实用程序 部分

不幸的是,使用 racket/contract/base 并不能产生比 racket/contract 小得多的内存占用, 但它仍然有助于将合约添加到 racket/contract 使用的库中,以实现合约系统中一些更复杂的部分。

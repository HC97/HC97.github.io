18.3 交互式帮助

 (require racket/help) package: base
The bindings documented in this section are provided by the racket/help and racket/init libraries, which means that they are available when the Racket executable is started with no command-line arguments. They are not provided by racket/base or racket.

syntax

help

(help string ...)
(help id)
(help id #:from module-path)
(help #:search datum ...)
对于一般的帮助,请看 main documentation page 。

help 表单会搜索文档并打开一个网络浏览器(使用用户选择的浏览器)来显示结果。

关于如何启动用户的浏览器来显示帮助信息,见 net/sendurl 。

一个简单的 help 或 (help) 表单会打开主要的文档页面。

(help string ...) 表单—使用字面字符串,而不是产生字符串的表达式—执行字符串匹配搜索。例如、

(help "web browser" "firefox")

在文档索引中搜索包含 "web 浏览器" 或 "firefox" 短语的引用。

(help id) 表单寻找特定于当前 id 绑定的文档。例如、

(require net/url)
(help url->string)

打开一个 web 浏览器,显示 net/url 库中 url->string 的文档。

为了 help 的目的, for-label 要求引入一个绑定,而不实际执行 net/url 库 —当你想检查文档,但不能或不想运行提供的模块。

(require racket/gui) ; does not work in racket
(require (for-label racket/gui)) ; ok in racket
(help frame%)

如果 id 没有 for-label 和正常的绑定,那么 help 会列出所有已知的为 id 输出绑定的库。

(help id #:from module-path) 变体与 (help id) 类似,但只使用 module-path 的出口。 (module-path 模块是临时命名空间中 for-label 的必要条件)。

(help frame% #:from racket/gui) ; equivalent to the above

(help #:search datum ...) 形式类似于 (help string ...) , 任何非字符串形式的 datum 使用 display 转换为字符串。 没有 datum 被作为表达式进行评估。

比如说、

(help #:search "web browser" firefox)

也会在文档索引中搜索包含 "web浏览器" 或 "firefox" 短语的引用。

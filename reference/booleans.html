4.2 布尔值

真和假的 booleans 分别用 #t 和 #f 来表示, 尽管依赖布尔值的操作通常将 #f 以外的任何东西都视为真。 #t 值总是 eq? 到它自己,而 #f 总是 eq? 到它自己。

See 读取布尔值 for information on reading booleans and 打印布尔值 for information on printing booleans.

参见 and, or, andmap 和 ormap。

procedure

(boolean? v) → boolean?

  v : any/c
如果 v 是 #t 或 #f ,则返回 #t ,否则返回 #f 。

Examples:
> (boolean? #f)

#t

> (boolean? #t)

#t

> (boolean? 'true)

#f

procedure

(not v) → boolean?

  v : any/c
如 果v 是 #f,返回 #t ,否则返回 #f 。

Examples:
> (not #f)

#t

> (not #t)

#f

> (not 'we-have-no-bananas)

#f

procedure

(immutable? v) → boolean?

  v : any/c
如果 v 是一个不可变的 string, byte string, vector, hash table 或 box , 则返回 #t,否则返回 #f 。

请注意, immutable? 不是一个通用的不可变性谓词(尽管它的名字)。 它只适用于少数数据类型,对于这些数据类型,一个谓词—string?, vector? 等–可以同时识别数据类型的可变和不可变的变体。 特别是, immutable? 对 pair 产生 #f , 尽管 pair 是不可变的,因为 pair? 意味着不可变性。

参见 immutable-string?, mutable-string?,等等。

Examples:
> (immutable? 'hello)

#f

> (immutable? "a string")

#t

> (immutable? (box 5))

#f

> (immutable? #(0 1 2 3))

#t

> (immutable? (make-hash))

#f

> (immutable? (make-immutable-hash '([a b])))

#t

> (immutable? #t)

#f

4.2.1 布尔型别名

 (require racket/bool) package: base
The bindings documented in this section are provided by the racket/bool and racket libraries, but not racket/base.

value

true : boolean?

#t 的别名。

value

false : boolean?

对 #f 的别名。

procedure

(symbol=? a b) → boolean?

  a : symbol?
  b : symbol?
返回 (equal? a b) （如果 a 和 b 是符号)。

procedure

(boolean=? a b) → boolean?

  a : boolean?
  b : boolean?
返回 (equal? a b) (如果 a 和 b 是布尔型)。

procedure

(false? v) → boolean?

  v : any/c
返回 (not v)。

syntax

(nand expr ...)

与 (not (and expr ...)) 相同。

Examples:
> (nand #f #t)

#t

> (nand #f (error 'ack "we don't get here"))

#t

syntax

(nor expr ...)

与 (not (or expr ...)) 相同。

在两个参数的情况下,如果参数中没有一个是真值,则返回 #t 。

Examples:
> (nor #f #t)

#f

> (nor #t (error 'ack "we don't get here"))

#f

syntax

(implies expr1 expr2)

检查以确定第一个表达式是否意味着第二个表达式。

与 (if expr1 expr2 #t) 相同。

Examples:
> (implies #f #t)

#t

> (implies #f #f)

#t

> (implies #t #f)

#f

> (implies #f (error 'ack "we don't get here"))

#t

procedure

(xor b1 b2) → any

  b1 : any/c
  b2 : any/c
返回 b1 和 b2 的或异。

如果 b1 和 b2 中正好有一个不是 #f ,那么返回它。否则,返回 #f。

Examples:
> (xor 11 #f)

11

> (xor #f 22)

22

> (xor 11 22)

#f

> (xor #f #f)

#f

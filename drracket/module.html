2.1 在源代码中声明的语言

DrRacket 中的 The Racket Language mode 是一种元语言, 由程序本身指定其语言,通常通过 #lang 行。

更一般地说,使用这种模式时, definitions window 必须包含某种形式的模块。 除了 #lang, 一个 Racket 模块可以写成 (module ...); 除了注释之外, definitions window 必须正好包含一个模块。

2.1.1 初始环境

DrRacket 在开始运行程序前初始化环境的方式与 racket 命令行二进制略有不同,以反映其提供的额外 GUI 功能。 使用语言对话框中的默认设置,它初始化本小节中描述的参数。 改变默认值会导致额外的参数变化,如下一小节所述。

DrRacket 初始化了 pretty-print-size-hook 和 pretty-print-print-hook ,以专门识别

  • snip%, 它只是将其插入编辑器中

  • 来自 pict/convert 的可转换值

  • 来自 file/convertible 的可转换值(它把这些值转换为png图像)

  • 是 exact? 和 real?, 但不是 integer? 的数字, 它专门打印这些数字,以允许显示重复的小数和混合分数,和

  • 匹配传递给 drracket:language:add-snip-value 的谓词的值。

它还初始化了 global-port-print-handler ,使其使用 pretty-print ,而不仅仅是 print 。

DrRacket 初始化了 error-display-handler ,使其特别突出错误的来源位置,并包括提供访问堆栈跟踪的图标。

2.1.2 语言对话框的细节窗格

在语言对话框的细节窗格中,一些配置选项对应于使用各种库,因此可以在没有 DrRacket 的情况下使用。 以下是如何做到的,对于那些简单明了的选项(这里没有提到的选项需要对各种库进行更复杂的配置)。

  • 动态属性: 单选按钮与 errortrace/errortrace-lib 库的各种用途相对应。

    No Debugging or profiling 选项意味着完全不使用这个库。

    Debugging 选项意味着 (current-compile (make-errortrace-compile-handler)) 以及在 use-compiled-file-paths 中加入 (build-path "compiled" "errortrace") 。

    Debugging and profiling 选项意味着使用 errortrace/errortrace-lib 库的 profiling-enabled 与 current-eval 结合使用。

    Syntactic test suite coverage 选项意味着与 current-eval 一起使用 test-coverage-enabled 。

    其他三个复选框保存编译的 .zo 文件并调整编译器。

    populate compiled/ directories 选项对应的是
    (current-load/use-compiled
     (make-compilation-manager-load/use-compiled-handler))
    (manager-skip-file-handler
     (λ (p)
       (file-date-in-paths
        p
        (cons (CACHE-DIR) (current-library-collection-paths)))))
    加上在 use-compiled-file-paths 前面添加 (build-path "compiled" "drracket") 或 (build-path "compiled" "drracket" "errortrace") ,取决于是否设置了 Debugging 选项。

    Preserve stacktrace 选项对应的是

    (compile-context-preservation-enabled #t)

    Enforce constant definitions (enables some inlining) 选项对应于调用 compile-enforce-module-constants; 检查它通过 #t ,不检查它通过 #f 。

  • 输出语法: 输出语法选项对应于 racket/pretty 库和 mzlib/pconvert 库的设置。

  • 集路径: 这对应于设置 current-library-collection-paths 参数。

  • 命令行参数: 这相当于设置当前的命令行参数 current-command-line-arguments 。

  • 自动 #lang 行: 这个面板控制 DrRacket 插入到新窗口(或标签)的默认 #lang 。 如果选择单选框的顶部选项(默认),DrRacket 会使用定义窗口中最后编辑的 #lang 。 如果选择单选框的下端选项,DrRacket 将始终使用文本字段中给出的同一 #lang 行。

1.3 用括号进行编辑

特别是在 Racket 模式下,DrRacket 的编辑器为管理程序中的小括号提供了特殊支持。 当闪烁的光标在小括号旁边时,DrRacket 会对小括号和其匹配的小括号之间的区域进行着色。 这一功能在平衡小括号以完成表达式时特别有用。

尽管空白在 Racket 中并不重要,但 DrRacket 鼓励采用 Racket 代码的特殊格式。 当你输入 Enter 或 Return 时,编辑器会插入一个新行并自动缩进。 要让 DrRacket 重新缩进现有的一行,请将闪烁的光标移到该行,然后按 Tab 键。 (你可以通过选择一个区域并按Tab键来重新缩进整个区域。

DrRacket 还可以在你输入小括号时对其进行改写,以使其更好地匹配。 如果你输入了结尾小括号 ) 、结尾方括号 ] 或结尾大括号 }, 如果 DrRacket 能将其与之前的开头小括号、大括号或小括号相匹配,那么 DrRacket 就会改变你输入的内容, 使之相匹配。DrRacket 也会改写开口方括号,通常是改成开口小括号。 但也有一些例外情况,开放的方括号不会自动改为小括号:

  • 如果方括号在类似于 cond 的关键词之后, 可能会跳过类似于 cond 的表达式中的一些子表达式(例如,在 case 表达式中,方括号从第二个子表达式开始)。

  • 如果方括号在 local 类似关键词之后立即开始一个新的表达式。 请注意,在 local 类似关键字之后的第二个表达式将自动成为一个普通的小括号。

  • 如果方括号在一个小括号之后,该小括号在一个类似 letrec 的关键词之后,

  • 如果方括号在一个序列中,并且序列中前面的 S 表达式是一个复合表达式,DrRacket 会使用与之前相同的小括号、大括号或括号,或

  • 如果方括号位于字符串、注释、字符或符号的中间。

DrRacket 的帮助的结果是,你可以一直使用键盘上的(估计是未移位的)方括号来输入小括号。 例如,当输入

(define (length l)
  (cond
   [(empty? l) 0]
   [else (+ 1 (length (rest l)))]))

如果你总是在出现任何一个方括号或小括号的地方输入 [ 和 ] , DrRacket 就会改变方括号来匹配上面的代码。

当然,这些功能可以在首选项对话框中禁用和定制,参见 首选项 。 另外,为了防止 DrRacket 不能产生你想要的字符,在输入时按住 control 键可以禁用 DrRacket 的小括号、大括号和括号转换器。

4 扩展 DrRacket

DrRacket 支持四种形式的编程环境扩展:键盘绑定、教学包、脚本和插件。

  • Keybindings 菜单项允许添加用户定义的键盘绑定。 关于创建用户定义的键盘绑定的信息,请参见 定义自定义快捷方式 。

  • Quickscript 插件提供的 Scripts 菜单,可以方便地用小型 Racket 脚本扩展 DrRacket, 这些脚本可以在定义(或交互)窗口中使用,也可以用图形与用户交互。 关于创建 scripts 的信息,见 Quickscript, DrRacket 的脚本插件 。

  • teachpack 扩展了 DrRacket 语言中内置的程序集。 例如,一个教学包可以用一个播放声音的程序来扩展初级学生语言。

    教材包在课堂上特别有用,老师可以提供一个为特定练习设计的教材包。 要使用教学包,每个学生必须下载教学包文件,并通过 Language|Add Teachpack... 菜单项选择它。

    有关创建教学包的信息,请参阅 教材包 。

  • plugin 扩展了 DrRacket 环境中的实用程序集。 例如,DrRacket 的 Check Syntax 按钮可以启动一个语法检查插件。 关于创建 plugin 的信息,请参见 DrRacket Plugins。

DrRacket 环境的定制可以通过 Preferences 菜单项来实现, 同时还可以定义新的 color schemes 。 现有的颜色方案包被标记为 tagged “colorscheme” 。

4.1 教材包

教具包的设计是为了补充学生程序中无法用教学语言表达的代码。 例如,为了使学生能够玩刽子手游戏,我们提供了一个教学包,它可以

  • 实现随机选择一个单词、

  • 维护猜错了多少个的状态变量,以及

  • 管理图形用户界面。

所有这些任务在第三周就超出了学生的能力范围,并且/或者将目前无用的知识强加给学生记忆。 然而,刽子手游戏的本质却不是这样。 教具包的使用使学生能够实现这个练习的有趣部分,并且仍然能够享受今天的图形,而不需要无用的记忆。

一个 Racket 源文件定义了一个教学包(尽管该文件可以通过 require 访问其他文件)。 该文件必须包含一个模块(见 模块)。 每个从模块导出的语法定义或值定义都分别作为一个新的原始形式或原始操作提供给用户。

作为一个例子,下面的教学包提供了一个懒惰的 cons 实现。 要测试它,请将以下内容保存在一个文件中,并将该文件作为一个教学包加入(或使用 require)。

#lang racket
 
(provide (rename-out [:lcons lcons]) lcar lcdr)
 
(define-struct lcons (hd tl))
 
(define-syntax (:lcons stx)
  (syntax-case stx ()
    [(_ hd-exp tl-exp)
     #'(make-lcons
               (delay hd-exp)
               (delay tl-exp))]))
 
(define (lcar lcons) (force (lcons-hd lcons)))
(define (lcdr lcons) (force (lcons-tl lcons)))

然后,在这个程序中:

(define (lmap f l)
  (lcons
   (f (lcar l))
   (lmap f (lcdr l))))
 
(define all-nums (lcons 1 (lmap add1 all-nums)))

列表 all-nums 被绑定到一个无限的升序数列表。

关于更多的例子,请看 PLT 安装的 "teachpack" 集合中的 "htdp" 子集合。

4.1.1 在教学包对话框中添加自己的教学包

Language|Add Teachpack... 对话框可以通过两种方式进行扩展。 首先,用户可以通过点击第三列底部的按钮将教学包添加到该列中。 这些添加的内容存储在偏好文件中,所以添加特定地点的教学包的一种方法是提供一个默认的偏好文件。

前两栏也是可扩展的。 当一个集合有一个 "info.rkt" 文件(见 "info.rkt" File Format) , 定义了 htdp-teachpacks 或 2htdp-teachpacks, 那么它们被期望是一个包含要添加到对话框的 teachpacks 的(与集合相关的)路径列表, 或者符号 'all, 这意味着集合中所有以模块后缀(包括 ".rkt", ".ss" 或 ".scm") 都是教学包( "info.rkt" 或 "info.ss" 除外)。

4.1.2 扩展帮助台搜索语境

在 How to Design Programs 中,可以通过在 info.rkt 文件中定义 2htdp:teachpack-modules 绑定来扩展弹出菜单中 Search in Help Desk for ... 项目的搜索上下文(参见 "info.rkt" File Format)。 2htdp:teachpack-modules 绑定应该评估为一个 module paths 的 symbols 列表,以包含在搜索上下文中。

例如,下面这个 info.rkt 文件

#lang info
 
(define scribblings '(("scribblings/intro101.scrbl")))
 
(define 2htdp:teachpack-modules
  '(intro101/file-operations intro101/iterated))

包括在帮助台搜索上下文中的模块 intro101/file-operations 和 intro101/iterated 。

4.2 环境变量

有几个环境变量可以影响 DrRacket 的行为:

  • PLTNOTOOLS : 当这个环境变量被设置时,DrRacket 不会加载任何工具。

  • PLTONLYTOOL : 设置该环境变量时,DrRacket 只加载由环境变量的值命名的集合中的工具。 如果该变量被绑定到一个括号内的集合列表, 则只加载这些集合中的工具(环境变量的内容是 read ,预计是单个符号或符号列表)。

  • PLTDRREPL : 当这个环境变量被设置时,DrRacket 会启动一个读-评-印的循环, 其中包括 racket 模块和 DrRacket Plugins 中描述的形式和函数。

    如果没有设置为 -q, 那么也会加载 (find-system-path 'init-file) 。

    会创建一个新的线程来评估 REPL 表达式,所以一定要使用 queue-callback 来评估改变 GUI 的表达式(以避免竞赛条件)。

  • PLTDRCM : 设置该环境变量后,DrRacket 会在启动前安装编译管理器, 这意味着随着 DrRacket(或工具)的源代码被修改, ".zo" 文件会自动保持更新。

    如果该变量被设置为 trace ,则会显示实际被重新编译的文件。

  • PLTDRPAR : 设置该环境变量后, DrRacket 会使用 parallel-compile-files 来并行编译框架和 drracket 集合, 然后在启动前安装编译管理器,这意味着随着 DrRacket(或工具)源代码的修改, ".zo" 文件会自动保持最新状态。

    如果该变量被设置为 trace ,那么将显示实际被重新编译的文件。

  • PLTDRDEBUG : 设置该环境变量后,DrRacket 启动时启用错误跟踪功能。 如果该变量被设置为 profile, DrRacket 也会记录自身的剖析信息。

  • PLTDRPROFILE : 设置此环境变量时, DrRacket 使用 profile 库(有一个 小GUI)来收集自身的剖析信息。

  • PLTDRBREAK : 设置此环境变量后,DrRacket 会在启动时创建一个带有中断按钮的窗口。 点击该按钮会中断 DrRacket 事件空间的主线程。这与 PLTDRDEBUG 结合使用效果很好,因为会报告中断的来源位置。

  • PLTSTRINGCONSTANTS : 设置该环境变量后,DrRacket 会打印出尚未翻译的字符串常量。 如果它被设置为某一特定语言(对应于 "string-constants" 集合中的某个文件), 则只显示与该语言相匹配的未设置的字符串常量。

    这个环境变量必须在制作 ".zo" 文件时设置。 为了确保你能正确看到它的输出,用 -c 标志运行 raco setup ,设置环境变量,然后再运行 raco setup 。

  • PLTDRXREFDELAY : 当这个环境变量被设置时, DrRacket 使用普通的 delay (而不是 delay/idle) 来延迟搜索指数的计算。 这意味着 Check Syntax 第一次启动会比较慢,但启动性能更容易预测。 此外,当环境变量被设置时,DrRacket 将打印出该变量被设置,并打印出索引开始加载的时间和加载结束的时间。

  • PLTDREASTERSECONDS : 当这个环境变量被设置时, DrRacket 会假装 current-seconds 的结果实际上是这个环境变量的值,以达到复活节的目的。 例如,将其设置为 1339390801 将模拟 2012 年卡美哈美哈国王日,并显示相应的复活节彩蛋闪屏。

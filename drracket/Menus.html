3.1 Menus

3.1.1 File

  • New : 创建一个新的 DrRacket 窗口。

  • New Tab : 在当前 DrRacket 窗口中创建一个新标签。

  • Open... : 打开查找文件对话框,选择文件载入 definitions window 。

  • Open Recent : 列出最近打开的文件。选择其中一个文件可以打开该文件进行编辑。

  • Open Require Path... : 打开一个对话框, 你可以输入一个类似 require 的模块路径(例如 racket/base.rkt 或 data/splay-tree.rkt 或 "x.rkt")并在 definitions window 中编辑相应的文件。

  • Reopen Closed Tab : 打开以前关闭的标签。 如果在当前会话中没有关闭的标签,则会打开以前会话中的文件。

  • Install PLT File... : 打开一个对话框, 询问 ".plt" 文件的位置(在本地磁盘或网络上)并安装该文件的内容。

  • Install Package... : 打开一个对话框,询问要安装的包的名称。 关于软件包的更多信息请参见 Racket 中的包管理 。

  • Package Manager... : 为命令行工具 raco pkg 提供一个图形界面。

  • Revert : 重新加载当前在 definitions window 中的文件。 自该文件最后一次被保存以来的所有变化都将丢失。

  • Save Definitions : 将 definitions window 中的程序保存。 如果该程序以前从未被保存过,会出现一个保存文件的对话框。

  • Save Definitions As... : 打开一个保存文件的对话框,选择一个目标文件来保存定义窗口中的程序。 随后的保存会写入新选择的文件。

  • Save Other : 包含这些子项目

    • Save Definitions As Text... : 和 Save Definitions As... 一样, 但文件是以纯文本格式保存的(见 程序文件)。 后续的保存也是以纯文本格式写入。

    • Save Interactions : 将交互作用窗口的内容保存到一个文件中。 如果以前没有保存过交互作用的常量,会出现一个保存文件的对话框。

    • Save Interactions As... : 打开一个保存文件的对话框,选择一个目标文件来保存交互窗口的内容。 随后的保存会写入新选择的文件。

    • Save Interactions As Text... : 与 Save Interactions As... 类似, 但文件是以纯文本格式保存的(见 程序文件)。 后续的保存也是以纯文本格式写入。

  • Log Definitions and Interactions... : 启动运行交互和定义窗口的文本日志,按执行情况组织。 在你选择的目录中,DrRacket 会在你与不同程序交互时,以 "01-definitions", "01-interactions", "02-definitions", "02-interactions" 等名称保存文件。

  • Page Setup... : 打开一个对话框来配置打印的各种细节。

  • Print Definitions... : 打开一个对话框,用于打印 definitions window 中的当前程序。

  • Print Interactions... : 打开一个对话框,用于打印 interactions window 的内容。

  • Search in Files... : 打开一个对话框,您可以指定多文件搜索的参数。 搜索结果将显示在单独的窗口中。

  • Close Window : 关闭此 DrRacket 窗口。 如果该窗口是唯一打开的 DrRacket 窗口,则 DrRacket 将退出,Mac OS 除外。

  • Close Tab : 关闭该 DrRacket 窗口中的活动标签。

  • {Quit 或 Exit} 退出 DrRacket。(在 Mac OS 上,该菜单项在 Apple 菜单中。)

3.1.2 编辑

所有 Edit 菜单项都在定义窗口或交互窗口中操作,取决于选择或闪烁的光标的位置。 每个窗口都有自己的撤销和重做历史。

  • Undo : 撤销一个编辑动作。每个窗口都有一个操作历史,所以多个 Undo 操作可以撤销多个编辑操作。

  • Redo : 逆转一个 Undo 操作。 每个窗口(和盒式子窗口)都保持自己的 Undo 操作历史, 所以多个 Redo 操作可以逆转多个 Undo 操作。

  • Cut : 将选定的文本复制到剪贴板上,并从窗口中删除。

  • Copy : 将选定的文本复制到剪贴板上。

  • Paste : 将当前剪贴板的内容粘贴到窗口中。

  • Paste and Indent : 将当前剪贴板的内容粘贴到窗口中并缩进,与 Tab 键的作用相同。

  • Delete : 或 Clear : 删除选中的文本。

  • Select All : 突出显示缓冲区内的全部文本。

  • Wrap Text : 在窗口中的包裹文本和未包裹的文本之间进行切换。

  • Find : 在框架的底部打开一个交互式搜索窗口,并将插入点移到搜索字符串编辑器中(如果插入点已经在那里,则移出它)。

    参见 (part "Searching") 。

  • Find From Selection : 就像 Find 一样,除了它将当前的选择复制到搜索窗口 (但不使用剪贴板,所以粘贴将粘贴最后被复制的东西,而不是新的搜索字符串)。

  • Find Next : 查找搜索窗口中下一个出现的文本。

  • Find Previous : 查找搜索窗口中下一个出现的文本,但要向后搜索。

  • Show Replace :

  • Replace : 用替换字符串替换所选内容(如果它与查找字符串相匹配), 并查找最后一次搜索的文本的下一次出现,向前查找。

  • Replace All : 用替换字符串替换所有搜索字符串的出现。

  • Find Case Sensitive : 在区分大小写和不区分大小写的搜索之间进行切换。

  • Find Longest Line : 将插入点移动到编辑器中最长的行。

  • Keybindings :

    • Show Active Keybindings : 显示当前窗口中所有可用的键盘绑定。

    • Add User-defined Keybindings... : 选择这个菜单项可以打开一个文件对话框, 你可以选择一个包含 Racket-definitions of keybindings 的文件。更多信息请参见 定义自定义快捷方式 。

  • Spell Check String Constants : 使用 aspell 或 ispell 来寻找字符串常量中的未知词(否则会被染成绿色的东西)。

  • Spell Check Text (between {} in Scribble : 使用 aspell 或 ispell 来查找 Scribble 文本中的未知单词。

  • Skip to Next Misspelled Word : 将插入点设置为编辑器中拼写错误的下一个单词(如果有的话)。

  • Suggest Spelling Corrections... : 打开一个窗口,显示插入点上的单词的拼写更正。

  • Spelling Dictionaries : 一个子菜单,列出所有可用的拼写字典。

  • Complete Word : 补全插入点上的单词,使用手册作为补全的来源。

  • Preferences... : 打开首选项对话框。参见 首选项 。 (在Mac OS上,这个菜单项在苹果菜单中)。

3.1.3 查看

在任何时候,以下显示/隐藏的菜单项目对各出现一个。

  • Toolbar :

    • Toolbar on Left : 将工具栏(默认在 DrRacket 窗口的顶部)移到左侧,垂直排列。

    • Toolbar on Top : 将工具栏移到 DrRacket 窗口的顶部。

    • Toolbar on Right : 将工具栏移到右侧,垂直排列。

    • Toolbar Hidden : 完全隐藏工具栏。

  • Split : 将当前窗口一分为二,使当前窗口的两个不同部分同时可见。

  • Collapse : 如果窗口之前已经被分割,这个菜单项就会被启用,允许你折叠分割的窗口。

  • Show Definitions : 显示定义窗口。

  • Hide Definitions : 隐藏定义窗口。

  • Show Interactions : 显示交互窗口。

  • Hide Interactions : 隐藏交互窗口。

  • Use Vertical Layout : 和 Use Horizontal Layout : 调整定义和交互窗口,使它们彼此相邻或定义在交互窗口之上。

  • Show Log : 显示当前的日志信息。

  • Hide Log : 隐藏当前的日志信息。

  • Show Tracing : 显示自上次点击 Run 后调用的函数的追踪。 这个菜单只有在你在 Choose Language... 对话框的 Details 部分启用了跟踪功能时才有用。 剖析并不适用于所有语言。

  • Hide Tracing : 隐藏跟踪显示。

  • Show Profile : 显示当前的剖析报告。 只有在 Choose Language... 对话框的 Details 部分中启用了剖析功能,此菜单才有用。 剖析并非适用于所有语言。

  • Hide Profile : 隐藏当前在 DrRacket 窗口中显示的任何剖析信息。

  • Show Program Contour : 沿 DrRacket 窗口边缘显示一个 "20,000英尺" 的概览窗口。 该窗口中的每个像素与程序文本中的一个字母相对应。

  • Hide Program Contour : 隐藏轮廓窗口。

  • Show Line Numbers : 在定义窗口中显示线号。

  • Hide Line Numbers : 隐藏定义窗口中的行号。

  • Show Column Width Guide at 102 Characters : 当当前文件的宽度大于 102 个字符时,显示列宽指南。

    102 这个数字在首选项对话框中的 Editing 标签的 General Editing 子标签中控制。

  • Hide Column Width Guide for 102 Characters : 隐藏列宽指南,即使文件的宽度大于 102 个字符。

    102 这个数字在偏好设置对话框中的 Editing 选项卡的 General Editing 子选项卡中控制。

  • Show Module Browser : 在 DrRacket 中显示当前打开文件的模块 DAG。

    参见 模块浏览器 。

  • Hide Module Browser : 隐藏模块浏览器。

    参见 模块浏览器 。

注意:每当程序运行时,如果交互窗口被隐藏,就会变成可见。

3.1.4 语言

  • Choose Language... : 打开一个选择当前评估语言的对话框。 点击 Run ,使语言在交互窗口中激活。 关于语言的更多信息,请参见 选择一种语言 。

  • Add Teachpack... : 打开一个查找文件的对话框,选择一个教学包来扩展当前的语言。 点击 Run ,使教学包在交互窗口中可用。 有关创建教学包的信息,请参见 扩展 DrRacket 。

  • Clear All Teachpacks : 清除当前所有的教学包。点击 Run 以清除交互窗口中的教学包。

除了上述项目外,每个教学包都有一个菜单项,只清除相应的教学包。

3.1.5 Racket

  • Run : 重置交互窗口,运行定义窗口中的程序。

  • Ask the Program to Quit : 打破当前的评估。

  • Force the Program to Quit : 终止当前的评估。

  • Reload #lang Extensions : 丢弃任何已经加载的 #lang 特定扩展,并重新加载它们。 参见 DrRacket support for #lang-based Languages 。

  • Limit Memory... : 允许你指定在 DrRacket 中运行的程序允许消耗的内存量的限制。

  • Clear Error Highlight : 删除提示错误来源位置的红色背景。 同时也会删除程序中未覆盖(未测试)的部分的高亮显示。

  • Create Executable... : 创建一个单独的启动器来运行你的程序。 更多信息请参见 创建可执行文件 。

  • Module Browser... : 提示输入一个文件,然后打开一个窗口,显示从所选文件中的模块开始的模块 DAG。

    参见 模块浏览器 。

  • Module Browser on file : 打开一个单独的窗口,显示从当前在 DrRacket 中编辑的文件开始的模块图,但使用磁盘上保存的文件,而不是 DrRacket 中的版本。

    参见 模块浏览器 。

  • Reindent : 根据标准的 Racket 格式惯例缩进选定的文本。(按 Tab 键有同样的效果)。

  • Reindent All : 根据选择的位置或闪烁的光标,缩进定义或交互窗口中的所有文本。

  • Comment Out with Semicolons : 在每一行选定的文本的开头都加上 ; 字符。

  • Comment Out with a Box : 用一个注释框框住所选的文本。

  • Uncomment : 移除每行选定文本开头的所有 ; 字符或移除文本周围的注释框。 取消注释只删除出现在行首的 ; ,并且只删除每行的第一个 ; 。

  • Disable Tests : 当程序运行时,停止在定义窗口中编写的测试。 测试可以通过 Enable Tests 菜单项来启用。 禁用测试将冻结任何现有测试报告窗口的内容。

  • Enable Tests : 允许定义窗口中的测试在程序运行时进行评估。 可以使用 Disable Tests 菜单项来禁用测试。

3.1.6 插入

  • Insert Comment Box : 插入一个被 DrRacket 忽略的框,用它来为阅读程序的人写评论。

  • Insert Image... : 打开查找文件对话框,选择 GIF、BMP、XBM、XPM、PNG 或 JPG 格式的图像文件。 该图像被当作一个值处理。

  • Insert Fraction... : 打开一个混合符号分数的对话框,并将给定的分数插入到当前编辑器中。

  • Insert Large Letters... : 打开一行文本的对话框,并插入大字体的文本(使用分号和空格)。

    对话框的大部分内容都是不言自明的:在顶部的空格中键入分号,以预览底部区域的分号。 字体选择项中的数字显示了字母 "w" 在给定字体中的(相对)宽度,以帮助你挑选出更多的粗体字(这些字体往往看起来更好)。

  • Insert λ : 将符号 λ (作为 Unicode 字符)插入程序中。 λ 符号通常和 lambda 一样被绑定。

  • Insert XML Box : 插入一个 XML;更多信息见 XML 框和 Racket 框 。

  • Insert Racket Box : 插入一个包含 Racket 代码的盒子,通常用于 XML 盒子内; 参见 XML 框和 Racket 框 。

  • Insert Racket Splice Box : 插入一个包含 Racket 代码的盒子,通常用于 XML 盒子内; 另见 XML 框和 Racket 框 。

3.1.7 窗口

  • Bring Frame to Front... : 打开一个窗口,列出所有打开的 DrRacket 框架。 选择其中一个,可将该窗口移到前面。

  • Most Recent Window : 在当前关注的窗口和最近关注的窗口之间进行切换。

在上述菜单项目之后,该菜单包含了 DrRacket 中每个窗口的条目。 选择一个菜单项可以将相应的窗口带到前面。

3.1.8 帮助

  • Help Desk : 打开帮助台。 这是关于 DrRacket 及其语言的所有文件的交流中心。

  • About DrRacket... : 显示 DrRacket 的荣誉。

  • Check for Updates... : 检查 DrRacket 的新版本是否已经发布。

  • Related Web Sites : 提供相关网站的链接。

  • Tool Web Sites : 提供已安装工具的网站链接。

  • Submit Bug Report... : 向 Racket 开发团队发送错误报告的首选机制。 它自动收集有关您的 Racket 安装的信息,有时可以帮助诊断问题。

  • Saved Bug Reports : 一个子菜单,用于保存您已经开始编辑但还没有发送的错误报告。

  • Saved Bug Reports : 一个子菜单,用于保存您已经开始编辑但尚未提交的错误报告。

  • Configure Command Line for Racket... : 这个菜单项在 Mac OS 和 Windows 下都可用。

    在 MacOS 上,在提示你输入密码后,它会在 "/etc/init.d/" 中添加一个名为 "racket" 的文件, 其中包含 "bin" 目录的位置。这具有将该路径添加到默认的 PATH 环境变量的效果(除非你的 shell 明确改变了默认行为)。

    在 Windows 中,它改变了 HKEY_CURRENT_USER\Environment\Path 注册表键, 以添加 "bin" 目录的位置。完成后,新创建的 command.com shells 的路径中应该有 Racket。

  • Interact with DrRacket in English : 将 DrRacket 的界面改为使用英语; 该菜单项仅在当前语言不是英语时出现。 其他菜单项可将 DrRacket 切换为其他语言。

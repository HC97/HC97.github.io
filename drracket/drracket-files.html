3.5 DrRacket 文件

3.5.1 程序文件

Racket 程序文件的标准 file extension 是 ".rkt" 。 扩展名 ".ss",".scm" 和 ".sch" 在历史上也很流行。

DrRacket 的编辑器可以用两种不同的格式保存程序文件:

  • Plain-text file format — 所有文本编辑器都可以读取这种格式。 DrRacket 默认以纯文本格式保存程序,除非程序包含图像或文本框。 (纯文本格式不保留图像或文本框)。

    纯文本格式是针对平台的,因为不同的平台有不同的换行习惯。 然而,大多数跨平台移动文件的工具支持 "文本" 传输模式,可以正确调整换行。

  • Multimedia file format — 这种格式是 DrRacket 特有的,其他编辑器无法识别它。 当程序包含图像、文本框或格式化文本时,DrRacket 默认以多媒体格式保存程序。

    多媒体格式与平台无关,它使用 ASCII 编码(因此不同的传输方式不可能损坏文件)。

3.5.2 备份和首次更改文件

每隔30秒,DrRacket 会检查每个打开的文件。 如果任何文件被修改而没有被保存,DrRacket 会将文件内容保存在备份文件中,以备发生断电或其他灾难性错误。 如果后来保存了原文件,或用户退出 DrRacket 并明确拒绝保存该文件,则备份文件将被删除。 备份文件与原文件保存在同一目录下,备份文件的名称根据原文件的名称生成

  • 在 Unix 和 Mac OS 上,文件名的开头和结尾会加上一个 "#" ,然后在结尾的 "#" 后面加上一个数字, 然后在数字后面再加上一个 "#" 。 选择这个数字是为了使备份文件名独一无二。

  • 在 Windows 中,文件的扩展名被替换为数字,以使备份文件名独一无二。

在 DrRacket 的实现中,这些文件被称为 "自动保存" 文件,而不是备份文件。

如果定义窗口被修改,且没有当前文件,那么备份文件将被写入用户的 “documents” 目录。 “documents” 目录由 (find-system-path 'doc-dir) 决定。

当DrRacket首次启动时,它会查看是否还有备份文件存在,如果有,就会提出恢复备份文件,前提是 DrRacket 是在无意中关闭的。 恢复文件时,DrRacket 会将原文件的当前版本移出(在名称中加入 “autorec” ), 并将备份文件移至原文件的位置。

另外,DrRacket 还创建了 "首次更改" 文件。当你在 DrRacket 中修改现有文件并保存时,如果不存在这样的文件, DrRacket 会将旧版本的文件复制到一个特殊的首改文件中。 首次更改文件与原文件保存在同一目录下,首次更改文件的名称由原文件的名称生成:

  • 在 Unix 和 Mac OS 上,文件名的末尾会加上一个 "~" 。

  • 在 Windows 上,文件的扩展名被替换为 ".bak" 。

3.5.3 首选项文件

启动时,DrRacket 会从首选项文件中读取配置信息。偏好文件的名称和位置取决于平台和用户:

表达式 (find-system-path 'pref-file) 会返回平台和用户的偏好文件路径。

  • 在 Unix 上,首选项存储在用户主目录下的 ".racket" 子目录中, 文件为 "racket-prefs.rktd" 。

  • 在 Windows 上,首选项存储在用户的应用数据文件夹中的 "Racket" 子目录下的 "racket-prefs.rktd" 文件中,该目录由 Windows 注册表指定; 应用数据文件夹通常是用户配置文件目录下的 "Application Data" ,该目录通常在 Windows GUI 中隐藏。

  • 在 Mac OS 上,首选项存储在用户的首选项文件夹中的 "org.racket-lang.prefs.rktd" 。

在修改首选项文件时,会使用一个锁文件,它与首选项文件创建在同一目录下。 在 Windows 中,锁文件被命名为 "_LOCKracket-prefs.rktd"; 在 Unix 中,它是 ".LOCK.racket-prefs.rktd"; 在 Mac OS 中,它是 ".LOCK.org.racket-lang.prefs.rktd" 。

如果用户特定的首选项文件不存在,而 "defaults" 集合中的 "racket-prefs.rktd" 文件存在,那么它将用于初始首选项设置。 (关于集合的更多信息,请参见 Libraries and Collections 。) 因此,这个文件允许针对特定站点的首选项默认值的配置。要设置这样的配置,请启动 DrRacket 并根据你的喜好配置偏好。 然后,退出 DrRacket,将你的偏好文件复制到 "defaults" 集合中,成为 "racket-prefs.rktd" 。 之后,已经没有偏好设置的用户会得到你选择的偏好设置。

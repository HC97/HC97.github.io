16 raco read: 读取和打印

raco read 命令 read 并打印指定文件的内容。 这个命令对于展示 #reader 或基于 #lang 的阅读器扩展如何将输入转换为 S 表达式很有用。 它对于漂亮地打印一个已经是 S 表达式形式的术语也很有用。

命令行标志:

  • -n ‹n› 或 --columns ‹n› — 为具有 ‹n› 列的显示格式输出

  • -h 或 --help — 显示此命令的帮助信息

  • -- — 不把其余参数当作开关

Added in version 1.3 of package compiler-lib.

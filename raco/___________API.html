1.11 用于读取编译依赖项的 API

 (require compiler/depend) package: base
compiler/depend 模块提供了一个检查和遍历由 raco make, raco setup 或 compiler/cm 生成的依赖信息的功能。

Added in version 6.90.0.13 of package base.

procedure

(module-recorded-dependencies module-file)

 → (listof (and path? (complete-path? path?)))
  module-file : path?
给出一个用 raco make, raco setup 或 compiler/cm 编译的文件的 module-file ,通过读取和遍历编译时留下的依赖信息文件,返回 module-file 的依赖信息列表。

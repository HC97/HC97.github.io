class

text-field% : class?

  superclass: object%

  extends: control<%>
text-field% 对象是一个可编辑的文本字段,前面显示一个可选的标签。 有两种文本字段风格:

  • 单行文本是可见的,当用户按下 Return 或 Enter(当文本字段有焦点时)时, 会产生一个特殊的控制事件,该事件不被文本字段的框架或对话框处理 (见顶层窗口中的 on-traverse-char in top-level-window<%> )。

  • 多行文本是可见的,Enter 不被特别处理。

每当用户改变一个文本字段的内容时,它的回调过程被调用。 当每个文本字段被创建时,一个回调过程被作为初始化参数提供。

文本字段是用 text% 编辑器实现的(有一个无法访问的显示)。 因此, text-field% 只提供 get-value 和 set-value 来操作文本字段中的文本, 而 get-editor 返回字段的编辑器,它提供了大量的方法来对文本进行更复杂的操作。

文本字段的编辑器的键图是通过调用当前键图初始化程序来初始化的, 该程序由 current-text-keymap-initializer 参数决定。

如果 label 不是 #f, 它被用作文本字段的标签。否则,文本字段不显示其标签。

如果 & 出现在 label 中,就会像 button% 一样被特别解析。

当用户改变文本字段中的文本或按下回车键时, callback 过程被调用(回车键不是由文本字段的框架或对话框处理的; 见 on-traverse-char in top-level-window<%>)。 如果用户按了回车键,传递给回调的事件类型是 'text-field-enter, 否则是 'text-field 。

如果 init-value 不是 "", 文本项目的 graphical minimum size 将被设定为足够宽以显示 init-value 。 否则,会选择一个内置的默认宽度。 对于单行模式的文本字段, graphical minimum size 被设置为显示一行,而且默认情况下只有控件的宽度是可以拉伸的。 对于一个多行文本字段, graphical minimum size 显示三行文本,而且默认情况下它可以向两个方向拉伸。

样式必须正好包含 'single 或 'multiple 中的一个; 前者指定一个单行字段,后者指定一个多行字段。 'hscroll 样式只适用于多行字段;当 'hscroll 被指定时, 该字段有一个水平滚动条,自动包装被禁用;否则,该字段没有水平滚动条,自动包装被启用。 一个多行文本字段总是有一个垂直滚动条。 'password 样式表示该字段应该使用一个通用符号而不是实际的字符来绘制其内容的每个字符。 如果 style 包括 'vertical-label, 那么 text field 会在控件上方创建标签; 如果 style 不包括 'vertical-label (可选择包括 'horizontal-label), 那么标签会在 text field 左侧创建。 如果 style 包含 'deleted, 那么 text field 就被创建为隐藏的,它不会影响其父辈的几何形状; text field 可以通过调用 parent 的 add-child 方法,在以后变成活动的。.

font 参数决定了控件的字体。 关于 enabled 参数的信息,请参见 window<%> 。 关于 horiz-margin 和 vert-margin 参数的信息,请参见 subarea<%> 。 关于 min-width, min-height, stretchable-width 和 stretchable-height 参数的信息,请参见 area<%> 。

method

(send a-text-field get-editor) → (is-a?/c text%)

返回用于实现文本字段的编辑器。

对于一个文本字段来说, text% 对象最有用的方法是以下这些:

  • (send a-text get-text) 返回编辑器的当前文本。

  • (send a-text erase) 删除编辑器中的所有文本。

  • (send a-text insert str) 将 str 插入到编辑器的当前光标位置。

method

(send a-text-field set-field-background color) → void?

  color : (or/c (is-a?/c color%) #f)
将字段的可编辑区域的背景色设置为 color 。 如果 color 是 #f ,则在黑暗模式下将背景色设置为黑色,如果不是在黑暗模式下则设置为白色。

method

(send a-text-field set-value val) → void?

  val : string?
设置文本字段中当前的文本。 (控件的回调过程 不 被调用)。

A text field’s value 可以被 the user typing into the control 改变,这种改变不经过这个方法;使用 control 回调程序(作为初始化参数提供) 来监控 value 的变化。

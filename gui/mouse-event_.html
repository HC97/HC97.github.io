class

mouse-event% : class?

  superclass: event%

一个 mouse-event% 对象封装了一个鼠标事件。 鼠标事件主要由 on-subwindow-event in window<%> 和 on-event in canvas<%> 。

参见 鼠标和键盘事件 。

为一个特定类型的事件创建一个鼠标事件。 事件类型有:

  • 'enter — 鼠标指针进入窗口

  • 'leave — 鼠标指针离开窗口

  • 'left-down — 鼠标左键被按下

  • 'left-up — 鼠标左键释放

  • 'middle-down — 鼠标中键被按下

  • 'middle-up — 鼠标中键释放

  • 'right-down — 按下鼠标右键(Mac OS:按住控制键点击)

  • 'right-up — 释放鼠标右键(Mac OS:按下控制键释放)

  • 'motion — 鼠标移动,有无按下的按钮

关于 left-down, middle-down, right-down, x, y, shift-down, control-down, meta-down, alt-down, time-stamp, caps-down, mod3-down, mod4-down 和 mod5-down 的信息, 请参见相应的 get- 和 set- 方法。

Changed in version 1.1 of package gui-lib: 增加了 mod3-down, mod4-down 和 mod5-down 。

method

(send a-mouse-event button-changed? [button]) → boolean?

  button : (or/c 'left 'middle 'right 'any) = 'any
如果这是一个鼠标按键的按下或释放事件(即输入 'left-down, 'left-up, 'middle-down, 'middle-up, 'right-down 或 'right-up) ,返回 #t ,否则返回 #f 。 参见 button-up? 和 button-down? 。

如果 button 不是 'any, 那么只有当它是一个特定按钮的按下或释放事件时才会返回 #t 。

method

(send a-mouse-event button-down? [button]) → boolean?

  button : (or/c 'left 'middle 'right 'any) = 'any
如果是按下按钮的事件(即输入 'left-down, 'middle-down 或 'right-down) ,则返回 #t ,否则返回 #f 。

如果 button 不是 'any, 那么只有当它是一个特定按钮的按压事件时,才会返回 #t 。

method

(send a-mouse-event button-up? [button]) → boolean?

  button : (or/c 'left 'middle 'right 'any) = 'any
如果事件是释放按钮(即输入 'left-up, 'middle-up 或 'right-up), 返回 #t ,否则返回 #f 。 (正如 鼠标和键盘事件 部分所指出的,按钮释放事件有时会被放弃。)

如果 button 不是 'any, 那么只有当它是一个特定按钮的释放事件时才会返回 #t 。

method

(send a-mouse-event dragging?) → boolean?

如果这是一个拖动事件,则返回 #t : 当一个按钮被按下时,键入 'motion ( get-left-down, get-middle-down 或 or get-right-down 所报告的),否则返回 #f 。

method

(send a-mouse-event entering?) → boolean?

如果这个事件是鼠标进入一个窗口(即输入 'enter),返回 #t ,否则返回 #f 。

当鼠标按钮向上时,Enter/leave 事件通知一个窗口它将开始/停止接收鼠标事件。 然而,当鼠标按钮向下时,接收到鼠标下拉事件的窗口会接收所有的鼠标事件,直到按钮被释放; 回车/离开事件不会被发送到其他窗口,也不会被可靠地传递给处理点击的窗口 (因为该窗口可以通过 get-x 和 get-y 检测其区域外的移动)。 另见 鼠标和键盘事件 。

method

(send a-mouse-event get-alt-down) → boolean?

如果 Option(Mac OS)键在该事件中被按下,则返回 #t 。 当 Alt 键在 Windows 中被按下时,它被报告为一个 Meta 键(见 get-meta-down) 。

method

(send a-mouse-event get-caps-down) → boolean?

如果事件中 Caps Lock 键是打开的,则返回 #t 。

method

(send a-mouse-event get-control-down) → boolean?

如果事件中 Control 键被按下,则返回 #t 。

在 Mac OS 上,如果控制键被按下与鼠标按键相结合,该事件被报告为右键点击, get-control-down 对该事件的报告为 #f 。

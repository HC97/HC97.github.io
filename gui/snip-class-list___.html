interface

snip-class-list<%> : interface?

每个事件空间都有自己的 snip-class-list<%> 的实例, 通过 (get-the-snip-class-list) 获得。 新的实例不能被直接创建。 每个实例都保持一个狙击类的列表。 这个列表在从文件中加载狙击时是需要的。参见 剪辑类 。

method

(send a-snip-class-list add snipclass) → void?

  snipclass : (is-a?/c snip-class%)
添加一个狙击类到列表中。 如果一个同名的类已经存在于列表中,这个类将不会被添加。

method

(send a-snip-class-list find name)

 → (or/c (is-a?/c snip-class%) #f)
  name : string?
从列表中找到一个具有给定名称的狙击类,如果没有找到则返回 #f 。

method

(send a-snip-class-list find-position class)

 → exact-nonnegative-integer?
  class : (is-a?/c snip-class%)
返回列表中指定类的索引。

method

(send a-snip-class-list nth n)

 → (or/c (is-a?/c snip-class%) #f)
  n : exact-nonnegative-integer?
返回列表中的第 n 个类,如果列表中有 n 个类或更少,则返回 #f 。

method

(send a-snip-class-list number) → exact-nonnegative-integer?

返回列表中 Snip 类的数量。

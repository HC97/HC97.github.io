class

style-delta% : class?

  superclass: object%

一个 style-delta% 对象封装了一个风格变化。 可由 delta 表达的变化包括:

  • 改变字体家族

  • 改变字体的表面

  • 改变字体大小到一个新的值

  • 通过一个加法量来扩大字体

  • 将字体放大到一个乘法量,等等。

  • 改变字体风格 (normal, italic 或 slant)

  • 切换字体样式

  • 如果字体目前是 slant,则将其改为 italic ,等等。

  • 改变字体重量,等等。

  • 改变下划线,等等。

  • 改变垂直排列,等等。

  • 改变前景颜色

  • 调暗或调亮前景色,等等。

  • 改变背景颜色,等等。

  • 改变文本背景的透明度

set-delta 方法对大多数风格 delta 设置来说是很方便的; 它接受一个高级 delta 规范,并设置内部 delta 信息。

为了充分利用样式三角洲,有必要了解可以通过 set-weight-on 等方法操作的内部开/关设置。 例如,字体重量的变化是通过 weight-on 和 weight-off 的内部设置指定的。 粗略地说,当一个重量设置不存在时, weight-on 会打开它,而当一个重量设置存在时, weight-off 会关闭它。 这两者以下述方式精确地相互作用:

  • 如果 weight-on 和 weight-off 都被设置为 'base, 那么字体的重量就不会改变。

  • 如果 weight-on 不是 'base, 那么权重就被设置为 weight-on 。

  • 如果 weight-off 不是 'base, 那么当基本样式的权重为 weight-off 时,权重将被设回 'normal 。

  • 如果 weight-on 和 weight-off 都被设置为相同的值,那么权重会根据该值进行切换: 如果基本样式有 weight-on, 那么权重会被改变为 'normal; 如果基本样式有不同的权重,它将被改变为 weight-on 。

  • 如果 weight-on 和 weight-off 都被设置了,但是是不同的值, 那么只有当基本样式有 weight-on 的时候,权重才会被改变为 weight-off 。

字体样式、平滑、下划线和对齐方式以类似的方式工作。

alignment-on 和 alignment-off 的可能值是:

  • 'base

  • 'top

  • 'center

  • 'bottom

style-on 和 style-off 的可能值是:

  • 'base

  • 'normal

  • 'italic

  • 'slant

smoothing-on 和 smoothing-off 的可能值是:

  • 'base

  • 'default

  • 'partly-smoothed

  • 'smoothed

  • 'unsmoothed

underlined-on 和 underlined-off 的可能值是:

  • #f (行为像 'base)

  • #t

size-in-pixels-on 和 size-in-pixels-off 的可能值是:

  • #f (行为像 'base)

  • #t

transparent-text-backing-on 和 transparent-text-backing-off 的可能值是:

  • #f (行为像 'base)

  • #t

weight-on 和 weight-off 的可能值是:

  • 'base

  • 'normal

  • 'bold

  • 'light

样式delta中的族和面的设置是相互依赖的:

  • 当一个 delta 的面是 #f ,它的族是 'base, 那么面和族都不会被 delta 修改。

  • 当 delta 的面是一个字符串,它的族是 'base, 那么只有面被 delta 修改。

  • 当 delta 的族不是 'base 时,那么面和族都被 delta 修改。 如果 delta 的面是 #f, 那么应用 delta 将一个样式的面设置为 #f, 这样在选择字体时就会以家族设置为准。

method

(send a-style-delta collapse delta) → boolean?

  delta : (is-a?/c style-delta%)
试图将给定的 delta 之后应用这个 delta 的变化折叠成一个单一的 delta。 如果返回值是 #f, 那么不可能执行折叠。 否则,返回值是 #t ,这个 delta 将包含折叠后的修改规范。

(send a-style-delta set-delta change-command 
  on?) 
 → (is-a?/c style-delta%)
  change-command : 
(or/c 'change-underline
      'change-size-in-pixels)
  on? : any/c
用高级规格配置 delta。 返回值是 delta 本身。

除了 'change-nothing 和 'change-normal, 该命令只改变 delta 的一部分。 因此,应用 'change-bold 然后 'change-italic 可以为样式和重量的改变设置 delta。

change-command 参数指定如何改变 delta;可能的值是:

  • 'change-nothing — 重置所有变化

  • 'change-normal — 关闭所有的样式和大小的调整

  • 'change-toggle-underline — 在当前没有下划线的区域下划线,反之亦然

  • 'change-toggle-size-in-pixels — 对目前以点为单位的区域解释为像素大小,反之亦然

  • 'change-normal-color — 将前景和背景分别改为黑色和白色

  • 'change-italic — 将字体的风格改为 italic

  • 'change-bold — 将字体的重量改为 bold

  • 'change-family — 改变字体家族 (param 是一个家族, 见 font%);也见 get-family

  • 'change-style — 改变字体的风格 (param 是一种风格;见 font%)

  • 'change-toggle-style — 切换字体的风格 (param 是一种风格;见 font%)

  • 'change-weight — 改变字体的重量 (param 是一个重量;见 font%)

  • 'change-toggle-weight — 切换字体的重量 (param 是重量,见 font%)

  • 'change-smoothing — 改变字体的平滑度 (param 是平滑度;见 font%)

  • 'change-toggle-smoothing — 切换字体的平滑度 (param 是平滑度;见 font%)

  • 'change-alignment — 改变对齐方式 (param 是一种对齐方式;见 style-delta%)

  • 'change-size — 改变尺寸为一个绝对值 (param 是一个尺寸)

  • 'change-bigger — 使文字变大(param 是一个加号)

  • 'change-smaller — 使文本变小 (param 是一个加数)

  • 'change-underline — 将下划线状态设置为下划线或普通状态

  • 'change-size-in-pixels — 将尺寸解释为像素或点数

method

(send a-style-delta set-delta-background name)

 → (is-a?/c style-delta%)
  name : string?
(send a-style-delta set-delta-background color)
 → (is-a?/c style-delta%)
  color : (is-a?/c color%)
使得 delta 编码的背景颜色变化与给定的绝对颜色相匹配; 也就是说,它将 get-background-mult 结果中的乘法系数设置为 0.0 , 并将 get-background-add 结果中的加法值设置为指定颜色的值。 此外,它还通过将 transparent-text-backing-on 设置为 #f ， transparent-text-backing-off 设置为 #t 来禁用透明文字的背景。 该方法的返回值是 delta 本身。

对于提供字符串颜色名称的情况,见 color-database<%> 。

class

editor-snip% : class?

  superclass: snip%

editor-snip% 对象是一个 snip% 对象,它包含并显示一个 editor<%> 对象。 这个 snip 类用于在另一个编辑器中插入一个编辑器作为单一 item 。

constructor

(new editor-snip% 
    [[editor editor] 
    [with-border? with-border?] 
    [left-margin left-margin] 
    [top-margin top-margin] 
    [right-margin right-margin] 
    [bottom-margin bottom-margin] 
    [left-inset left-inset] 
    [top-inset top-inset] 
    [right-inset right-inset]

    [max-height max-height]]) 
 → (is-a?/c editor-snip%)
  editor : (or/c (is-a?/c editor<%>) #f) = #f
  with-border? : any/c = #t
  left-margin : exact-nonnegative-integer? = 5
  top-margin : exact-nonnegative-integer? = 5
  right-margin : exact-nonnegative-integer? = 5
  bottom-margin : exact-nonnegative-integer? = 5
  left-inset : exact-nonnegative-integer? = 1
  top-inset : exact-nonnegative-integer? = 1

  max-width : (or/c (and/c real? (not/c negative?)) 'none)
   = 'none
  min-height : (or/c (and/c real? (not/c negative?)) 'none)
   = 'none
  max-height : (or/c (and/c real? (not/c negative?)) 'none)
   = 'none
如果 editor 是非 #f, 那么它将被用作 snip 所包含的编辑器。 参见 set-editor 。

如果 with-border? 不是 #f, 那么将在 snip 周围画一个边界。 编辑器将按照 -margin 参数中指定的数量嵌入到剪贴区域中。 边框将以 -inset 参数指定的嵌入量绘制。

参见 get-inset 和 get-margin 关于 inset 和 margin 参数的信息。

Overrides get-extent in snip%.
调用其编辑器的 get-extent 方法,然后添加编辑器的 snip 页边距。

顶部空间总是对应于编辑器的顶行空间,加上剪贴的顶部边距。 通常情况下,下降空间对应于编辑器最后一行的下降空间加上剪贴的底边距。 然而,如果狙击手处于对齐顶行模式(见 set-align-top-line), 则下降相当于顶行的下降,加上编辑器的其余行的高度,再加上狙击手的底边距。

如果编辑器是一个文本编辑器,那么通常从 get-extent 返回的编辑器宽度中减去 1 , 因为结果看起来更适合编辑。 如果剪贴画处于紧贴文本模式(见 set-tight-text-fit), 那么 2 会从文本编辑器的宽度中减去,消除文本编辑器为闪烁的光标保留的两个像素。 此外,tight-text-fit 模式会从编辑器的高度中减去与行距相等的数量。 默认情况下,紧贴文本模式是禁用的。

method

(send an-editor-snip set-editor editor) → void?

  editor : (or/c (or/c (is-a?/c text%) (is-a?/c pasteboard%)) #f)
设置剪贴所包含的编辑器,释放剪贴中的旧编辑器(如果有的话)。 如果新的编辑器已经有一个管理员,那么新的编辑器 不 会被安装到剪贴中。

当 editor-snip% 对象没有被插入一个编辑器时,它没有管理员。 在这段时间里,它也不会给它所包含的编辑器一个管理员。 因此,没有管理员的所包含的编辑器可以 "叛变" 到其他有管理员的 display 。 当一个被包含的编辑器叛变,并且该狙击手最终被插入到一个不同的编辑器中时, 该狙击手会放弃叛变的被包含的编辑器,将其被包含的编辑器设置为 #f 。

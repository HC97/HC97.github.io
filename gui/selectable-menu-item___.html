interface

selectable-menu-item<%> : interface?

  implements: labelled-menu-item<%>
一个 selectable-menu-item<%> 对象是一个用户可以选择的 labelled-menu-item<%> 。 它也可以有一个键盘快捷键;该快捷键显示在菜单中,当快捷键组合被按下时, 菜单框架中默认的 on-subwindow-char 方法会派发给菜单项。

method

(send a-selectable-menu-item command event) → void?

  event : (is-a?/c control-event%)
调用菜单项的回调过程,该过程在创建 menu-item% 或 checkable-menu-item% 的实例时提供。

method

(send a-selectable-menu-item get-shortcut)

 → (or/c char? symbol? #f)
获取该菜单项的键盘快捷键字符或虚拟键。这个字符或键与快捷键的前缀相结合, 后者由 get-shortcut-prefix 报告。

如果该菜单项没有快捷键,则返回 #f 。

菜单项名称中的快捷键部分不包括在 get-label 返回的标签中。

关于允许的按键符号列表,请参见 get-key-code in key-event%, 但以下内容是不允许的: 'shift, 'control, 'numlock, 'scroll, 'wheel-up, 'wheel-down, 'release 和 'press 。

method

(send a-selectable-menu-item get-shortcut-prefix)

 → 
(and/c (listof (or/c 'alt 'cmd 'meta 'ctl
                     'shift 'option))
       (λ (x) (implies (equal? 'unix (system-type))
                       (not (and (member 'alt x)
                                 (member 'meta x)))))
       (λ (x) (equal? x (remove-duplicates x))))
返回一个符号列表,表示用于菜单项的键盘快捷方式的键盘前缀。 列表中允许的符号有以下几种:

  • 'alt — Meta (仅 Windows 和 X)

  • 'cmd — Command (仅 Mac OS)

  • 'meta — Meta (仅 Unix)

  • 'ctl — 控制

  • 'shift — 移位

  • 'option — Option (仅 Mac OS)

在 Unix 上,最多只能提供 'alt 和 'meta 中的一个; 'alt 和 'meta 的唯一区别是组合键在菜单中的显示。

默认的快捷键前缀可以通过 get-default-shortcut-prefix 获得。

由 get-shortcut 确定的快捷键, 与使用通常报告的键码或其他 Shift/AltGr 键码 (由 get-other-shift-key-code in key-event% 产生)的键事件匹配。 当快捷键是一个键码符号或 ASCII 字母或数字时,那么该快捷键只与前缀中列出的修饰键的确切组合相匹配。 然而,对于 ASCII 字母和数字以外的字符快捷键,那么快捷键前缀只是确定了一组最小的修改键, 因为可能需要额外的修改键来访问该字符; 一个例外是,在 Windows 或 Unix 上,Alt/Meta 键的按下必须完全匹配前缀(即包括或不包括)。 在所有情况下,最精确的匹配优先;参见 map-function in keymap% ,了解更多关于匹配排序的信息。

一个空的列表可以用来作为快捷键的前缀。 然而,默认的 on-menu-char in frame% 方法只在按键事件包括非 Shift 修改器或 Function 键时检查菜单快捷方式。 因此,一个空的快捷键前缀通常只有在快捷键是一个功能键时才有用。

method

(send a-selectable-menu-item set-shortcut shortcut) → void?

  shortcut : (or/c char? symbol? #f)
设置菜单项的键盘快捷键字符。更多信息见 get-shortcut 。

如果快捷键字符被设置为 #f ,那么菜单项就没有键盘快捷键。

method

(send a-selectable-menu-item set-shortcut-prefix prefix)

 → void?
  prefix : 
(and/c (listof (or/c 'alt 'cmd 'meta 'ctl
                     'shift 'option))
       (λ (x) (implies (equal? 'unix (system-type))
                       (not (and (member 'alt x)
                                 (member 'meta x)))))
       (λ (x) (equal? x (remove-duplicates x))))
设置一个符号列表,表示用于菜单项的键盘快捷方式的键盘前缀。

更多信息见 get-shortcut-prefix 。

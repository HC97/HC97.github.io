interface

readable-snip<%> : interface?

open-input-text-editor 生成的 port 会对 readable-snip<%> 对象进行特殊处理: 当输入流遇到一个 readable-snip<%> 对象时, 它的 read-special 方法会被调用, 以生成该狙击手的读取结果, 并作为 read-char-or-special 意义上的 "特殊" 值从端口返回。

由于 read 和 read-syntax 建立在 read-char-or-special 的基础上, 一个 snip 可以实现 readable-snip<%> ,这样当 read 被用于包含该 snip 的流时, 它就会产生一个完整的 S 表达式或者其他类型的值。

method

(send a-readable-snip read-special source    
  line    
  column    
  position) → any/c
  source : any/c
  line : (or/c exact-nonnegative-integer? #f)
  column : (or/c exact-nonnegative-integer? #f)
  position : (or/c exact-nonnegative-integer? #f)
参数与自定义输入端口的 read-in 所返回的过程的参数相同; 详见 Custom Ports 。 结果也与 read-in 产生的过程的结果相同。

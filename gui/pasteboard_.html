class

pasteboard% : class?

  superclass: object%

  extends: editor<%>
pasteboard% 对象是一个编辑器,用于显示具有任意 location 的剪贴。

constructor

(new pasteboard%) → (is-a?/c pasteboard%)

该编辑器不会被显示出来,直到它被连接到一个 editor-canvas% 对象或其他 display 方式。

一个新的 keymap% 对象将为新的编辑器创建。 参见 get-keymap 和 set-keymap 。

为新的编辑器创建一个新的 style-list% 对象。 参见 get-style-list 和 set-style-list 。

(send a-pasteboard add-selected snip) → void?

  x : real?
  y : real?
  w : (and/c real? (not/c negative?))
  h : (and/c real? (not/c negative?))
选择剪辑而不取消对其他剪辑的选择。 当给定坐标时,这个方法选择所有与给定矩形相交的片段(在编辑器坐标中)。

The selection in a pasteboard 可以被 the system in response to other method calls 改变,这种改变不经过这个方法;使用 on-select 来监控 selection 的变化。

method

(send a-pasteboard change-style [style snip]) → void?

  style : (or/c (is-a?/c style-delta%) (is-a?/c style<%>) #f)
   = #f
  snip : (or/c (is-a?/c snip%) #f) = #f
将 snip 的样式改为特定的样式,或者通过应用一个样式 delta。 如果 snip 是 #f, 那么所有当前选择的 snip 都被改变。 如果 style 是 #f, 那么将使用默认的样式,根据 default-style-name 。

如果要将大量的片段从一种风格改为另一种风格,可以考虑提供一个 style<%> 实例而不是 style-delta% 实例。 否则, change-style 必须为每个片段将 style-delta% 实例转换为 style<%> 实例;这种转换会消耗时间和(临时)内存。

When a style is provided: 编辑器的样式列表必须包含 style, 否则样式不会被改变。参见 convert in style-list% 。

method

(send a-pasteboard find-snip x y [after])

 → (or/c (is-a?/c snip%) #f)
  x : real?
  y : real?
  after : (or/c (is-a?/c snip%) #f) = #f
找到与给定 location 相交的最前面的狙击点(在一个给定的狙击点之后)。 参见 编辑器结构和术语 ,了解关于粘贴板中的剪裁顺序的信息。

x 和 y 参数是编辑器的坐标。 如果没有提供 after ,将返回 x 和 y 处最前面的片段, 否则将返回 after 后面最前面的片段。 如果 after 是一个不在粘贴板中的片段,则返回 #f 。

method

(send a-pasteboard on-default-event event) → void?

  event : (is-a?/c mouse-event%)
Overrides on-default-event in editor<%>.
选择、拖动和调整剪辑的大小:

  • 点击一个片段可以选择该片段。 按住 Shift 键可以将当前的选区扩展到该片段。

  • 在片段之间的空间点击拖动一个选择框;一旦释放鼠标按钮,所有触及选择框的片段都被选中。 按住Shift键,用新的剪贴扩展当前的选区。

  • 双击一个片段会调用 on-double-click 。

  • 点击一个选中的片段,可以将选中的片段拖到一个新的 location 。

  • 点击一个被选中的对象上的 iliting 标签可以调整该对象的大小。

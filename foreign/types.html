3 C 类型

C types 是 FFI　的主要概念,要么是原始类型,要么是用户定义的类型。 FFI 在内部处理原始类型,将它们转换为 C 类型或从 C 类型转换。 用户类型是根据现有的原始类型和用户类型,以及与现有类型之间的转换函数定义的。

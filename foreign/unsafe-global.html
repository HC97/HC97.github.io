5.12 全进程和全地点的注册

 (require ffi/unsafe/global) package: base
ffi/unsafe/global 库提供了一个注册信息的工具,这些信息是本地的,还是跨越 Racket 进程中的所有地方。

Added in version 6.9.0.5 of package base.

procedure

(register-process-global key val) → cpointer?

  key : bytes?
  val : cpointer?
获取或设置进程全局表中的一个值(即在多个地方共享,如果有的话)。

如果 val 是 #f, 将报告 key 的当前映射。

如果 val 不是 #f, 并且没有为 key 安装任何值,那么该值将被安装并返回 #f 。 如果已经安装了一个值,那么就不会再安装新的值,而是返回旧值。 给定的 val 不能指向垃圾收集的内存。

这个函数是为不经常使用的少量的键而设计的。

procedure

(get-place-table) → hash?

返回一个特定地点的、可变的、基于 eq? 的哈希表。 对于一个特定的地方,其结果总是相同的。

Added in version 6.11.0.6 of package base.

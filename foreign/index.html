Racket 外国接口

Eli Barzilay

 (require ffi/unsafe) package: base

ffi/unsafe 库可以在 Racket 程序中直接使用基于 C 的 API,而不需要编写任何新的 C 代码。 从 Racket 的角度来看,基于 C 语言的 API 的函数和数据是 foreign, 因此称为 foreign interface 。 此外,由于大多数 API 主要由函数组成,外国接口有时被称为 foreign function interface, 缩写为 FFI 。

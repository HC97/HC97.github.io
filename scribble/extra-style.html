6.11.1 实现样式

当一个字符串被用作 element, multiarg-element, paragraph, table,itemization, nested-flow 或 compound-paragraph 中的样式时,它对应于 HTML 输出的 CSS 类或 Latex 输出的 Latex 宏/环境。 在 Latex 输出中,该字符串被用作 paragraph 的命令名,以及 table, itemization, nested-flow 或 compound-paragraph 的环境名; 如果该样式对于 nested-flow 或 compound-paragraph 有一个 'command style property ,那么该样式名被用作命令而不是环境; 如果该样式对于 nested-flow 有一个 'multicommand style property , 那么该样式名被用作一个有多个参数的命令。 此外,对于一个项目化,样式字符串以 "Item" 为后缀,并被用作 CSS 类或 Latex 宏的名称, 用于项目化的项目(在 Latex 的情况下代替 \item )。

要添加一个从你自己的样式名称到 CSS 配置的映射, 请添加一个 css-addition 结构实例到一个样式的 style property 列表。 要把一个样式名称映射到一个 Latex 宏或环境,请添加一个 tex-addition 结构实例。 css-addition 或 tex-addition 通常与样式相关联, 其名称是由 addition 实现的,但它也可以被添加到一个包围部分的样式中。

Scribble 包括一些预定义的样式,它们被 scribble/base 的出口所使用。 你可以使用它们或重新定义它们。 这些样式是由 "scribble" 集合中的 "scribble.css" 和 "scribble.tex" 指定。

scribble/manual 使用的样式由 "scribble" 集合中的 "racket.css" 和 "racket.tex" 实现。 其他库,比如 scriblib/autobib, 也同样通过 css-addition 和 tex-addition style properties 关联的文件实现样式。

为了避免与 Scribble 未来的添加物发生冲突,请以非 S 的大写字母开始你的样式名称。 大写字母有助于避免与 Latex 包所定义的宏相冲突, 而且将来 scribble/base 和 scribble/manual 所需要的样式将以 S 开头。

例如,一个 Scribble 文档

  #lang scribble/manual

  @(require scribble/core

            scribble/html-properties

            scribble/latex-properties)

  

  @(define inbox-style

     (make-style "InBox"

                 (list (make-css-addition "inbox.css")

                       (make-tex-addition "inbox.tex"))))

  

  @title{Quantum Pet}

  

  Do not open: @elem[#:style inbox-style]{Cat}

与包含 "inbox.css" 的

  .InBox {

    padding: 0.2em;

    border: 1px solid #000000;

  }

和 "inbox.tex" 的

  \newcommand{\InBox}[1]{\fbox{#1}}

生成

Quantum Pet

Do not open: Cat

Scribble 文档也可以嵌入特定的 html 标签和属性。 例如,这个 Scribble 文档:
#lang scribble/base
 
@(require scribble/core
          scribble/html-properties)
 
@(define external-image
   (elem
    #:style
    (style #f
           (list (alt-tag "img")
                 (attributes
                  '((src . "http://racket-lang.org/icon.png")))))))
 
@external-image

在产生 html 时,渲染成 Racket 的标志,网址是 http://racket-lang.org/logo.png 。

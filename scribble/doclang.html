6.6 文档语言

 #lang scribble/doclang2 package: scribble-lib
scribble/doclang2 语言提供了 racket/base 的所有内容,除了它取代了 #%module-begin 形式。
scribble/doclang2 的 #%module-begin 本质上是将模块的主体打包成对 decode 的调用,将结果与 doc 绑定,并导出 doc 。
任何除表达式之外的模块级形式(例如, require 或 define) 都保留在顶层, 并且将 doc 绑定在模块的最后。 像往常一样,一个模块顶层的 begin 会被切入模块顶层。

例如:
#lang racket
(module example scribble/doclang2
  "hello world, this is"
  " an example document")
(require 'example)
doc

scribble/doclang2 的行为可以通过在模块的最开始提供 #:id, #:post-process, #:begin 和 #:exprs 参数来定制。

  • #:id 命名顶级的文档绑定。默认情况下,它是 doc 。

  • #:post-process 在 decode 后处理模块的主体。默认情况下,这是 values 。

  • #:begin 在不考虑文档内容的情况下,将额外的表达式序列预置到模块主体的开头。 例如,默认的 configure-runtime 子模块可以使用 #:begin 替换, 因为使用 #:exprs 替换的嵌套太深,不能作为覆盖使用。 默认情况下,这是一个空序列 () 。

  • #:exprs 在模块主体的开始处预置一个额外的表达式序列,但在 #:begin 之后。 默认情况下,这是一个空序列 () 。

这个例子明确地使用了所有三个关键字的默认值:
#lang racket
(module example scribble/doclang2
  #:id doc
  #:post-process values
  #:exprs ()
  "hello world, this is an example document")
(require 'example)
doc
下一个玩具例子为文档绑定使用了一个不同的名字,并且还增加了一个额外的绑定,包含了文档中的部分计数:
#lang racket
(module example scribble/doclang2
  #:id documentation
  #:post-process (lambda (decoded-doc)
                   (set! number-of-parts (length (part-parts decoded-doc)))
                   decoded-doc)
  #:exprs ((title "My first expression!"))
 
  (require scribble/core
           scribble/base)
 
  (define number-of-parts #f)
  (provide number-of-parts)
  (section "part 1")
  "hello world"
  (section "part 2")
  "this is another document")
 
(require 'example)
number-of-parts
documentation

Changed in version 1.41 of package scribble-lib: 添加了 #:begin 。

6.6.1 scribble/doclang

 #lang scribble/doclang package: scribble-lib
scribble/doclang 语言提供了大部分与 scribble/doclang2 相同的功能,其中的配置选项是位置性的和强制性的。 #%module-begin 的主体中的前三个元素必须是 id, post-process 和 exprs 参数。

例如:
#lang racket
(module* example scribble/doclang
  doc
  values
  ()
  (require scribble/base)
  (provide (all-defined-out))
  (define foo (para "hello again"))
  "hello world, this is an example document"
  (para "note the " (bold "structure")))
 
(module+ main
  (require (submod ".." example))
  (printf "I see doc is: ~s\n\n" doc)
  (printf "I see foo is: ~s" foo))

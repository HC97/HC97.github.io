3.7 JFP 论文格式

 #lang scribble/jfp package: scribble-lib
scribble/jfp 语言和 scribble/base一样,但是配置了 Latex 风格的默认值,以使用 "jfp1.cls" 类文件。 由于许可证问题,该类文件不包括在 Scribble 中,但是如果该文件没有被手动安装到 scribble/jfp 集合中,那么它会被下载到 (find-system-path 'addon-dir) 。

使用 scribble/jfp 输出使用提供给 title 的主文件版本作为短式文件名(用于页眉)。

procedure

(abstract pre-content ...) → block?

  pre-content : pre-content?
为论文摘要生成一个 nested flow 。

syntax

(include-abstract module-path)

类似于 include-section, 但将文件纳入指定的模块作为一个摘要。 该文件必须没有标题或子部分。

procedure

(author name ...) → block?

  name : pre-content?
替换 scribble/base 中的 author 。

procedure

((author/short short-name ...) long-name ...) → block?

  short-name : pre-content?
  long-name : pre-content?
与 author 类似,但允许将短格式的名字(用于页眉)与长格式的名字分开指定。

procedure

(affiliation place ...) → element?

  place : pre-content?

procedure

(affiliation-mark mark ...) → element?

  mark : pre-content?

procedure

(affiliation-sep) → element?

使用 author 内的 affiliation 或 author/short 的长名部分来指定所有作者后的 affiliation。 如果不同的作者有不同的隶属关系,在每个作者后面使用带有数字的 affiliation-mark , 然后在单个 affiliation 中的每个不同隶属关系之前使用 affiliation-mark , 使用 (affiliation-sep) 来分隔隶属关系。

例子:
#lang scribble/jfp
 
@title{My First Love Story}
 
@((author/short "Romeo M. and Juliet C.")
  "ROMEO" (affiliation-mark "1")
  " and "
  "JULIET" (affiliation-mark "2")
  @affiliation[
    "House Montague" (affiliation-mark "1")
    (affiliation-sep)
    "House Capulet" (affiliation-mark "2")])

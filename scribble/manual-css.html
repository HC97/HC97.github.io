6.11.4 手册 CSS 样式类

以下是 "demo-manual.scrbl" 的渲染图展示了除了 base style classes 之外, scribble/manual 表单和功能所使用的所有 CSS 样式类。

  • S1 Manual All-Styles Document 在一个没有搜索框的多页渲染中显示原始样式。

  • M1 Manual All-Styles Document shows the original style in a multi-page rendering without a search box.

  • S2 Manual All-Styles Document 在一个带有搜索框的单页渲染中显示当前手动样式的调整。

  • M2 Manual All-Styles Document 显示了当前手动样式在多页渲染中的调整,并有一个搜索框。

风格类:

 

RktSym

 

没有 for-label 绑定的标识符。

 

RktValLink

 

具有 for-label 绑定到一个变量定义的标识符。

 

RktValDef

 

变量的定义部位,通常与 RktValLink 结合。

 

RktStxLink

 

具有 for-label 绑定到句法形式定义的标识符。

 

RktStxDef

 

句法形式的定义点,通常与 RktStxLink 结合。

 

RktSymDef

 

句法形式的定义点,通常与 RktSym 结合。

 

RktVar

 

本地变量或元变量。

 

RktRes

 

REPL 结果。

 

RktOut

 

写到当前输出端口的输出。

 

RktErr

 

写到当前错误端口的输出。

 

RktCmt

 

Racket代码中的一个注释。

 

RktVal

 

Racket代码中的一个字面值。

 

RktPn

 

Racket代码中的括号、关键字和类似的定界符。

 

RktRdr

 

Racket代码中的读者速记符号,逗号除外。

 

RktMeta

 

Racket代码中的去引号逗号。

 

highlighted

 

高亮的代码(例如,通过 racketblock 中的 code:highlight )。

 

RktIn

 

用 litchar 写的文字的前景。

 

RktInBG

 

用 litchar 书写的文字的背景。

 

RktModLink

 

链接到模块定义的模块名称。

 

RktMod

 

一个模块名称(通常是 RktModLink, 代替)。

 

RktKw

 

一个 "关键字;"通常不使用。

 

RktOpt

 

可选参数的方括号(在函数定义中)。

 

 

 

 

 

RktBlk

 

多线Racket代码块的封装器。

 

 

 

 

 

defmodule

 

模块定义块。

 

RpackageSpec

 

模块定义块中的包规范。

 

 

 

 

 

RBoxed

 

定义块;总是与 boxed 结合。

 

together

 

在 together 分组中的表。

 

RBackgroundLabel

 

包裹 "程序"、"语法 "等,在定义框中支持。

 

RBackgroundLabelInner

 

RBackgroundLabel 中的封装器。

 

RForeground

 

在 RBackgroundLabel 上出现的元素的包装器。

 

prototype

 

多行程序定义原型的包装器。

 

argcontract

 

多行参数契约和缺省值的封装器。

 

specgrammar

 

语法的封装器,带有句法形式定义框。

 

inherited

 

边缘 "继承方法 "表的封装器。

 

inheritedlbl

 

用于 "继承方法:"和 "来自 "标签的封装器。

 

 

 

 

 

leftindent

 

向左缩进的块,如表格 specsubform 。

 

insetpara

 

Inset block.

 

 

 

 

 

Rfilebox

 

文件框的封装器(通过 filebox),

 

Rfiletitle

 

文件框标题的外包装。

 

Rfilename

 

文件框标题的内层包装器。

 

Rfilecontent

 

文件框内容的封装器。

 

 

 

 

 

SHistory

 

history 段落的包装器。

 

 

 

 

 

RBibliography

 

bibliography 部分的包装器。

class

brush% : class?

  superclass: object%

画笔是一种具有颜色和样式的绘图工具,用于填充区域,如矩形或椭圆的内部。 在单色目标中,所有非白色的笔刷都被绘制成黑色。

除了它的颜色和样式,画笔还可以有一个 brush stipple 位图。 用笔刷画画类似于调用 draw-bitmap ,其笔刷位图在填充区域。

作为颜色、样式和僵局的替代,画笔可以有一个 gradient , 即 linear-gradient% 或 radial-gradient% 。 当一个画笔有一个渐变,并且绘制的目标不是单色的,那么其他的画笔设置就会被忽略。 对于梯度,对于绘图目标中的每一个点, 梯度会根据起始和结束的颜色以及起始和结束的线(对于线性梯度)或圆(对于径向梯度)为该点关联一个颜色; 当用画笔绘图时,梯度分配的颜色会应用于每个被触摸的点。

默认情况下,当使用画笔时,僵局或渐变中的坐标由绘图上下文的变换来转换, 但画笔可以有自己的 brush transformation 来代替。 画笔变换与 get-transformation in dc<%> 的表示和含义相同。

brush style 是下列之一(但如果画笔有 gradient ,而且目标不是单色的,则会被忽略):

  • 'transparent — 绘制时没有任何效果(对绘制的形状的内部)。

  • 'solid — 使用画笔的颜色绘制。如果在画笔中安装了一个单色 brush stipple , 来自纹路的黑色像素会使用画笔的颜色转移到目标,而来自纹路的白色像素则不会被转移。

  • 'opaque — 与彩色 brush stipple 的 'solid 相同。 对于单色画笔,画笔中的白色像素会使用目的地的背景色转移到目的地。

  • 'xor — 与 'solid 相同,只接受部分后向兼容。

  • 'hilite — 以黑色和 0.3 的 alpha 绘制。

  • 'panel — 与 'solid 同,只接受部分向后兼容。

  • 以下模式对应于在 'solid 模式下绘制的内置 brush stipples :

    • 'bdiagonal-hatch — 对角线,从左上到右下

    • 'crossdiag-hatch — 交叉的斜线

    • 'fdiagonal-hatch — 对角线,从右上方到左下方

    • 'cross-hatch — 交叉的水平线和垂直线

    • 'horizontal-hatch — 水平线

    • 'vertical-hatch — 垂直线

    然而,当一个特定的 brush stipple 被安装到笔刷中时,上述模式被忽略,而使用 'solid 。

要 绘制轮廓形状(如未填充的方框和椭圆),请使用 'transparent 画笔样式。

为了避免创建具有相同特征的多个笔刷,使用全局的 brush-list% 对象 the-brush-list, 或者向 set-brush in dc<%> 提供一个颜色和样式。

也请看 make-brush 。

用给定的颜色、 brush style, brush stipple, gradient 和 brush transformation (仅在渐变或纹路为非 #f 时保留)创建一个刷子。 对于使用名称指定颜色的情况,请参阅 color-database<%> 了解有关颜色名称的信息;如果不知道名称,画笔的颜色是黑色。

method

(send a-brush get-color) → (is-a?/c color%)

返回画笔的颜色。

method

(send a-brush get-gradient) → 
(or/c (is-a?/c linear-gradient%)
      (is-a?/c radial-gradient%)
      #f)
获取 gradient ,如果画笔没有梯度,则获取 #f 。

method

(send a-brush get-handle) → (or/c cpointer? #f)

返回画笔内容的低级手柄,但只适用于用 make-handle-brush 创建的画笔;否则,结果是 #f 。

method

(send a-brush get-stipple) → (or/c (is-a?/c bitmap%) #f)

获取 brush stipple 位图,如果画笔没有纹路,则返回 #f 。

method

(send a-brush get-style) → brush-style/c

返回 brush style 。关于画笔样式的信息请参见 brush% 。

method

(send a-brush get-transformation)

 → 
(or/c #f (vector/c (vector/c real? real? real? real? real? real?)
                   real? real? real? real? real?))
返回画笔的 brush transformation ,如果有的话。

如果一个带有僵局或渐变的画笔也有一个变换,那么该变换适用于僵局或渐变的坐标,而不是目标绘图环境的变换; 否则,目标绘图环境的变换适用于僵局和渐变的坐标。

method

(send a-brush is-immutable?) → boolean?

如果画笔对象是不可改变的,返回 #t 。

method

(send a-brush set-color color) → void?

  color : (is-a?/c color%)
(send a-brush set-color color-name) → void?
  color-name : string?
(send a-brush set-color red green blue) → void?
  red : byte?
  green : byte?
  blue : byte?
设置画笔的颜色。 如果一个画笔是从一个 brush-list% 中获得的,或者当它被选择到一个绘图上下文中时,它不能被修改。

对于使用字符串指定颜色的情况,请参阅 color-database<%> 了解有关颜色名称的信息。

method

(send a-brush set-stipple bitmap    
  [transformation]) → void?
  bitmap : (or/c (is-a?/c bitmap%) #f)
  transformation : 
(or/c #f (vector/c (vector/c real? real? real?
                             real? real? real?)
                    real? real? real? real? real?))
   = #f
设置或删除 brush stipple 位图,其中 #f 会删除波纹。 brush transformation 也同时被设置为 transformation 。 参见 brush% ,了解关于用笔触绘图的信息。

如果 bitmap 在与画笔相关联时被修改,对画笔的影响是不明确的。 如果一个画笔是从一个 brush-list% 中获得的,或者当它被选入一个绘图环境时,就不能被修改。

method

(send a-brush set-style style) → void?

  style : brush-style/c
设置 brush style 。 有关可能的样式的信息,请参见 brush% 。

如果一个画笔是从 brush-list% 中获得的,或者当它被选中进入绘图环境时,就不能修改它。

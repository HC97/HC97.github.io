class

ps-setup% : class?

  superclass: object%

ps-setup% 对象包含使用 post-script-dc% 对象产生 PostScript 文件的配置信息。 在较小的程度上,它包含了使用 printer-dc% 对象打印的信息。

当一个 post-script-dc% 对象被创建时, 它的配置由 current-ps-setup 参数的 ps-setup% 值决定。 创建 post-script-dc% 对象后, 它不会受到 current-ps-setup 参数的变化或 ps-setup% 对象的突变的影响。

constructor

(new ps-setup%) → (is-a?/c ps-setup%)

创建一个新的 ps-setup% 对象,具有(特定平台)默认配置。

method

(send a-ps-setup copy-from source    
  [copy-filename?]) → void?
  source : (is-a?/c ps-setup%)
  copy-filename? : any/c = #f
复制设置 copy-from 到 a-ps-setup, 不包括文件名,除非 copy-filename? 为真。

method

(send a-ps-setup get-command) → string?

历史上,在 Unix 中获得用于打印文件的打印机命令。默认是 "lpr" 。 这个值目前不被任何平台使用。

method

(send a-ps-setup get-editor-margin h-margin    
  v-margin) → void?
  h-margin : (box/c (and/c real? (not/c negative?)))
  v-margin : (box/c (and/c real? (not/c negative?)))
返回打印 editor<%> 时水平和垂直边距的当前设置。 参见 set-editor-margin 。

method

(send a-ps-setup get-file) → (or/c path-string? #f)

获取 PostScript 输出文件名。 #f 值(默认值)表示在创建 post-script-dc% 对象时应提示用户输入文件名。

method

(send a-ps-setup get-level-2) → boolean?

报告 Level 2 命令是否在 PostScript 文件中输出。

目前,只有在 PostScript 输出中包含彩色位图图像(用 draw-bitmap 绘制)或位图笔和笔尖时才需要第 2 级命令。 当第2级命令被禁用时,位图将被转换为灰度图像,并且不支持硬笔画。

method

(send a-ps-setup get-margin h-margin    
  v-margin) → void?
  h-margin : (box/c (and/c real? (not/c negative?)))
  v-margin : (box/c (and/c real? (not/c negative?)))
返回水平和垂直 PostScript 边距的当前设置。请参见 set-margin 。

method

(send a-ps-setup get-mode) → (or/c 'preview 'file 'printer)

获取决定输出被送往何处的打印模式: 'preview, 'file 或 'printer 。 X 的默认值是 'preview 。 Windows 和 Mac OS 中的值总是 'file 。

method

(send a-ps-setup get-orientation)

 → (or/c 'portrait 'landscape)
获取方向: 'portrait 或 'landscape 。 默认是 'portrait 。 与其他大多数设置不同,这个设置会影响本地打印(通过 printer-dc%)以及 PostScript 输出。

横向方向会影响 get-size 所报告的绘图区域的尺寸:由所选纸张类型决定的水平和垂直尺寸被移位,然后被缩放。

method

(send a-ps-setup get-paper-name) → string?

返回当前纸张类型的名称: "A4 210 x 297 mm", "A3 297 x 420 mm", "Letter 8 1/2 x 11 in" 或 "Legal 8 1/2 x 14 in" 。 默认是 "Letter 8 1/2 x 11 in" 。

纸张名称决定了 get-size 所报告的绘图区域的尺寸 (连同来自 get-orientation 的横向变换和/或 get-scaling 的缩放系数)。 当创建 post-script-dc% 上下文时,如果初始化参数 use-paper-bbox? 的值为 true, 它还将决定 PostScript 输出的边界框。

method

(send a-ps-setup get-preview-command) → string?

获取用于查看 X 的 PostScript 文件的命令。 默认是 "gv" 。这个值不被其他平台使用。

method

(send a-ps-setup get-scaling x y) → void?

  x : (box/c (and/c real? (not/c negative?)))
  y : (box/c (and/c real? (not/c negative?)))
Gets the scaling factor for PostScript output. The x box is filled with the horizontal scaling factor. The y box is filled with the vertical scaling factor. 默认是 0.8 乘 0.8 。

这个比例是在 post-script-dc% 上下文中可以通过 set-scale 设置的比例之外的。 get-size 所报告的尺寸是所选纸张类型的尺寸(横向模式下移位)除以这个比例。

method

(send a-ps-setup get-translation x y) → void?

  x : (box/c (and/c real? (not/c negative?)))
  y : (box/c (and/c real? (not/c negative?)))
获取 PostScript 输出的平移(从左下角开始)。 The x box is filled with the horizontal offset. The y box is filled with the vertical offset. 默认是 0.0 和 0.0 。

平移不以 get-scaling 返回的数字为尺度,平移不影响绘图区的大小。

method

(send a-ps-setup set-command command) → void?

  command : string?
在历史上,设置在 Unix 上用于打印文件的打印机命令。参见 get-command 。

method

(send a-ps-setup set-editor-margin h v) → void?

  h : exact-nonnegative-integer?
  v : exact-nonnegative-integer?
设置用 print 方法打印编辑器时使用的水平和垂直边距。 无论绘图目的地是 post-script-dc% 还是 printer-dc% ,这些边距总是用于打印。 这些边距是以目的地 printer-dc% 或 post-script-dc% 为单位的。 在 post-script-dc% 打印的情况下, 编辑器的边距是在由 set-margin 决定的 PostScript 边距之外的。

method

(send a-ps-setup set-file filename) → void?

  filename : (or/c path-string? #f)
设置 PostScript 输出文件名。参见 get-file 。

method

(send a-ps-setup set-level-2 on?) → void?

  on? : any/c
设置第 2 级命令是否在 PostScript 文件中输出。 参见 get-level-2 。

method

(send a-ps-setup set-margin h v) → void?

  h : (and/c real? (not/c negative?))
  v : (and/c real? (not/c negative?))
设置 PostScript 的水平和垂直边距。 当绘制到 post-script-dc% 时, 由 get-size 报告的页面大小会从正常的页面面积中减去这些边距(在考虑到缩放影响之前)。 此外,在 post-script-dc% 中绘图产生的 PostScript 输出被边距所抵消。

当把 post-script-dc% 的输出作为 Encapsulated PostScript 使用时,边距值实际上是不相关的。 改变边距会在绝对坐标上移动 PostScript 图像,但它也会移动边界框。

边距是以未缩放的 post-script-dc% 为单位,也就是点。默认的边距是 16 点。

method

(send a-ps-setup set-mode mode) → void?

  mode : (or/c 'preview 'file 'printer)
设置打印模式,控制输出的发送位置。参见 get-mode 。

在 Windows 和 Mac OS 上,如果提供了 'preview 或 'printer, an exn:fail:contract exception is raised 。

method

(send a-ps-setup set-orientation orientation) → void?

  orientation : (or/c 'portrait 'landscape)
设置方向。参见 get-orientation 。

method

(send a-ps-setup set-paper-name type) → void?

  type : string?
设置当前纸张类型的名称。 参见 get-paper-name 。

method

(send a-ps-setup set-preview-command command) → void?

  command : string?
设置用于在 Unix 上查看 PostScript 文件的命令。 见 get-preview-command 。

method

(send a-ps-setup set-scaling x y) → void?

  x : (and/c real? (not/c negative?))
  y : (and/c real? (not/c negative?))
设置 PostScript 输出的缩放系数。 见 get-scaling 。

method

(send a-ps-setup set-translation x y) → void?

  x : real?
  y : real?
设置 PostScript 输出的翻译(从左下角开始)。 参见 get-translation 。

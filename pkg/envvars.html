7.7 包管理环境变量

如果设置了 PLT_PKG_SSL_NO_VERIFY 环境变量,HTTPS 连接的服务器证书就不会被验证。 当通过 HTTPS 访问 Git 服务器时, GIT_SSL_NO_VERIFY 也必须被设置,以禁用证书验证。

正如 GitHub-repository 包源的规范中所指出的,如果设置了 PLT_USE_GITHUB_API 环境变量, GitHub 包将使用 GitHub API 协议而不是使用 Git 协议获得。

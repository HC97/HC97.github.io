2.7 Text Formatting Helpers

 (require slideshow/text) package: slideshow-lib

该模块为格式化文本提供便利的功能。

syntax

(with-size size expr)

在运行 expr 时将 current-font-size 设置为 size 。

Added in version 1.2 of package slideshow-lib.

syntax

(with-scale scale expr)

在运行 expr 时将 current-font-size 乘以 scale 。

Added in version 1.2 of package slideshow-lib.

syntax

(big text)

syntax

(small text)

在运行 text 时, 将 current-font-size 分别按 3/2 或 2/3 的比例进行调整。

Added in version 1.2 of package slideshow-lib.

syntax

(with-font font expr)

在运行 expr 时,将 current-main-font 设置为 font 。

Added in version 1.2 of package slideshow-lib.

syntax

(with-style style expr)

在运行 expr 时,将 style 添加到 current-main-font (通过 cons) 。

Added in version 1.2 of package slideshow-lib.

syntax

(bold text)

syntax

(italic text)

syntax

(subscript text)

syntax

(superscript text)

syntax

(caps text)

在运行 text 时,将粗体、斜体、上标、下标或小写文字的属性分别添加到 current-main-font 。

Added in version 1.2 of package slideshow-lib.

procedure

(blank-line) → pict?

添加一个当前字体大小高度的空行。

Added in version 1.2 of package slideshow-lib.

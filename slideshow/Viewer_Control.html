2.3 Viewer Control

procedure

(start-at-recent-slide) → void?

将谈话的起始幻灯片设置为最近创建的幻灯片。 如果这个功能被多次使用,最后一次的使用会覆盖之前的使用。

procedure

(enable-click-advance! on?) → void?

  on? : any/c
启用或禁用因鼠标点击而产生的幻灯片前进功能。

procedure

(set-use-background-frame! on?) → void?

  on? : any/c
启用或禁用创建背景框架,这通常只在使用 make-slide-inset 时有用。 在第一个幻灯片注册前的最后一次启用/禁用,会一劳永逸地生效。

procedure

(set-page-numbers-visible! on?) → void?

  on? : any/c
决定幻灯片编号最初是否在查看器中可见。

parameter

(current-page-number-font) → (is-a?/c font%)

(current-page-number-font font) → void?
  font : (is-a?/c font%)
决定用于绘制页码的字体的参数(如果可见)。

parameter

(current-page-number-color) → (or/c string? (is-a?/c color%))

(current-page-number-color color) → void?
  color : (or/c string? (is-a?/c color%))
决定用于绘制页码的颜色的参数(如果可见)。

parameter

(current-page-number-adjust) → (-> number? string? string?)

(current-page-number-adjust proc) → void?
  proc : (-> number? string? string?)
控制显示页码的精确文本的参数(如果可见)。 该函数的输入是默认字符串和幻灯片编号,其结果是在右下角绘制的内容。 默认参数值只是返回其第二个参数。

procedure

(set-spotlight-style! [#:size size    
  #:color color]) → void?
  size : (or/c #f (>=/c 0)) = #f
  color : (or/c #f string? (is-a?/c color%)) = #f
调整 "聚光灯" 的大小和颜色,它可以在 Slideshow 中启用,作为鼠标的替代品。 注意,颜色通常应该有小于1的alpha值(以使其部分透明)。 如果 size 或 color 是 #f, 相应的设置就不会改变。

procedure

(set-allow-new-slides-after-close! on?) → void?

  on? : any/c
设置在幻灯片窗口被用户关闭后是否允许新的幻灯片。 默认情况下,在窗口关闭后试图通过 slide 注册一个新的幻灯片会触发一个错误。 用 #t 调用该函数可以使新的幻灯片开始一个新的幻灯片。

Added in version 1.3 of package slideshow-lib.
